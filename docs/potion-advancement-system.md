# 魔药晋升系统设计文档

> 文档用途：定义游戏中的魔药材料获取、调配、服食、消化、晋升仪式、失控与非凡特性替代规则。  
> 适用范围：区域开放世界剧情 RPG 的角色成长系统、背包系统、任务系统、魔药调配 UI、晋升 UI 和后续 Codex 数据开发。  
> 数据联动：本系统读取 `data/pathways.json`、`data/sequences.json`、`data/materials.json`、`data/rituals.json` 和 `data/characteristics.json`。

## 1. 系统目标

魔药晋升系统不是单纯的升级按钮，而是把调查、材料收集、任务选择、风险管理和角色成长连接起来的核心玩法。

玩家每次晋升都应经历：

1. 获得配方或序列情报。
2. 收集主材料、辅助材料或对应非凡特性。
3. 在安全地点调配魔药。
4. 判断当前成功率与失控风险。
5. 完成必要的晋升仪式。
6. 服食魔药并承受结果。
7. 通过扮演法和剧情行为继续消化魔药。

系统重点不是让玩家反复刷材料，而是让每次晋升都像一次小型案件或章节任务：有线索、有准备、有代价、有风险。

## 2. 核心循环

```text
获得配方
  ↓
确认目标序列
  ↓
收集材料 / 非凡特性
  ↓
调配魔药
  ↓
完成晋升仪式
  ↓
服食魔药
  ↓
成功晋升 / 部分失败 / 失控事件
  ↓
解锁能力并进入消化阶段
```

低序列更偏教学和资源收集；中序列开始要求仪式、地点、NPC 或社会影响；高序列晋升应绑定大型任务、组织选择、区域主线和高危污染事件。

## 3. 如何获得材料

材料来源应和地图、组织、途径主题绑定，避免所有材料都从商店购买。

| 来源 | 适用材料 | 玩法作用 |
| --- | --- | --- |
| 主线任务奖励 | 关键主材料、晋升仪式道具 | 保证主线可推进，避免玩家卡死 |
| 支线任务奖励 | 辅助材料、稀有替代材料 | 鼓励探索和区域支线 |
| 地下黑市 | 常见材料、情报、低阶主材料 | 提供花钱解决的路线，同时带来被骗和污染风险 |
| 官方组织仓库 | 合法材料、封存特性 | 绑定阵营声望、权限和任务贡献 |
| 野外采集 | 植物、矿物、灵性环境材料 | 让区域探索产生资源收益 |
| 精英敌人掉落 | 怪物器官、非凡生物材料 | 让战斗服务成长，而不是纯刷怪 |
| 封印物事件 | 特殊材料、非凡特性 | 高风险高收益，常伴随污染 |
| 交易与交换 | 玩家不走战斗路线时的替代获取 | 支持社交、谈判、情报玩法 |

### 3.1 材料分类

系统使用 `materials.json` 中的字段来判断材料用途：

| 字段 | 用途 |
| --- | --- |
| `material_type` | 区分主材料、辅助材料、仪式材料、特殊要求或未知占位 |
| `related_pathway` | 材料归属途径 |
| `related_sequence` | 材料对应序列 |
| `rarity` | 影响掉落、商店刷新和交易价格 |
| `is_core_material` | 判断是否为魔药必要核心 |
| `can_be_replaced_by_characteristic` | 判断非凡特性能否替代 |
| `source_region` | 决定材料主要出现在哪些地图 |

### 3.2 获取规则

- 序列 9 到序列 7：至少提供一条稳定获取路线，例如官方组织、黑市或教学任务。
- 序列 6 到序列 5：材料应进入区域探索、精英敌人、支线任务和黑市竞价。
- 序列 4 到序列 3：材料应绑定区域主线、副本、组织权限或高危封印事件。
- 序列 2 到序列 0：材料不应作为普通掉落出现，必须绑定大型章节、神话副本、阵营选择或终局任务。

### 3.3 材料品质

材料品质影响调配质量、成功率和失控风险。第一版可以先做品质标签，不必为每个材料做复杂数值。

| 品质 | 效果 |
| --- | --- |
| 污染 | 能满足材料标签，但提高污染值和失控风险 |
| 普通 | 标准效果，无额外修正 |
| 纯净 | 提高调配质量，略微降低污染风险 |
| 契合 | 与当前途径、仪式或地点高度匹配，额外提高成功率 |

材料品质可以来自获取方式：官方组织仓库更容易给出“纯净”材料，黑市可能出现“污染”或伪劣材料，特定地点采集或仪式奖励可能出现“契合”材料。

## 4. 如何调配魔药

调配魔药是一个带风险的制作流程，不是普通合成。

### 4.1 调配前置条件

玩家必须满足：

- 已知目标序列配方。
- 当前途径和序列允许晋升。
- 主材料齐全，或拥有可替代主材料的非凡特性。
- 辅助材料齐全，或拥有系统允许的替代品。
- 角色精神状态没有处于严重污染、重伤、恐惧崩溃或高失控预警。
- 调配地点满足最低安全要求。

### 4.2 调配地点等级

| 地点等级 | 示例 | 效果 |
| --- | --- | --- |
| 临时地点 | 旅馆、野外、安全屋 | 可调配低序列，风险较高 |
| 标准地点 | 官方组织调配室、可信黑市工坊 | 可调配中低序列，风险正常 |
| 专业地点 | 教会实验室、隐秘组织仪式室 | 可调配中高序列，降低失控风险 |
| 特殊地点 | 灵界坐标、遗迹核心、神话副本场景 | 高序列或仪式指定地点 |

### 4.3 调配步骤

1. 选择目标序列。
2. 系统自动读取 `sequence.main_materials` 和 `sequence.auxiliary_materials`。
3. 玩家确认使用普通材料还是非凡特性替代。
4. 玩家选择是否加入可选催化材料或安定材料。
5. UI 显示成功率、污染风险、材料损耗和可能后果。
6. 玩家执行调配。
7. 调配成功生成“未服食魔药”物品。
8. 调配失败进入材料损耗、污染残留或异常事件。

### 4.4 可选材料

可选材料不替代配方要求，而是用于修正调配结果。

| 类型 | 作用 | 设计限制 |
| --- | --- | --- |
| 催化材料 | 提高调配品质或强化能力偏向 | 可能增加污染或副作用 |
| 安定材料 | 降低失控风险和精神压力 | 稀有，通常来自官方组织、治疗支线或净化任务 |
| 象征材料 | 提高仪式完成度或途径契合度 | 必须和途径主题、地点或任务行为匹配 |
| 情报材料 | 发现配方陷阱、禁忌条件或替代路线 | 不直接参与调配，但影响 UI 提示和成功率判断 |

### 4.5 调配结果

| 结果 | 说明 |
| --- | --- |
| 完美调配 | 魔药稳定，服食成功率提高 |
| 普通调配 | 魔药可服食，无额外修正 |
| 瑕疵调配 | 可服食但增加失控风险 |
| 调配失败 | 损失部分材料，可能生成污染残渣 |
| 异常反应 | 触发小型事件，例如灵性污染、低语、怪物吸引 |

## 5. 如何判断成功率

成功率应由材料、精神状态、消化程度、仪式完成度和环境共同决定。

### 5.1 基础公式

```text
晋升成功率 =
基础成功率
+ 材料完整度修正
+ 魔药调配质量修正
+ 当前序列消化度修正
+ 晋升仪式完成度修正
+ 安全地点修正
+ 组织/NPC辅助修正
- 污染值惩罚
- 精神压力惩罚
- 强行晋升惩罚
- 非凡特性精神烙印惩罚
```

### 5.2 推荐基础值

| 目标序列 | 基础成功率 | 设计说明 |
| --- | --- | --- |
| 序列 9 | 85% | 教学阶段，应以成功为主 |
| 序列 8-7 | 75% | 引入材料质量和精神状态 |
| 序列 6-5 | 60% | 开始要求扮演消化和仪式准备 |
| 序列 4-3 | 45% | 半神门槛，必须绑定大型任务 |
| 序列 2-1 | 30% | 天使层级，成功应非常依赖仪式 |
| 序列 0 | 不使用普通概率 | 作为终局剧情或特殊系统处理 |

### 5.3 关键修正项

| 修正项 | 范围 | 说明 |
| --- | --- | --- |
| 材料完整度 | -30% 到 +10% | 缺材料不可晋升；材料品质影响稳定性 |
| 调配质量 | -20% 到 +10% | 由调配地点、工具和玩家准备决定 |
| 消化度 | -40% 到 +25% | 未消化强行晋升是主要失控来源 |
| 仪式完成度 | -50% 到 +30% | 高序列核心条件 |
| 安全地点 | -15% 到 +10% | 危险地点可能带来污染 |
| NPC辅助 | 0% 到 +15% | 官方组织、导师或队友辅助 |
| 污染值 | 0% 到 -50% | 污染越高越危险 |
| 精神压力 | 0% 到 -30% | 恐惧、幻听、梦境污染等 |
| 非凡特性烙印 | 0% 到 -35% | 使用特性替代主材料时出现 |

### 5.4 UI 展示

UI 不必直接显示精确百分比，可以使用区间：

| 显示 | 内部范围 |
| --- | --- |
| 稳定 | 80% 以上 |
| 可控 | 60%-79% |
| 危险 | 40%-59% |
| 极危 | 20%-39% |
| 近乎失控 | 20% 以下 |

高序列晋升建议隐藏精确数值，只展示风险描述、仪式缺口和污染预警。

## 6. 什么情况下失控

失控不是随机惩罚，而是玩家准备不足、污染累积或剧情选择的后果。

### 6.1 主要触发条件

| 条件 | 说明 |
| --- | --- |
| 未消化完全就晋升 | 当前序列消化度过低 |
| 使用错误配方 | 材料或途径不匹配 |
| 缺少核心材料 | 尝试用不合法材料替代 |
| 使用高污染非凡特性 | 特性中残留强烈精神烙印 |
| 晋升仪式失败 | 高序列尤其危险 |
| 精神状态崩溃 | 恐惧、低语、梦境污染、幻觉累积 |
| 被外力干扰 | 战斗中、被诅咒、仪式场被破坏 |
| 服食相邻或不相邻途径错误魔药 | 途径切换不合法或缺少设计许可 |
| 长期滥用封印物 | 污染值超过安全阈值 |

### 6.2 失控等级

| 等级 | 表现 | 玩法处理 |
| --- | --- | --- |
| 轻度失控 | 幻听、短暂僵直、灵性外泄 | 临时负面状态，可治疗 |
| 中度失控 | 能力暴走、敌我不分、污染外溢 | 触发战斗或救援任务 |
| 重度失控 | 身体畸变、精神被侵蚀、召来异常 | 进入特殊副本或濒死状态 |
| 完全失控 | 角色变成怪物或剧情死亡 | 作为重大失败结局或回档节点 |

### 6.3 失控预警

游戏应在真正失控前给玩家足够提示：

- 画面边缘出现污染视觉效果。
- 案件笔记出现异常文字。
- NPC 提醒玩家状态不稳定。
- 灵视看到自身气场异常。
- 灰雾或梦境中出现危险象征。
- 魔药调配 UI 显示“极危”或“近乎失控”。

### 6.4 失控与污染数值系统

失控与污染系统是魔药晋升的底层状态机。它不应该只在晋升时结算，而要贯穿探索、调查、战斗、占卜、阅读神秘知识、使用封印物和扮演法全过程。

本节用于把“是否失控”拆成可被系统读取的六个核心数值：理智值、污染值、灵性值、锚点值、消化度、神秘知识承受度。第 10 章会继续说明消化度如何通过扮演法提升；本节只定义这些数值如何参与失控、污染和晋升判定。

#### 6.4.1 核心数值总表

| 数值 | 建议字段 | 区间 | 方向 | 主要作用 |
| --- | --- | --- | --- | --- |
| 理智值 | `sanity` | 0-100 | 越高越稳定 | 判断精神稳定、幻觉抵抗、低语抵抗和失控惩罚 |
| 污染值 | `pollution` | 0-100 | 越低越安全 | 记录神秘污染、封印物副作用、错误知识和高污染材料影响 |
| 灵性值 | `spirituality` / `spirituality_max` | 0-上限 | 当前资源 | 支付占卜、灵视、仪式、技能、调配和探索消耗 |
| 锚点值 | `anchor` | 0-100 | 越高越稳定 | 维持自我认知、身份连续性和高序列精神稳定 |
| 消化度 | `digestion` | 0-100 | 越高越安全 | 判断当前序列魔药是否被真正掌握，并修正晋升风险 |
| 神秘知识承受度 | `occult_knowledge_tolerance` | 0-100 | 越高越能承受 | 判断玩家能否阅读、理解或接触高位格神秘知识 |

#### 6.4.2 理智值

理智值代表角色当下的精神清醒程度。它不是生命值，而是角色是否还能正确理解现实、区分幻觉、抵抗低语和执行仪式的能力。

| 区间 | 状态 | 玩法表现 |
| --- | --- | --- |
| 80-100 | 稳定 | 调查、占卜、仪式正常进行 |
| 60-79 | 紧绷 | 偶发低语或轻微画面噪点 |
| 40-59 | 不安定 | 占卜结果可能出现干扰，NPC 可提醒状态异常 |
| 20-39 | 危险 | 触发失控检定，战斗和仪式失误率提高 |
| 0-19 | 崩溃 | 进入强制异常事件、短暂失控或救援任务 |

理智值下降来源：

- 目击不可理解的神秘事件。
- 接触高污染材料、封印物或非凡特性。
- 在低灵性状态下强行使用能力。
- 晋升仪式失败、占卜反噬、梦境污染。
- 长时间处于灵界、灰雾或异空间。

理智值恢复方式：

- 在安全屋休息。
- 完成稳定锚点的社交互动。
- 使用镇定药剂、净化仪式或教会治疗。
- 结束污染源接触并等待自然恢复。
- 完成与自身身份相关的回忆、日记或关键任务。

#### 6.4.3 污染值

污染值代表角色被神秘力量侵蚀的程度。污染越高，越容易出现幻觉、身体异变、能力暴走和晋升失败。

| 区间 | 状态 | 玩法表现 |
| --- | --- | --- |
| 0-19 | 清洁 | 无额外惩罚 |
| 20-39 | 轻微污染 | UI 出现轻微异常提示，部分 NPC 能察觉气息 |
| 40-59 | 明显污染 | 失控风险上升，灵视、梦境和占卜更容易被干扰 |
| 60-79 | 高危污染 | 禁止安全晋升，封印物副作用增强 |
| 80-100 | 临界污染 | 进入污染事件链，可能强制触发失控或净化任务 |

污染值上升来源：

- 使用污染材料或残留精神烙印强的非凡特性。
- 长期佩戴或滥用封印物。
- 阅读超出承受度的神秘知识。
- 仪式失败、错误配方、错误途径魔药。
- 进入污染区域、灵界裂隙、灰雾异常区。

污染值降低方式：

- 完成净化仪式或教会治疗。
- 使用低风险净化药剂。
- 交出、封存或远离污染源。
- 完成锚点强化任务。
- 在安全区域长期休养。

#### 6.4.4 灵性值

灵性值是非凡能力、占卜、灵视、仪式和魔药调配的主要消耗资源。它应当像法力值一样被玩家理解，但在诡秘风格中，灵性耗尽不是“不能施法”这么简单，而是会让角色更容易被污染和低语侵入。

| 状态 | 判定 | 玩法表现 |
| --- | --- | --- |
| 过载 | 当前灵性高于上限 | 能力短暂增强，但污染与神秘知识冲击会追加理智损失 |
| 充足 | 当前灵性高于 60% | 技能、占卜和仪式稳定 |
| 偏低 | 当前灵性 30%-60% | 高消耗行为显示黄色风险提示 |
| 枯竭 | 当前灵性 1%-29% | 理智恢复变慢，污染抗性下降 |
| 透支 | 当前灵性小于等于 0 | 立即扣除理智值，并增加失控风险 |

MVP 中可以先把灵性值当作当前资源；当仪式、封印物或特殊场景让灵性短暂超过 `spirituality_max` 时，再进入灵性过载状态。过载适合提供高收益，但必须放大污染、幻听、占卜反噬和知识超载风险。

灵性值消耗来源：

- 主动技能、灵视、占卜和战斗能力。
- 魔药调配、仪式布置和灵界探索。
- 抵抗精神攻击、污染侵蚀或梦境入侵。

灵性值恢复方式：

- 休息、冥想、安全屋恢复。
- 使用恢复灵性的药剂或仪式道具。
- 完成与途径特性一致的稳定扮演。
- 借助特定 NPC、教会设施或神秘学地点恢复。

#### 6.4.5 锚点值

锚点值代表角色对“我是谁”的稳定认知。它来自真实姓名、过往记忆、可信关系、组织身份、日记记录和玩家长期选择。锚点值尤其适合约束无面人、梦境、寄生、长期伪装和高序列晋升。

| 区间 | 状态 | 玩法表现 |
| --- | --- | --- |
| 70-100 | 稳固 | 身份稳定，失控修正较低 |
| 40-69 | 摇晃 | 伪装、梦境和精神污染更容易造成干扰 |
| 20-39 | 裂痕 | 长期扮演可能导致自我认知下降 |
| 0-19 | 崩塌 | 触发身份危机、记忆错乱或强制回归锚点任务 |

锚点值下降来源：

- 长期扮演他人身份。
- 频繁更换人格、身份、阵营或记忆叙事。
- 被梦境、寄生、幻觉或高位格存在影响。
- 过度使用与自我认知冲突的封印物。
- 连续完成违背角色核心关系的剧情选择。

锚点值恢复方式：

- 与重要 NPC 建立或修复关系。
- 回到安全屋、据点或个人房间。
- 阅读日记、确认真实姓名、整理案件记录。
- 完成个人剧情、组织认证或信仰仪式。
- 暂停伪装，使用本来身份完成关键任务。

#### 6.4.6 消化度

消化度代表角色对当前序列魔药的理解和掌控。消化度越高，晋升越安全；消化度不足时强行晋升，会显著提高失控风险。

| 区间 | 状态 | 晋升影响 |
| --- | --- | --- |
| 0-39 | 未掌握 | 不建议晋升，失控风险极高 |
| 40-69 | 初步适应 | 可进行低风险任务，但晋升惩罚明显 |
| 70-89 | 接近稳定 | 可准备仪式，仍需关键扮演节点 |
| 90-99 | 临界完成 | 允许高风险晋升尝试 |
| 100 | 完全消化 | 达成标准晋升条件 |

消化度提升来源：

- 按照当前序列的扮演原则完成任务。
- 在剧情、战斗、调查或社交中正确使用该序列能力。
- 触发“理解本质”的关键领悟节点。
- 完成与该序列职责一致的仪式、委托或案件。

错误扮演不会简单地“不给经验”，而应产生副作用：消化度增长降低，同时污染值、理智惩罚或锚点惩罚上升。过度扮演可以短期提高消化度，但会降低锚点值，适合做成高收益高风险选择。

#### 6.4.7 神秘知识承受度

神秘知识承受度代表角色能否安全理解高阶神秘信息。它不是普通学识，而是角色的大脑、灵性和位格能否承受“知道真相”这件事。

建议同时记录一个临时压力值：

| 字段 | 说明 |
| --- | --- |
| `occult_knowledge_tolerance` | 角色当前可承受的神秘知识强度 |
| `occult_knowledge_load` | 最近阅读、聆听、目击或占卜得到的知识压力 |

知识判定规则：

| 判定 | 结果 |
| --- | --- |
| 知识强度小于等于承受度 | 正常阅读或解锁情报 |
| 知识强度高于承受度 1-10 点 | 获得情报，但扣除理智值 |
| 知识强度高于承受度 11-25 点 | 获得部分情报，增加污染值并触发幻觉 |
| 知识强度高于承受度 26 点以上 | 阻断阅读，触发污染事件或失控检定 |

承受度提升方式：

- 序列提升。
- 完成导师教学、图书馆研究、教会培训。
- 通过白塔、隐者等知识型途径能力获得加成。
- 在安全仪式保护下阅读高阶资料。
- 分阶段解读同一份危险文献。

承受度下降或临时受损来源：

- 理智值过低。
- 污染值过高。
- 灵性透支。
- 被高位格存在注视、梦境入侵或错误占卜反噬。

#### 6.4.8 失控风险公式

失控风险可以被设计成一个 0-100 的结算值，用于晋升、仪式、封印物使用、灵界探索和高危知识阅读。

```text
失控风险 =
  基础事件风险
  + 污染修正
  + 理智惩罚
  + 灵性透支惩罚
  + 锚点惩罚
  + 消化不足惩罚
  + 神秘知识超载惩罚
  + 当前剧情或仪式修正
```

建议修正项：

| 条件 | 风险修正 |
| --- | --- |
| 污染值 40-59 | +10 |
| 污染值 60-79 | +25 |
| 污染值 80 以上 | +40 |
| 理智值 20-39 | +20 |
| 理智值 0-19 | +40 |
| 灵性低于 30% | +10 |
| 灵性透支 | +25 |
| 锚点值 20-39 | +15 |
| 锚点值 0-19 | +35 |
| 消化度低于 70 | +25 |
| 消化度 70-89 | +10 |
| 知识强度高于承受度 11 点以上 | +15 |
| 知识强度高于承受度 26 点以上 | +35 |

风险结果：

| 最终风险 | 结果 |
| --- | --- |
| 0-24 | 安全，正常结算 |
| 25-49 | 低危，给出 UI 提醒 |
| 50-69 | 中危，可能出现副作用 |
| 70-89 | 高危，触发失控检定或污染事件 |
| 90-100 | 极危，强制触发重大后果 |

#### 6.4.9 晋升前检查

玩家尝试晋升前，系统应检查以下条件：

| 检查项 | 推荐条件 | 不满足时 |
| --- | --- | --- |
| 消化度 | `digestion >= 100` | 可允许冒险晋升，但大幅提高失控风险 |
| 理智值 | `sanity >= 40` | 阻止安全晋升，并提示精神状态不稳定 |
| 污染值 | `pollution < 60` | 要求先净化或承受高危晋升 |
| 灵性值 | 当前灵性高于 30% | 仪式中断、调配失败或进入透支 |
| 锚点值 | `anchor >= 40` | 高序列、长期伪装或精神系晋升必须先稳固锚点 |
| 知识承受度 | 足以理解配方和仪式 | 配方说明变成不可读、错误解读或污染反噬 |

#### 6.4.10 状态联动示例

| 行为 | 理智 | 污染 | 灵性 | 锚点 | 消化度 | 知识承受度 |
| --- | --- | --- | --- | --- | --- | --- |
| 正确完成占卜调查 | 小幅恢复 | 不变 | 消耗 | 不变 | 提升 | 小幅提升 |
| 错误占卜高位格对象 | 下降 | 上升 | 大量消耗 | 不变 | 不变 | 临时下降 |
| 长期扮演他人身份 | 不变或下降 | 不变 | 轻微消耗 | 下降 | 提升 | 不变 |
| 使用污染非凡特性替代主材料 | 下降 | 大幅上升 | 消耗 | 可能下降 | 不变 | 不变 |
| 在安全仪式中阅读危险文献 | 轻微下降 | 轻微上升 | 消耗 | 不变 | 不变 | 提升 |
| 灵性透支后继续战斗 | 下降 | 上升 | 透支 | 下降 | 不变 | 临时下降 |

#### 6.4.11 MVP 实现建议

第一版不需要做成非常复杂的模拟系统，建议先实现以下内容：

- 角色状态面板显示六个核心数值。
- 每次高危行为只修改 1-3 个相关数值，避免玩家难以理解。
- 晋升、调配、占卜、灵视、封印物使用都调用同一套失控风险公式。
- 理智低、污染高、灵性透支、锚点低时，优先通过 UI、音效、文本和 NPC 提醒玩家。
- 污染值达到 60 后禁止“安全晋升”，但允许玩家选择高危晋升路线。
- 消化度达到 100 是标准晋升条件；90-99 可以作为剧情中特殊冒险选择。
- 神秘知识承受度先用于文献阅读、配方解读和高位格情报解锁，不急着接入所有对话。

## 7. 晋升失败有什么后果

失败后果应分层，避免所有失败都直接死亡。

| 失败类型 | 后果 |
| --- | --- |
| 调配失败 | 损失部分材料，产生污染残渣或异常气味 |
| 服食失败但未失控 | 进入虚弱、幻听、灵性紊乱状态 |
| 晋升中断 | 魔药效果残留，需要任务清除 |
| 仪式失败 | 触发对应仪式反噬、区域事件或 NPC 死亡 |
| 部分失控 | 进入战斗、逃脱、救援或净化任务 |
| 完全失控 | 角色死亡、怪物化、重要剧情失败或读取关键存档 |

### 7.1 可恢复失败

低序列和部分中序列失败应可恢复：

- 消耗治疗道具。
- 寻找导师或官方组织处理。
- 完成净化任务。
- 在安全屋休养。
- 用灰雾、占卜或仪式清除污染。

### 7.2 不可轻易恢复失败

高序列失败应有长期代价：

- 永久增加污染上限或精神裂痕。
- 失去部分材料或非凡特性。
- 重要 NPC 受伤、死亡或背叛。
- 区域进入灾难状态。
- 解锁坏结局分支。

## 8. 晋升仪式怎么做成任务

晋升仪式应被设计成任务链，而不是 UI 中的一行勾选。

### 8.1 仪式任务结构

```text
获得仪式情报
  ↓
解析仪式条件
  ↓
准备地点 / 物品 / 见证者 / 时间
  ↓
完成前置行动
  ↓
执行仪式场景
  ↓
处理干扰与反噬
  ↓
服食魔药
```

### 8.2 仪式任务类型

| 类型 | 适用场景 | 示例玩法 |
| --- | --- | --- |
| 地点型 | 必须在特定地点晋升 | 前往遗迹、灵界坐标、海域、教堂或战场 |
| 行为型 | 必须完成某种象征行为 | 守护、欺骗、审判、狩猎、治疗、见证 |
| 社会型 | 必须影响一定数量 NPC 或组织 | 获得名声、制造事件、完成公开身份行动 |
| 战斗型 | 必须击败或承受特定敌人 | Boss 战、守点战、追猎战 |
| 调查型 | 必须理解某个秘密或真相 | 案件推理、档案解读、梦境回溯 |
| 时间型 | 必须在特定时间窗口完成 | 满月、黄昏、风暴夜、灾难发生时 |
| 代价型 | 必须牺牲资源、身份或关系 | 放弃身份、失去声望、付出珍贵材料 |

### 8.3 任务拆分规则

每个仪式任务至少包含：

- 一个明确目标。
- 一个准备阶段。
- 一个风险提示。
- 一个可失败点。
- 一个晋升场景。
- 一个结果反馈。

高序列仪式应拆成 3 到 5 个任务节点，并和区域主线或大型支线绑定。

### 8.4 数据字段联动

`rituals.json` 中的字段可这样使用：

| 字段 | 游戏用途 |
| --- | --- |
| `ritual_id` | 任务链 ID 或任务目标引用 |
| `required_items` | 仪式任务必须交付的物品 |
| `required_location` | 解锁对应地图或场景 |
| `required_npc_or_condition` | NPC、阵营、时间或世界状态要求 |
| `success_condition` | 任务完成判断 |
| `failure_result` | 失败后果 |
| `loss_control_risk_change` | 晋升 UI 的风险变化 |
| `quest_design_note` | 任务设计备注 |

### 8.5 仪式任务模板

每个重要晋升仪式可以按下面模板落成任务文档或任务数据：

```markdown
## 晋升仪式任务：[任务名称]

**目标序列**：
**关联途径**：
**任务目的**：
**玩家体验目标**：
**前置条件**：
**关键地点**：
**关键 NPC / 敌人**：
**必需道具**：
**禁忌条件**：
**成功条件**：
**失败条件**：
**可选目标**：
**完成度评分规则**：
**成功率修正**：
**失控率修正**：
**失败后恢复路径**：
```

仪式任务的完成度不一定只有成功和失败。建议输出“完美完成、完成、勉强完成、中断、失败”五档，用于修正最终晋升成功率和失控风险。

## 9. 非凡特性能否替代主材料

可以，但必须是高风险替代路线。

### 9.1 基本规则

非凡特性可以替代对应途径、对应序列的主材料，但不能随意替代辅助材料和仪式条件。

合法替代必须满足：

- `characteristic.pathway_id` 与目标途径一致，或属于系统明确允许的相邻途径切换。
- `characteristic.sequence_number` 与目标序列一致。
- 目标材料的 `can_be_replaced_by_characteristic` 为 `true`。
- 玩家拥有保存完好的非凡特性。
- 玩家知道直接使用特性的风险。

### 9.2 替代效果

| 项目 | 普通主材料 | 非凡特性替代 |
| --- | --- | --- |
| 获取难度 | 可通过任务、交易、采集获得 | 通常来自死亡、封印物事件或高危战斗 |
| 成功率 | 稳定 | 基础成功率下降 |
| 污染风险 | 正常 | 明显提高 |
| 精神烙印 | 低 | 可能残留前任人格、执念或污染 |
| 后续剧情 | 较少 | 可触发梦境、幻听、追踪或身份问题 |

### 9.3 风险修正

使用非凡特性替代主材料时，建议应用：

```text
成功率 -10% 到 -35%
污染值 +10 到 +40
失控风险 +1 档
可能触发精神烙印事件
```

风险大小由 `characteristics.json` 中的 `risk_level` 和 `mental_imprint` 决定。

### 9.4 什么时候推荐允许替代

- 玩家错过了某个材料获取路线，需要备用方案。
- 剧情中击败了对应非凡者，获得了特性。
- 高风险路线能带来更强剧情张力。
- 玩家愿意牺牲安全性换取更快晋升。

### 9.5 什么时候禁止替代

- 目标序列仪式明确要求某种象征性材料。
- 非凡特性被严重污染或封印不完整。
- 途径不相邻，且没有特殊剧情许可。
- 当前角色精神状态已经接近失控。
- 序列 0 级别终局晋升需要专门剧情系统处理。

## 10. 扮演法与消化

晋升成功后，玩家不是立刻进入下一次晋升，而是进入消化阶段。

### 10.1 消化来源

| 来源 | 说明 |
| --- | --- |
| 主线行为 | 按当前序列定位完成关键剧情 |
| 支线任务 | 使用当前序列能力解决问题 |
| 日常扮演 | 在对话、调查、战斗中符合序列定位 |
| 风险克制 | 不滥用能力，不越界接触高位污染 |
| 组织反馈 | 获得导师、队友或官方组织认可 |

### 10.2 消化度用途

消化度影响：

- 下一次晋升成功率。
- 当前能力稳定性。
- 失控预警频率。
- NPC 对玩家状态的判断。
- 是否解锁高序列仪式线索。

### 10.3 扮演法系统

扮演法用于连接“角色行为”和“魔药消化”。玩家不是通过单纯刷经验晋升，而是通过理解当前序列的定位，并在世界中做出符合该定位的行为，逐步消化魔药。

一次行为是否增加消化度，必须同时满足三个条件：

- 行为和当前序列主题一致。
- 行为解决了真实任务问题，而不是无意义刷动作。
- 玩家在扮演中保持自我认知，没有被序列角色完全吞没。

推荐新增或在角色成长状态中维护以下变量：

| 变量 | 作用 |
| --- | --- |
| `digestion_progress` | 当前序列消化度，达到门槛后才建议晋升 |
| `acting_alignment` | 本次行为与序列主题的契合度 |
| `self_identity_stability` | 自我认知稳定值，身份伪装、梦境、寄生等玩法会重点影响 |
| `overacting_pressure` | 过度扮演压力，表示玩家是否为了扮演而偏离人性和任务目标 |
| `cognitive_bias` | 认知偏差，表示玩家是否误解了当前序列的核心定位 |
| `social_exposure` | 社会暴露值，表示 NPC 或组织是否察觉玩家异常 |
| `pollution_value` | 污染值，和失控风险联动 |
| `mental_pressure` | 精神压力，受恐惧、幻听、失败晋升、封印物副作用影响 |

### 10.4 消化度获取规则

| 行为类型 | 消化度建议 | 说明 |
| --- | --- | --- |
| 主线关键扮演 | 高 | 用当前序列方式解决章节核心问题 |
| 支线任务扮演 | 中 | 用当前序列能力完成委托、调查、救援、潜入或战斗 |
| 日常小行为 | 低 | 对话、探索、轻量能力使用，只能少量增长 |
| 完成仪式行为 | 高 | 和晋升仪式或序列象征强绑定 |
| 克制错误冲动 | 中 | 明明可以滥用能力但选择保持边界 |
| 错误扮演 | 负 | 行为只像名字表面，不符合序列本质 |
| 过度扮演 | 负 | 为了消化主动伤害无辜、制造灾难或丢失自我 |

建议结算公式：

```text
消化度变化 =
基础消化值
× 行为匹配系数
× 场景压力系数
× 结果反馈系数
× 新鲜度系数
× 稳定性修正
```

消化度不建议靠重复刷同一动作获得。同一地图、同一任务、同一行为类型需要有递减收益，鼓励玩家通过不同任务理解序列含义。

### 10.5 错误扮演与过度扮演

| 类型 | 说明 | 后果 |
| --- | --- | --- |
| 表面扮演 | 只模仿职业名称，不理解序列本质 | 不增加消化度，可能增加精神压力 |
| 功利扮演 | 为了刷消化度重复无意义动作 | 消化收益递减，触发导师或笔记提醒 |
| 过度扮演 | 把序列身份当成真正自我 | 降低 `self_identity_stability` |
| 违背人性 | 为了符合序列而主动伤害无辜 | 增加污染值和失控风险 |
| 错误象征 | 用相反方式理解序列，例如把守护理解成控制 | 增加 `overacting_pressure` |
| 强行跳层 | 低序列模仿高序列权柄行为 | 显著增加失控风险 |

当 `self_identity_stability` 过低时，玩家应收到明确反馈：镜中形象异常、梦境身份混乱、案件笔记出现陌生称呼、NPC 发现玩家“不像自己”。

### 10.6 UI 反馈

| UI 项 | 表现 | 说明 |
| --- | --- | --- |
| 魔药消化度 | 环形进度或瓶中液体减少 | 表示距离完全消化的进度 |
| 稳定度 | 心智纹路、裂痕或颜色变化 | 表示当前安全程度 |
| 扮演契合度 | 面具图标亮度 | 表示近期行为是否符合序列定位 |
| 失控风险 | 暗色噪点、红色脉冲、低语文本 | 风险越高，界面越不稳定 |
| 自我锚点 | 真名、日记、关系、私人物品图标 | 对身份伪装、梦境、寄生类序列尤其重要 |

关键行为前可以给出倾向提示，但不要把公式完全暴露给玩家：

```text
【占卜】这符合你当前的扮演。
预计收益：中
精神压力：低
认知偏差：无
```

```text
【无视预兆，强行行动】这与你当前序列存在冲突。
预计收益：无
认知偏差：上升
失控风险：轻微上升
```

占卜家的关键领悟是：占卜不是答案，而是提问的方式。小丑的关键领悟是：小丑不是没有恐惧，而是在恐惧面前仍能控制表情。无面人的关键领悟是：无面人可以成为任何人，但必须知道何时不是任何人。

无面人及类似身份类序列需要额外使用“自我锚点”：真名、固定关系、私人物品、日记记录或只有真实身份才保留的行为习惯。锚点不足时，伪装收益可以提高，但所有伪装行为都应额外增加人格侵蚀。

## 11. MVP 实现范围

第一版建议先实现以下内容：

1. 支持序列 9 到序列 7 的魔药调配和服食。
2. 背包中区分主材料、辅助材料、非凡特性和仪式道具。
3. 魔药调配 UI 显示材料是否齐全、风险等级和调配结果。
4. 晋升成功后解锁 `abilities.json` 中对应技能。
5. 引入污染值、精神压力和消化度三个核心变量。
6. 低序列失败以可恢复后果为主。
7. 至少做一个完整晋升任务链，作为后续模板。

暂时不做：

- 序列 2 到序列 0 的完整终局晋升模拟。
- 所有途径的独立仪式演出。
- 精确到每个材料的复杂炼金小游戏。
- 大规模随机掉落刷装系统。

## 12. 开发数据建议

后续可以新增一个角色成长状态表：

```ts
interface PlayerAdvancementState {
  current_pathway_id: string;
  current_sequence_number: number;
  digestion_progress: number;
  pollution_value: number;
  mental_pressure: number;
  known_recipes: string[];
  completed_rituals: string[];
  advancement_history: AdvancementRecord[];
}

interface AdvancementRecord {
  sequence_id: string;
  result: "success" | "normal_failure" | "severe_failure" | "pollution_failure" | "loss_control" | "death";
  used_materials: string[];
  used_characteristic_id?: string;
  ritual_id?: string;
  timestamp: string;
}
```

魔药调配结果可记录为：

```ts
interface BrewedPotion {
  potion_id: string;
  sequence_id: string;
  quality: "perfect" | "normal" | "flawed" | "unstable";
  success_rate_modifier: number;
  pollution_modifier: number;
  used_materials: string[];
  used_characteristic_id?: string;
}
```

材料品质可以补充为：

```ts
type MaterialQuality = "polluted" | "normal" | "pure" | "compatible";

interface AdvancementMaterialInstance {

  pollution_value: number;
  compatibility_tags: string[];
  source_type: "quest" | "shop" | "black_market" | "field_gathering" | "enemy_drop" | "artifact_event";
}
```

## 13. 总结

魔药晋升系统的核心不是“集齐材料后升级”，而是“玩家为了变强，必须理解自己正在接触什么、准备什么、承担什么”。材料收集提供探索目标，调配提供风险判断，晋升仪式提供任务结构，失控提供代价，非凡特性替代提供高风险捷径。

只要这个系统成立，调查、战斗、黑市、组织、地图探索和剧情选择都会自然汇入角色成长，成为整个 RPG 的主干循环。
