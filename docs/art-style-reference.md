# 美术风格设定文档

> 文档用途：统一项目的整体美术方向、区域视觉关键词、UI 元素、色彩、材质、光照、角色、怪物和特效参考。  
> 适用范围：概念设计、场景设计、UI 设计、角色设计、怪物设计、图标设计、特效设计、宣传图和后续美术资源制作。  
> 当前目标：先服务第一卷“雾城疑案”和后续区域扩展，避免每个系统各自长出不一致的视觉语言。  
> 注意：若项目后续公开发布或商业化，应将强识别名词、符号和具体设定做原创化替换。

## 1. 整体风格

整体风格：维多利亚蒸汽朋克 + 神秘学 + 克苏鲁氛围。

核心概念：工业理性被不可名状之物侵蚀。

表层是维多利亚晚期的城市秩序、蒸汽机械、礼仪与阶级；中层是神秘学符号、禁书、仪式、炼金术与灵媒技术；底层是克苏鲁式宇宙恐怖，即人类文明面对非人尺度时的崩塌感。

视觉基调应是“精密、腐蚀、压抑、神圣而不洁”。世界曾经辉煌，但建筑、机械、服饰和 UI 都正在被灰雾、盐蚀、煤灰、灵质与异星污染慢慢吞没。

这三个方向的比例建议：

| 风格来源 | 占比 | 作用 |
| --- | --- | --- |
| 维多利亚蒸汽朋克 | 45% | 提供真实世界质感：城市、服装、机械、报纸、煤烟、阶层差异 |
| 神秘学 | 35% | 提供玩法识别：仪式、星象、塔罗牌、封印符号、魔药、占卜 |
| 克苏鲁氛围 | 20% | 提供未知恐惧：不可直视、污染、低语、畸变、空间异常 |

美术基调不是夸张奇幻，而是“现实城市中有无法解释的裂缝”。玩家第一眼看到的是煤气灯、钟楼、石板路和湿冷街巷；第二眼才发现墙角符号、镜中倒影、雾里的非人轮廓和被污染的文字。

## 5. 字体与排版

| 场景 | 字体方向 |
| --- | --- |
| 标题 | 有书籍感、报纸感的宋体或衬线风格 |
| 正文 | 清晰易读的黑体或现代正文体 |
| 档案编号 | 等宽字体或打字机风格 |
| 手写批注 | 少量使用手写风格，不能影响阅读 |
| 仪式文字 | 符号化、几何化、不可完全读懂 |

排版原则：

- 任务日志像档案，不像现代聊天软件。
- 调查笔记像真实案件板，有线索卡和连线。
- 魔药调配像实验记录和仪式台，不像普通合成菜单。
- 秘密会议像神秘仪式空间，不像普通多人大厅。
- 封印物管理像收容档案，不像普通装备仓库。

## 8. 重点区域设定

### 8.1 雾都

关键词：维多利亚、煤气灯、雨夜、黑雾。

雾都是第一卷的主舞台。它必须有强烈的城市调查感：街角报童、湿冷巷道、警局公告、钟声、马车、煤烟和若隐若现的教堂尖顶。玩家应该感觉这座城市是活的，但每条街都藏着秘密。

视觉要点：

- 远景建筑被灰雾吞没，只露出钟楼、烟囱和尖顶。
- 路面常有积水，反射煤气灯。
- 普通街区暖黄灯光，污染区域灯光发红或发绿。
- 案件现场比周围更安静，使用更少环境 NPC。
- 黑雾不直接铺满画面，而是在巷口、窗后、镜中出现。

### 8.2 海上

关键词：风暴、灯塔、幽灵船、热带港口。

海上区域从“城市调查”转向“冒险和航线风险”。视觉上要更开阔，但仍保留未知感。

视觉要点：

- 天空云层低压，远处有闪电和风暴墙。
- 灯塔是安全点，也是危险预告。
- 幽灵船不应一开始完整出现，先通过船灯、桅杆影子、破碎歌声暗示。
- 热带港口可提高色彩饱和度，但仍保持潮湿、危险和殖民地混杂感。
- 海盗势力使用黑帆、盐痕、旧火枪、船舱地图和私掠印章。

### 8.3 灵界

关键词：破碎空间、彩色雾气、漂浮道路。

灵界不是普通异世界，而是现实规则松动后的区域。它应让玩家感到美丽、不稳定、危险。

视觉要点：

- 建筑碎片漂浮，方向不完全一致。
- 道路可以像玻璃、雾、星图或记忆碎片。
- 彩色雾气可以出现，但不要做成明亮童话色，必须带不安感。
- 远处生物轮廓不完整，像符号、骨架、剪影或旧照片残影。
- UI 在灵界中可出现轻微错位和文字漂移。

### 8.4 神弃之地

关键词：永夜、火光、废墟、巨人遗迹。

神弃之地是高压、生存和古老残骸的区域。这里不是“黑色地图”，而是以火光、遗迹尺度和孤独感制造压迫。

视觉要点：

- 大面积暗色中保留少量火光和红色余烬。
- 巨人遗迹要体现尺度差：门、柱、阶梯都远大于人类。
- 天空没有正常日夜感，像被永夜压住。
- 角色队伍显得很小，画面留出巨大的黑暗空间。
- 怪物轮廓粗重、古老、像从废墟和阴影中长出。

### 8.5 灰雾空间

关键词：青铜长桌、星辰、灰白迷雾。

灰雾空间是秘密会议和高层神秘感的核心视觉。它必须安静、庄重、超现实。

视觉要点：

- 中心是青铜长桌或类似仪式桌，具有强烈构图秩序。
- 周围不是普通房间，而是灰白迷雾和遥远星辰。
- 角色座位可以用剪影、卡牌、星点或徽记表示。
- 光源来自桌面、星辰和雾中不可见处。
- UI 使用极少颜色，以灰白、暗金、青铜为主。

## 9. 角色服装风格

| 类型 | 服装方向 |
| --- | --- |
| 官方组织 | 深色长风衣、皮革手套、徽章、档案袋、黄铜怀表 |
| 教会人员 | 深色或骨白长袍、圣徽、钟形饰物、净化手套 |
| 贵族 | 暗金刺绣、丝绒、手杖、礼帽、面纱、珠宝 |
| 普通市民 | 粗呢外套、围巾、旧帽子、工装、补丁、煤灰 |
| 黑市商人 | 半遮脸、深色围巾、多口袋外套、暗号饰品 |
| 海盗 | 防水长衣、盐痕皮革、旧枪、刀、航海符物 |
| 邪教徒 | 普通衣服外叠加面具、蜡烛、红线、反常洁净感 |
| 非凡者 | 不必全都华丽，应通过小道具和气质区分能力路线 |

角色设计原则：

- 低序列角色应像普通人，只带少量异常细节。
- 高危角色可以有明显反常点，例如影子不对、眼神空洞、衣服过分整洁。
- 不要让所有神秘角色都穿长袍，容易单调。
- 服装要体现阶层：贵族、工人、官方、黑市、教会必须一眼不同。

## 10. 怪物与污染视觉

污染不是简单“长触手”，而是认知、身体和环境同时出现错误。

| 类型 | 视觉方向 |
| --- | --- |
| 轻度污染 | 眼角黑纹、手指颤抖、文字错位、气场浑浊 |
| 中度污染 | 肢体比例异常、影子延迟、皮肤出现符号 |
| 重度污染 | 人体结构破裂、器官符号化、局部非人化 |
| 完全失控 | 人形只保留少量痕迹，动作和声音不再符合人类 |
| 梦境污染 | 重复面孔、错位房间、温柔但不合理的光 |
| 灵界污染 | 身体边缘碎裂、半透明、星雾和骨白线条 |
| 邪教污染 | 蜡泪、面具、红线、倒置祷文、舞台感 |

禁止事项：

- 不要把所有污染都做成触手怪。
- 不要大面积使用鲜艳荧光色。
- 不要让怪物一眼完全解释清楚。
- 不要只靠血腥表现恐怖，更应靠“不对劲”。

| --- | --- |
| 黄铜 | 仪器、齿轮、边框、机械装置 |
| 黑铁 | 封印门、收容柜、武器、地下设施 |
| 湿石 | 街道、教堂、下水道、港口 |
| 旧纸 | 档案、配方、地图、调查笔记 |
| 皮革 | 风衣、手套、书本、箱包 |
| 红丝绒 | 贵族、剧院、邪教仪式 |
| 银箔 | 封印物、净化仪式、防污染容器 |
| 煤灰 | 工厂、下城区、普通人生活痕迹 |

纹理要求：

- 表面要有旧化和使用痕迹。
- 金属不能太新，黄铜和铜绿要有氧化层。
- 纸张要有折痕、污渍、墨迹，但正文区域保持可读。
- 封印物材质要让人感觉“被保管很久”，不是普通魔法装备。

## 12. 光照与天气

| 场景 | 光照方向 |
| --- | --- |
| 雾都街道 | 低对比、煤气灯暖光、远处冷灰雾 |
| 案件现场 | 局部硬光、黑色阴影、线索点微弱高亮 |
| 教堂 | 高窗光束、尘埃、冷白和暗金 |
| 黑市 | 低灯、局部红光或铜绿光、边缘黑暗 |
| 工厂 | 炉火、蒸汽遮挡、强背光剪影 |
| 海港 | 潮湿反光、灯塔扫光、风暴闪电 |
| 灵界 | 无固定光源、漂浮光点、局部颜色异常 |
| 灰雾空间 | 桌面微光、星光、雾中漫反射 |

天气建议：

- 雨夜用于案件和追踪。
- 大雾用于未知和封锁。
- 细雪或灰烬用于高压区域。
- 风暴用于海上和暴君类能力。
- 反常晴天可用于“表面正常但内里危险”的贵族区。

## 13. 图标风格

图标应以线刻、版画、旧报纸插图、仪式符号为主要方向。

| 图标类型 | 视觉方向 |
| --- | --- |
| 技能图标 | 单色或双色线刻，带途径符号 |
| 材料图标 | 小型静物画，强调瓶、粉末、器官、草药 |
| 封印物图标 | 编号牌 + 物品剪影 + 风险角标 |
| 任务图标 | 档案夹、蜡封、羽毛笔、放大镜 |
| 地图图标 | 旧地图符号，少量黄铜边框 |
| 污染图标 | 黑雾、裂纹、红线、眼形但不直白 |
| 理智图标 | 烛火、心智轮廓、闭眼、锚点 |
| 势力图标 | 徽章、印章、家徽、旗帜、暗号 |

图标不建议做成现代扁平彩色风；应更像“刻在旧档案上的符号”。

UI 可读性要求：

- 普通正文必须优先保证阅读，不要直接压在高噪点纹理上。
- 灵异发光色不能承载大段文字，只适合用于边缘光、状态提示、符文激活和危险反馈。
- 装饰线、铆钉、齿轮、符文只在结构处出现，不能把每个按钮都做成复杂装饰品。
- 玩家会长时间阅读任务、线索和材料说明，界面必须清晰、稳定、可扫描。

## 14. UI 页面视觉方向

| 页面 | 美术方向 |
| --- | --- |
| 主菜单 | 雾中城市远景，钟楼和煤气灯，少量暗金标题 |
| 角色面板 | 黑木档案桌，角色证件照，属性像档案字段 |
| 途径面板 | 星象盘 + 卡牌路径，序列像可翻阅的牌位 |
| 魔药调配界面 | 实验台、玻璃瓶、羊皮纸配方、风险仪表 |
| 材料背包 | 旧皮箱或档案柜，物品分类像铜质标签 |
| 任务日志 | 案件档案夹，任务像卷宗页 |
| 地图界面 | 旧城市地图，雾区遮挡未探索区域 |
| 调查笔记 | 案件板、线索卡、红线、手写批注 |
| 秘密会议界面 | 青铜长桌、灰白迷雾、星辰席位 |
| 封印物管理界面 | 收容柜、编号牌、封条、危险等级灯 |
| 污染与理智状态界面 | 心智仪表、裂纹、低语文字、锚点符号 |

## 15. 特效方向

| 特效 | 表现 |
| --- | --- |
| 灵视 | 冷青轮廓、空气微粒、边缘光，不要全屏变色过重 |
| 占卜 | 星点、细线、卡牌翻面、墨迹自动书写 |
| 魔药调配 | 液体颜色变化、气泡、瓶壁符号、风险闪烁 |
| 晋升 | 画面压缩、低语字幕、符号闪回、锚点光 |
| 净化 | 暗金或冷白光，污染像灰烬或黑雾散开 |
| 污染 | 噪点、红黑裂纹、文字错位、声音延迟 |
| 封印 | 银线、封条、几何符号闭合、锁链声 |
| 灵界传送 | 空间折叠、彩雾、道路碎片重组 |

特效原则：

- 低序列技能特效克制，像现实中出现异常。
- 高序列特效才逐步接近规则级和空间级变化。
- 污染特效要影响 UI、声音和环境，不只影响角色模型。

## 16. 美术禁区

不建议使用：

- 把蒸汽朋克简化成到处贴齿轮。
- 大面积纯紫蓝渐变作为神秘感。
- 过于干净明亮的奇幻魔法 UI。
- 所有区域都只用黑、红、金三种颜色。
- 现代科幻全息屏。
- 过度可爱的卡通比例。
- 大量血浆替代心理恐怖。
- 无意义的装饰圆球、渐变光斑和纯氛围背景。
- 所有神秘符号都画成同一种眼睛。
- 所有怪物都做成触手。
- 地图和 UI 过度花纹化导致信息不可读。

## 19. 总结

本项目的美术核心不是“华丽魔法”，而是“现实城市被神秘学悄悄污染”。

玩家应当先相信这是一座真实的蒸汽都市，然后逐渐发现：

```text
雾不只是天气
煤气灯照不到的地方藏着仪式
档案里的文字会变化
漂亮的贵族宅邸也可能是污染源
海港之外有更古老的恐惧
灰雾中的长桌不是梦
```

如果美术能始终保持“真实生活 + 隐秘裂缝 + 不可解释的代价”，项目的气质就会稳定。
