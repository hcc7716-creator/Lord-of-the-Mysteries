# MVP 最小可玩版本范围

> 文档用途：锁定第一版可玩的最小范围，防止开发初期被完整开放世界、22 条途径、220 个序列和大量剧情支线拖大。  
> 适用阶段：第一版 Demo / 原型验证 / 最小可玩版本。  
> 核心原则：先做一条完整闭环，再扩展更多城市、途径、序列、组织和开放世界内容。  
> 当前版本：v0.1  
> 创建日期：2026-07-08

## 1. MVP 目标

第一版只验证一件事：

玩家能否在一个雾气笼罩的蒸汽都市中，通过调查案件、收集材料、调配魔药、晋升序列、参与小规模战斗，并在失控风险压力下完成一条神秘死亡案件主线。

MVP 不追求完整开放世界，也不追求还原所有途径和所有组织。它要证明以下核心体验成立：

| 目标 | 说明 |
| --- | --- |
| 调查成立 | 玩家能通过线索、占卜、NPC 对话和现场勘查推进案件 |
| 成长成立 | 玩家能从普通人逐步晋升到低序列非凡者 |
| 风险成立 | 使用神秘学、调配魔药和战斗都会带来失控压力 |
| 城市成立 | 雾城具备可探索区域、关键 NPC、黑市和案件地点 |
| 战斗成立 | 玩家能使用基础技能与失控者、邪教徒或污染生物交战 |
| 文档闭环成立 | 调查笔记能记录线索、人物、地点、任务和推理结果 |

### 1.1 建议体验时长

| 项目 | MVP 建议 |
| --- | --- |
| 主线通关时长 | 3-5 小时 |
| 补充探索时长 | 5-8 小时 |
| 晋升节点 | 序列 9 占卜家 → 序列 8 小丑 → 序列 7 魔术师 |
| 城市规模 | 4-6 个功能明确的可探索区域 |
| 战斗规模 | 单场 1-3 名敌人，少量精英战斗和 1 个主线小 Boss |

## 3. 玩家体验闭环

MVP 的核心循环如下：

```text
接到案件
  ↓
探索雾城地点
  ↓
调查现场 / 询问 NPC / 收集线索
  ↓
使用占卜或灵视获得额外信息
  ↓
遭遇敌人或异常事件
  ↓
战斗 / 逃离 / 记录线索
  ↓
收集魔药材料
  ↓
调配魔药并承担失控风险
  ↓
晋升到新序列
  ↓
解锁新技能，推进案件真相
```

第一版结束时，玩家应完成一次完整的低序列成长体验：从接触神秘案件，到成为占卜家，再通过剧情和材料收集逐步解锁小丑、魔术师相关能力。

### 3.1 玩法节奏分层

| 层级 | 时间尺度 | 玩家行为 |
| --- | --- | --- |
| 瞬时循环 | 30 秒内 | 探索街区、检查物件、询问 NPC、获得线索或危险提示 |
| 单次游玩循环 | 15-30 分钟 | 接到案件目标，调查 1-2 个地点，收集材料，回到安全点整理笔记 |
| 长期循环 | 3-5 小时 | 推进死亡案件，完成两次晋升，最终处理真凶并解锁后续伏笔 |

## 4. 主角范围

第一版只做 1 个主角，不做多角色切换。

| 项目 | 范围 |
| --- | --- |
| 身份 | 雾城普通居民 / 见习调查员 / 被卷入案件的人 |
| 初始状态 | 尚未成为非凡者，但具备接触神秘学的契机 |
| 成长方向 | 占卜家 / 愚者途径 |
| 核心数值 | 生命值、灵性值、理智值、污染值、消化度、失控风险 |
| 装备范围 | 普通武器、调查工具、纸人、灵摆、魔药材料、少量消耗品 |
| 不做内容 | 多主角、职业自由选择、完整捏脸、复杂社交养成 |

## 5. 途径与序列范围

第一版只开放占卜家 / 愚者途径，并只做到序列 7 魔术师。

| 序列 | 中文名 | 英文名 | MVP 作用 |
| --- | --- | --- | --- |
| 序列 9 | 占卜家 | Seer | 新手调查、线索判断、灵性入门 |
| 序列 8 | 小丑 | Clown | 闪避、误导、危险预判、战斗生存 |
| 序列 7 | 魔术师 | Magician | 纸人替身、火焰跳跃、魔术戏法、战斗与解谜升级 |

### 5.1 占卜家

占卜家是第一版的核心入门序列，主要服务调查玩法。

| 能力方向 | MVP 表现 |
| --- | --- |
| 灵视 | 观察异常气息、灵体残留、污染痕迹 |
| 灵摆占卜 | 判断物品、地点或 NPC 是否与案件有关 |
| 梦境启示 | 在休息或特殊仪式后获得模糊线索 |
| 占卜失败 | 若目标涉及高位存在或污染源，污染值上升 |

### 5.2 小丑

小丑用于把调查玩法推进到战斗和潜入。

| 能力方向 | MVP 表现 |
| --- | --- |
| 危险预感 | 提前提示陷阱、伏击或致命攻击 |
| 身体协调 | 提升闪避、翻越、攀爬、近战机动 |
| 表情控制 | 在对话中隐藏情绪，提高欺骗或试探成功率 |
| 误导动作 | 战斗中短暂扰乱敌人判断 |

### 5.3 魔术师

魔术师是 MVP 的阶段性成长终点，提供更鲜明的战斗和解谜能力。

| 能力方向 | MVP 表现 |
| --- | --- |
| 纸人替身 | 抵挡一次致命伤害或关键攻击 |
| 火焰跳跃 | 在短距离内位移，可用于战斗脱离和地图机关 |
| 魔术戏法 | 制造烟雾、假动作、视线遮挡或道具转移 |
| 表演式施法 | 技能更强，但连续使用会提高失控风险 |

## 6. 城市范围：雾城 / 廷根原型

第一版只做 1 个城市，不做跨城旅行。

| 区域 | 功能 |
| --- | --- |
| 主角住所 | 存档、休息、整理调查笔记、触发梦境线索 |
| 警署 / 官方组织据点 | 接任务、汇报案件、获得基础调查权限 |
| 案发街区 | 第一主线核心调查区 |
| 教堂 / 礼拜场所 | 神秘学信息、理智恢复、部分 NPC 线索 |
| 地下黑市 | 购买材料、交换情报、接触灰色 NPC |
| 废弃工厂 / 仓库 | 战斗、邪教线索、污染源 |
| 墓园 / 偏僻旧区 | 灵体痕迹、仪式现场、晋升材料线索 |

### 6.1 城市规模控制

雾城不需要一开始做成完整开放世界。MVP 可以采用“区域节点 + 小地图”的方式：

| 地图类型 | 数量建议 |
| --- | --- |
| 可自由移动街区 | 2-3 个 |
| 室内调查场景 | 4-6 个 |
| 战斗场景 | 3-5 个 |
| 安全据点 | 2 个 |
| 黑市 / 特殊交易点 | 1 个 |

城市开放节奏建议：初始只开放主角住所、官方据点和案发街区，随着主线推进再解锁黑市、墓园、废弃工厂等高风险区域。这样可以保持探索感，同时避免第一版地图过早失控。

## 7. 主线范围：神秘死亡案件

第一版只做 1 条主线，主题为“神秘死亡案件”。

### 7.1 主线结构

| 阶段 | 内容 |
| --- | --- |
| 第一幕：卷入案件 | 玩家醒来或被召唤到一场离奇死亡现场 |
| 第二幕：接触官方 | 玩家接触官方非凡组织，获得基础调查权限 |
| 第三幕：学习神秘学 | 玩家学习灵视、占卜、仪式和调查笔记 |
| 第四幕：发现异常 | 线索指向邪教、失控者或被污染的物品 |
| 第五幕：收集材料 | 玩家为了自保和深入调查，开始寻找占卜家魔药材料 |
| 第六幕：首次晋升 | 玩家调配魔药，晋升为占卜家 |
| 第七幕：追查真凶 | 玩家利用占卜家能力锁定嫌疑人与仪式地点 |
| 第八幕：战斗与风险 | 玩家遭遇失控者或邪教徒，体验简单战斗 |
| 第九幕：序列推进 | 玩家通过材料、任务和扮演推进到小丑 / 魔术师阶段 |
| 第十幕：案件收束 | 玩家解决死亡案件，但发现更高层隐秘，为后续章节留钩子 |

### 7.2 MVP 主线要求

主线不需要大量分支，但必须具备：

- 明确起点：玩家为什么调查案件。
- 明确目标：找出死亡原因和幕后仪式。
- 明确阻力：信息封锁、污染、敌人、材料缺口、失控风险。
- 明确成长：调查推进必须解锁或强化非凡能力。
- 明确结尾：案件解决，但更大的神秘被揭开。

## 8. 材料收集系统范围

第一版材料系统只服务占卜家 / 愚者途径前三个序列。

| 内容 | MVP 范围 |
| --- | --- |
| 材料类型 | 主材料、辅助材料、仪式材料、普通消耗品 |
| 获取方式 | 调查奖励、敌人掉落、黑市购买、任务交换、地图采集 |
| 稀有度 | 普通、稀有、危险、唯一线索物 |
| 背包显示 | 名称、类型、用途、来源、是否可用于魔药 |
| 材料提示 | 缺少材料时提示来源线索，不直接显示完整答案 |

### 8.1 第一版材料设计原则

- 每次晋升至少需要 2 个主材料和若干辅助材料。
- 主材料应绑定具体任务或危险地点。
- 辅助材料可以通过黑市、探索和支线快速获得。
- 部分材料可被非凡特性替代，但替代会提高污染或失控风险。
- 材料不做复杂经济系统，先保证获取路径清楚。

## 9. 魔药晋升系统范围

第一版只做低序列魔药晋升流程。

| 步骤 | MVP 表现 |
| --- | --- |
| 查看配方 | 在途径面板或调查笔记中查看已知配方 |
| 收集材料 | 背包记录材料是否齐全 |
| 调配魔药 | 根据材料完整度、污染情况和仪式条件计算成功率 |
| 饮用魔药 | 播放晋升事件，改变序列和数值上限 |
| 消化魔药 | 通过扮演、调查、战斗和剧情任务提升消化度 |
| 晋升失败 | 造成污染上升、理智下降、负面状态或临时失控事件 |

### 9.1 MVP 晋升成功率

成功率可以先用简单公式：

```text
基础成功率
+ 材料完整度加成
+ 消化度加成
+ 仪式完成度加成
- 污染值惩罚
- 理智值不足惩罚
- 非凡特性替代惩罚
= 最终成功率
```

第一版不需要复杂数学平衡，但必须让玩家理解：准备越充分，晋升越安全；冒险越激进，失控越容易。

## 10. 简单战斗系统范围

第一版战斗只做“可理解、可迭代、能服务剧情”的基础版本。

| 内容 | MVP 范围 |
| --- | --- |
| 战斗形式 | 小规模即时战斗或回合制均可，优先选择实现成本更低的方案 |
| 玩家能力 | 普通攻击、闪避、防御、占卜辅助、小丑机动、魔术师保命技能 |
| 敌人类型 | 普通邪教徒、污染生物、失控者、小 Boss |
| 战斗资源 | 生命值、灵性值、理智值、消耗品 |
| 战斗风险 | 长时间战斗、使用高风险能力、接触污染源会增加污染或失控风险 |
| 战斗奖励 | 材料、线索、任务推进、消化度 |

### 10.1 不追求的战斗内容

第一版不做大型 Boss 战、多武器流派、复杂连招、职业平衡、队友系统和完整敌人生态。战斗只需要支撑调查过程中的危险感。

## 11. 调查笔记系统范围

调查笔记是第一版最重要的 UI 系统之一，用于把案件、线索、人物和地点串起来。

| 页面 | 内容 |
| --- | --- |
| 案件页 | 当前主线目标、案件阶段、关键问题 |
| 线索页 | 已发现线索、来源地点、可信度、关联对象 |
| 人物页 | NPC 身份、关系、嫌疑程度、已知秘密 |
| 地点页 | 可调查地点、异常记录、未完成搜索点 |
| 推理页 | 玩家选择线索组合，触发新任务或结论 |
| 神秘学页 | 已知仪式、魔药配方、占卜结果、污染警告 |

### 11.1 调查笔记最低要求

- 能自动记录主线关键线索。
- 能显示线索之间的关联。
- 能标记当前缺少的证据。
- 能承载占卜结果和失败风险提示。
- 能作为任务日志与案件推理之间的桥梁。

## 12. 失控风险系统范围

失控风险是诡秘风格的 MVP 核心压力系统。

| 数值 | 作用 |
| --- | --- |
| 理智值 | 表示精神稳定程度，过低会影响调查、战斗和晋升 |
| 污染值 | 表示接触异常、邪神、失控者和污染物后的积累 |
| 灵性值 | 使用占卜、灵视、仪式和技能的资源 |
| 锚点值 | 表示自我认知与现实联系，越低越容易失控 |
| 消化度 | 表示当前序列魔药掌握程度，影响晋升安全 |
| 神秘知识承受度 | 表示玩家能否阅读高风险神秘学资料 |

### 12.1 失控触发来源

| 来源 | 风险 |
| --- | --- |
| 占卜高位存在 | 污染值上升，可能获得错误线索 |
| 阅读禁忌资料 | 神秘知识承受度检查失败会降低理智 |
| 魔药调配失败 | 污染上升，产生临时负面状态 |
| 强行晋升 | 进入失控检定或触发失败事件 |
| 战斗拖延 | 理智下降，污染源持续影响 |
| 使用非凡特性替代材料 | 成功率提高或材料补足，但失控风险显著增加 |

### 12.2 MVP 失控结果

第一版失控不需要做复杂永久死亡系统，可以先做分级后果：

| 等级 | 后果 |
| --- | --- |
| 轻度异常 | 画面噪声、幻听、调查判定下降 |
| 中度污染 | 技能消耗提高、NPC 对话异常、短暂敌我识别混乱 |
| 重度失控 | 触发强制战斗、昏迷、任务失败或被官方组织收容 |
| 晋升失败 | 序列晋升中断，损失材料，获得污染创伤或负面特质 |

## 13. 暂时不做的内容

为了保证第一版可以完成，以下内容明确不进入 MVP：

| 暂不做内容 | 原因 |
| --- | --- |
| 22 条途径完整开放 | 数据量过大，会拖慢核心闭环验证 |
| 220 个序列全部制作 | 不适合第一版，后续按途径逐步扩展 |
| 多城市开放世界 | 第一版只验证雾城 |
| 海上区域 | 留到后续五海版本 |
| 灵界完整探索 | 第一版只可作为事件、梦境或短暂场景出现 |
| 神弃之地 | 作为后续大版本内容 |
| 塔罗会完整系统 | 第一版可只作为剧情伏笔或菜单占位 |
| 多主角 / 队友系统 | 增加叙事和战斗复杂度 |
| 大规模阵营声望 | 第一版只保留少量 NPC 态度变化 |
| 复杂经济系统 | 材料够用即可，不做市场模拟 |
| 装备词条刷宝 | 会偏离调查 RPG 核心 |
| 大型 Boss 战 | 第一版只需要小 Boss 或失控者事件 |
| 完整昼夜和天气模拟 | 可先做固定氛围和剧情切换 |
| 多结局 | 第一版只做主线收束和少量分支反馈 |

## 14. 开发优先级

### P0：必须完成

| 模块 | 完成标准 |
| --- | --- |
| 主角控制 | 能移动、交互、查看状态 |
| 雾城基础地图 | 至少包含主角住所、官方据点、案发地、黑市、战斗地点 |
| 主线任务 | 神秘死亡案件能从开始推进到结尾 |
| 调查笔记 | 能记录线索、人物、地点和当前目标 |
| 材料收集 | 能获得并检查占卜家途径材料 |
| 魔药晋升 | 能完成至少一次魔药调配和晋升 |
| 简单战斗 | 能与基础敌人战斗并获得结果 |
| 失控风险 | 使用神秘学和晋升时能产生风险反馈 |
| 存档 / 读档 | 玩家能保存进度，并在失败后回到合理节点 |

### P1：应该完成

| 模块 | 完成标准 |
| --- | --- |
| 三个序列技能 | 占卜家、小丑、魔术师各有代表性技能 |
| 扮演消化 | 调查、表演、战斗等行为能提升消化度 |
| 黑市交易 | 可购买或交换部分材料 |
| NPC 信息网 | 关键 NPC 能提供线索或阻碍 |
| 小 Boss | 主线结尾有一个失控者或邪教徒战斗 |

### P2：有余力再做

| 模块 | 完成标准 |
| --- | --- |
| 支线任务 | 2-3 个小支线，用于补充材料和城市氛围 |
| 简易声望 | 官方组织、黑市 NPC 对玩家态度有轻微变化 |
| 梦境片段 | 占卜或休息后进入短场景获得线索 |
| 封印物试用 | 只放 1 件低风险封印物作为任务道具 |
| 额外结局反馈 | 根据污染值或调查完整度改变结尾台词 |

## 15. MVP 完成标准

当以下条件全部满足时，可以认为第一版 MVP 完成：

- 玩家能从新游戏开始进入雾城。
- 玩家能接到“神秘死亡案件”主线。
- 玩家能调查至少 5 个关键线索。
- 玩家能通过调查笔记理解下一步目标。
- 玩家能收集至少 1 次魔药所需材料。
- 玩家能调配并饮用魔药。
- 玩家能晋升到占卜家，并继续体验小丑或魔术师能力。
- 玩家能经历至少 3 场战斗或危险事件。
- 玩家能看到理智值、污染值、灵性值、消化度或失控风险的变化。
- 玩家能完成主线结尾，并获得下一阶段城市或更大隐秘的伏笔。

## 18. 总结

MVP 的重点不是“把诡秘世界做全”，而是把第一条低序列体验做扎实。

第一版只要完成：

- 一个主角；
- 一个城市；
- 一条主线；
- 一条途径；
- 三个序列；
- 材料、魔药、战斗、调查笔记、失控风险五个核心系统；

就已经足够证明项目的核心玩法方向。后续版本再逐步扩展其他途径、城市、海上区域、灵界、神弃之地、塔罗会和更高序列内容。
