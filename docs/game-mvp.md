# 《诡秘之主》世界地图顺序 MVP 开发文档

> 项目性质：个人学习/个人游玩用粉丝向原作内容改编原型  
> 文档版本：v0.3  
> 创建日期：2026-07-06  
> 编写方向：按照用户给定世界地图顺序，逐区域拆分 MVP/章节开发内容。  
> 说明：本版不再只写廷根篇，而是把整部《诡秘之主》可改编内容按地图顺序组织为阶段式开发路线。

## 0. 文档目标

这份文档用于规划一款 2D《诡秘之主》个人向游戏的“地图章节式 MVP”。这里的 MVP 不是只做一个孤立 Demo，而是指每个大地图区域都要有一个可独立游玩的最小章节：有入口、有核心任务、有主要 NPC、有至少一个区域机制、有结尾反馈，并能把玩家引向下一个地图。

开发顺序严格按照当前世界地图树推进：

```text
世界地图
├── 鲁恩王国
├── 因蒂斯共和国
├── 弗萨克帝国
├── 费内波特王国
├── 五海区域
├── 南大陆
├── 神弃之地
└── 异空间
```

每个区域不追求一次性做成完整开放世界。第一轮开发只做“该区域最有代表性的剧情、地点、系统和敌人”。等全地图顺序跑通后，再回头扩写支线、隐藏地点和高阶玩法。

## 1. 总体开发原则

### 1.1 地图顺序优先

玩家从鲁恩王国进入超凡世界，再逐步接触因蒂斯、弗萨克、费内波特、五海、南大陆、神弃之地，最后进入灵界、灰雾之上、梦境世界和历史迷雾等异空间。这个顺序既符合玩家认知递进，也方便开发从城市调查逐渐扩展到海上冒险、战争遗迹、神话副本和高阶空间。

### 1.2 原作内容优先

本项目是个人向原作改编，因此每个地图都直接绑定《诡秘之主》原作内容：

- 鲁恩王国：克莱恩开局、廷根、值夜者、贝克兰德、复仇、塔罗会成长。
- 因蒂斯共和国：罗塞尔遗产、特里尔、机械与蒸汽、博物馆与地下陵墓。
- 弗萨克帝国：北境、战神教会、战争与巨人/黄昏意象。
- 费内波特王国：大地母神、丰饶、生命、森林与古老信仰。
- 五海区域：格尔曼·斯帕罗、海盗、拜亚姆、罗思德群岛、海神线。
- 南大陆：拜朗、死神遗产、殖民冲突、陵寝与亡灵。
- 神弃之地：白银城、黑暗荒原、巨人王庭、神话真相。
- 异空间：灵界、灰雾、梦境、历史迷雾，高阶能力与终局系统。

### 1.3 每章一个核心系统

每个地图章节都至少解锁或强化一个系统，避免只是换场景：

- 鲁恩王国：调查、占卜、值夜者任务、塔罗会。
- 因蒂斯共和国：罗塞尔日记解码、博物馆潜入、机械机关。
- 弗萨克帝国：战争压力、阵营战场、寒冷生存。
- 费内波特王国：生命/丰饶、治疗、自然仪式。
- 五海区域：航海、悬赏、海盗声望、岛屿探索。
- 南大陆：殖民地声望、亡灵陵寝、死神污染。
- 神弃之地：光源管理、黑暗生存、城邦集体命运。
- 异空间：灵界穿梭、灰雾会议、梦境副本、历史投影。

## 3. 第一阶段 MVP：鲁恩王国

### 3.1 区域定位

鲁恩王国是整个游戏的入门区域，也是玩家从普通世界进入非凡世界的主线起点。它承担教学、主角成长、值夜者体系、塔罗会雏形、都市调查、贵族阴谋和复仇线铺垫。

### 3.2 地图一：廷根市

改编重点：

- 周明瑞穿越成为克莱恩·莫雷蒂。
- 原主死亡、韦尔奇、娜娅与安提哥努斯家族笔记。
- 邓恩、伦纳德、戴莉审查。
- 黑荆棘安保公司和值夜者入门。
- 占卜家魔药、灵视、灵摆占卜、扮演法。
- 老尼尔失控。
- 梅高欧丝与真实造物主子嗣。
- 因斯·赞格威尔和 0-08 的幕后阴影。

核心地点：

- 莫雷蒂家。
- 廷根街区。
- 黑荆棘安保公司。
- 查尼斯门入口。
- 占卜俱乐部。
- 韦尔奇住所。
- 地下交易市场。
- 老尼尔住所。
- 梅高欧丝终局场景。
- 墓地。

核心任务：

1. 愚者苏醒：房间调查、转运仪式、首次灰雾。
2. 值夜者审查：邓恩、伦纳德、戴莉对克莱恩进行审查。
3. 成为占卜家：加入黑荆棘，服食占卜家魔药。
4. 安提哥努斯笔记：调查韦尔奇事件和瑞尔·比伯线。
5. 占卜俱乐部：消化魔药，学习扮演法。
6. 老尼尔失控：见证非凡代价。
7. 梅高欧丝事件：真实造物主子嗣降临。
8. 廷根终局：邓恩牺牲，克莱恩死亡并复苏。

解锁系统：

- 基础移动与交互。
- 对话与调查。
- 案件笔记。
- 灵视。
- 灵摆占卜。
- 魔药消化。
- 失控风险。
- 灰雾之上。
- 塔罗会第一次会议。

完成标准：

- 玩家能完整体验第一卷《小丑》的主线情绪。
- 玩家理解魔药、序列、扮演法、失控、封印物。
- 玩家在终局后自然进入“离开廷根，前往更大世界”的状态。

### 3.3 地图二：贝克兰德

改编重点：

- 克莱恩以夏洛克·莫里亚蒂身份进入首都。
- 侦探委托、地下非凡圈、军情九处、贵族圈。
- 塔罗会扩张。
- 因斯·赞格威尔复仇线继续推进。
- 贝克兰德大雾霾事件。
- 贫民区、工厂污染、贵族政治和邪神阴谋。

核心地点：

- 侦探事务所。
- 东区贫民区。
- 乔伍德区与希尔斯顿区。
- 圣赛缪尔教堂。
- 地下聚会地点。
- 贵族沙龙。
- 工厂区。
- 大雾霾事件核心场景。

核心任务：

1. 新身份夏洛克：建立侦探身份，接取普通委托。
2. 地下聚会：进入贝克兰德非凡者交易圈。
3. 塔罗会扩张：更多成员加入，情报系统升级。
4. 贵族与贫民：通过多条任务线理解城市结构。
5. 大雾霾前兆：调查污染、献祭和异常疾病。
6. 贝克兰德灾难：完成城市级灾难章节。

解锁系统：

- 多身份系统。
- 城市声望。
- 地下交易升级。
- 侦探委托系统。
- 大规模灾难事件。

完成标准：

- 玩家能感受到贝克兰德比廷根更复杂、更冷酷。
- 夏洛克身份与克莱恩原身份形成明确差异。
- 塔罗会从“伪装会议”变成真实情报网络。

### 3.4 地图三：普利兹港

改编重点：

- 鲁恩重要港口，连接首都与五海。
- 可作为贝克兰德篇到海上篇的过渡区域。
- 引出海盗、船票、走私、风暴教会、海外殖民地。

核心任务：

1. 离港准备：获得合法或非法出海渠道。
2. 风暴教会检查：面对海上非凡者体系。
3. 走私货物：发现异常物品流向五海。
4. 海上身份铺垫：为格尔曼·斯帕罗身份做准备。

解锁系统：

- 船只移动。
- 航线选择。
- 货物检查。
- 海上悬赏预告。

### 3.5 地图四：地下黑市

改编重点：

- 黑市贯穿鲁恩王国多个阶段，不是单一地点。
- 廷根黑市偏教学，贝克兰德黑市偏交易，普利兹黑市偏走私和海上通道。

核心功能：

- 魔药材料交易。
- 封印物/神奇物品信息。
- 悬赏与情报。
- 非法身份文件。
- 隐藏任务入口。

完成标准：

- 玩家能通过黑市获得推进主线的替代路线。
- 黑市既有诱惑，也有背叛和污染风险。

## 4. 第二阶段 MVP：因蒂斯共和国

### 4.1 区域定位

因蒂斯共和国代表罗塞尔遗产、蒸汽与机械、政治浪漫、博物馆收藏、秘密陵墓和前穿越者历史。它适合作为玩家从鲁恩主线进入“世界历史真相”的第一大区域。

### 4.2 地图一：特里尔

核心内容：

- 因蒂斯首都风格，政治、艺术、机械和神秘学并存。
- 罗塞尔影响无处不在。
- 蒸汽与机械教会、知识相关组织、地下沙龙。

核心任务：

1. 罗塞尔日记线索：塔罗会或贝尔纳黛线引导玩家前往特里尔。
2. 机械沙龙：调查一场与罗塞尔遗物有关的展览。
3. 政治暗杀：城市任务引出因蒂斯政治复杂性。
4. 罗塞尔真相碎片：获得前穿越者历史信息。

解锁系统：

- 罗塞尔文字解码。
- 机械机关谜题。
- 博物馆潜入。

### 4.3 地图二：罗塞尔博物馆

核心内容：

- 收集罗塞尔遗物、日记页、机械造物和隐秘展品。
- 博物馆可做成大型室内副本。

核心任务：

1. 展品失窃：调查被替换的罗塞尔遗物。
2. 日记页鉴定：识别真伪日记。
3. 隐藏展厅：进入对普通游客封锁的区域。
4. 贝尔纳黛线索：为后续海上或高阶剧情铺垫。

### 4.4 地图三：地下陵墓

核心内容：

- 罗塞尔生前秘密、因蒂斯旧时代贵族、机械机关与亡灵防护。
- 可以作为因蒂斯章节终局副本。

核心任务：

1. 找到地下入口。
2. 破解机械封锁。
3. 面对被遗留的非凡守卫。
4. 取得罗塞尔关键资料，但不完整揭示全部真相。

完成标准：

- 因蒂斯章节结束时，玩家对罗塞尔和旧世界真相产生更强兴趣。

## 5. 第三阶段 MVP：弗萨克帝国

### 5.1 区域定位

弗萨克帝国体现北境、战争、军国、战神信仰、黄昏巨人意象和冰原遗迹。这个区域适合引入战争压力、寒冷生存和军事要塞玩法。

### 5.2 地图一：北境冰原

核心任务：

1. 冰原行军：玩家需要管理寒冷、补给和可视范围。
2. 失踪巡逻队：调查冰原上消失的军队。
3. 黄昏痕迹：发现巨人或古代战争遗留。
4. 极夜异常：面对黑暗和梦境侵入。

解锁系统：

- 寒冷值。
- 补给。
- 冰原随机事件。

### 5.3 地图二：战神要塞

核心任务：

1. 进入要塞：通过身份、军方许可或潜入。
2. 战神教会档案：调查与战争相关的封印物。
3. 前线会议：参与或旁听军事阴谋。
4. 要塞守卫战：小型战场事件。

解锁系统：

- 阵营战场。
- 军事声望。
- 队伍协同防守。

### 5.4 地图三：冰封遗迹

核心任务：

1. 寻找遗迹入口。
2. 解冻古代仪式场。
3. 处理巨人/黄昏相关污染。
4. 带出与神弃之地相关的伏笔。

完成标准：

- 玩家获得“战争不是背景，而是神明与国家争斗延伸”的认知。

## 6. 第四阶段 MVP：费内波特王国

### 6.1 区域定位

费内波特王国以大地、丰饶、生命、森林、古老信仰和大地母神教会为核心。它适合做一个节奏相对舒缓但暗藏腐化的区域。

### 6.2 地图一：丰饶平原

核心任务：

1. 异常丰收：调查农作物过度生长。
2. 村庄病变：部分居民出现生命力过剩或畸变。
3. 丰饶祭典：参与地方仪式并发现问题。
4. 大地教会介入：选择配合或独立调查。

解锁系统：

- 治疗与生命状态。
- 作物/资源采集。
- 民俗仪式。

### 6.3 地图二：古老森林

核心任务：

1. 迷失森林：地图路径会随时间改变。
2. 生命学派线索：接触与生命、命运相关的组织。
3. 植物异常：面对活化植物和生命污染。
4. 森林核心：发现古老祭坛。

解锁系统：

- 动态迷宫。
- 生命污染。
- 自然灵性材料采集。

### 6.4 地图三：大地神庙

核心任务：

1. 进入神庙。
2. 调查丰饶异变源头。
3. 处理生命与母性意象相关的危险。
4. 获得关于母亲/月亮途径的资料。

完成标准：

- 玩家理解“生命”在《诡秘之主》中不只是治疗，也可能是过度繁殖、畸变和污染。

## 7. 第五阶段 MVP：五海区域

### 7.1 区域定位

五海区域是格尔曼·斯帕罗、海盗、悬赏、殖民地、风暴教会、拜亚姆、罗思德群岛和海神信仰的主舞台。这里应从城市调查转为海上冒险。

### 7.2 地图一：罗思德群岛

核心任务：

1. 抵达群岛：获得海上活动许可或伪装身份。
2. 殖民秩序：理解鲁恩殖民体系。
3. 海盗悬赏：追踪一个低级海盗目标。
4. 风暴教会巡查：处理海上教会势力。

解锁系统：

- 海图。
- 悬赏榜。
- 岛屿探索。
- 海盗声望。

### 7.3 地图二：拜亚姆

核心任务：

1. 格尔曼身份登场。
2. 拜亚姆地下任务。
3. 海神信仰与卡维图瓦线索。
4. 殖民者、本地反抗者、海盗三方冲突。

解锁系统：

- 多身份声望升级。
- 海神信仰任务。
- 城市压迫与反抗选择。

### 7.4 地图三：幽灵船海域

核心任务：

1. 进入异常海域。
2. 调查失踪船只。
3. 面对亡灵船员或海上封印物。
4. 获得南大陆或死神线索。

解锁系统：

- 船只耐久。
- 海上随机灾难。
- 幽灵船副本。

### 7.5 地图四：秘密岛屿

核心任务：

1. 解读藏宝图或罗塞尔遗留线索。
2. 登陆未知岛屿。
3. 探索封印遗迹。
4. 取得关键神奇物品或高阶材料。

完成标准：

- 玩家完成从“侦探克莱恩”到“疯狂冒险家格尔曼”的体验转换。

## 8. 第六阶段 MVP：南大陆

### 8.1 区域定位

南大陆承载拜朗、死神遗产、殖民冲突、亡灵、陵寝和古代帝国残响。这里是死亡途径和殖民主题的重要区域。

### 8.2 地图一：拜朗遗迹

核心任务：

1. 搜寻拜朗旧城。
2. 解读古代死亡仪式。
3. 遭遇亡灵守卫。
4. 获得死神相关历史碎片。

解锁系统：

- 死亡污染。
- 亡灵对话。
- 古代文字解读。

### 8.3 地图二：殖民城镇

核心任务：

1. 进入殖民城镇。
2. 调查本地反抗组织与殖民官员冲突。
3. 处理走私、疫病、矿场和教会任务。
4. 玩家选择帮助哪一方或保持中立。

解锁系统：

- 殖民地声望。
- 多阵营任务。
- 资源压迫与道德选择。

### 8.4 地图三：死神陵寝

核心任务：

1. 找到陵寝入口。
2. 破解死亡仪式机关。
3. 面对高阶亡灵或死神残留。
4. 取得通往神弃之地或灵界深处的线索。

完成标准：

- 玩家意识到南大陆不是支线地图，而是世界历史和神明遗产的重要拼图。

## 9. 第七阶段 MVP：神弃之地

### 9.1 区域定位

神弃之地是整部作品中最具神话重量的地图之一。这里应从普通国家政治转入末世生存、黑暗荒原、白银城、巨人王庭和古代真相。

### 9.2 地图一：白银城

核心任务：

1. 以戴里克视角或塔罗会远程视角进入白银城。
2. 了解没有太阳的生存规则。
3. 参与探索小队训练。
4. 通过塔罗会把白银城与克莱恩主线连接。

解锁系统：

- 多视角角色。
- 城邦资源。
- 光源管理。
- 塔罗会远程干预。

### 9.3 地图二：黑暗荒原

核心任务：

1. 离开白银城探索。
2. 管理光源和队伍精神。
3. 面对黑暗怪物、污染生物、古代遗迹。
4. 带回生存物资和历史线索。

解锁系统：

- 光源值。
- 队伍恐惧。
- 黑暗遭遇。

### 9.4 地图三：巨人王庭

核心任务：

1. 取得进入条件。
2. 探索古代巨人王庭。
3. 了解古神时代和第二纪线索。
4. 触发神话级场景与高阶敌人。

解锁系统：

- 神话副本。
- 高阶污染压制。
- 古代历史档案。

### 9.5 地图四：畸变巢穴

核心任务：

1. 调查神弃之地畸变源头。
2. 清理或绕过怪物巢穴。
3. 取得白银城迁徙或开路所需关键材料。
4. 为后续外神/末日主题铺垫。

完成标准：

- 玩家真正理解“追寻光明”的重量。

## 10. 第八阶段 MVP：异空间

### 10.1 区域定位

异空间不是普通地图，而是贯穿全游戏的高阶系统集合。它包括灵界、灰雾之上、梦境世界和历史迷雾。前期作为辅助系统出现，后期成为主战场。

### 10.2 地图一：灵界

核心任务：

1. 初次灵界穿梭。
2. 寻找灵界生物或信使。
3. 处理坐标迷失。
4. 通过灵界到达现实难以抵达的地点。

解锁系统：

- 灵界坐标。
- 信使系统。
- 灵界危险事件。
- 空间穿梭。

### 10.3 地图二：灰雾之上

核心任务：

1. 召开塔罗会。
2. 整理罗塞尔日记。
3. 远程帮助成员。
4. 进行灰雾占卜。
5. 管理愚者形象和塔罗会信任。

解锁系统：

- 塔罗会席位。
- 情报交换。
- 成员委托。
- 灰雾占卜。
- 愚者声望。

### 10.4 地图三：梦境世界

核心任务：

1. 进入目标梦境。
2. 寻找记忆碎片。
3. 面对梦境守卫或心理暗示。
4. 从梦境中带回关键线索。

解锁系统：

- 梦境副本。
- 心理防御。
- 梦境污染。

### 10.5 地图四：历史迷雾

核心任务：

1. 使用高阶能力进入历史迷雾。
2. 召唤或观察历史投影。
3. 调查被遗忘的历史。
4. 在终局中使用历史与命运相关能力。

解锁系统：

- 历史投影。
- 古代学者能力。
- 奇迹/愚弄机制。

完成标准：

- 异空间从“功能界面”升级为真正可探索区域。
- 玩家能理解克莱恩高阶能力与世界真相之间的关系。

## 11. 全地图开发顺序

推荐按以下顺序制作：

1. 鲁恩王国 - 廷根市：完成核心基础系统。
2. 鲁恩王国 - 贝克兰德：完成多身份与都市复杂任务。
3. 鲁恩王国 - 普利兹港/地下黑市：完成海上过渡。
4. 因蒂斯共和国：完成罗塞尔遗产和机械机关。
5. 弗萨克帝国：完成战争与寒冷生存。
6. 费内波特王国：完成生命/自然/丰饶系统。
7. 五海区域：完成航海、悬赏、海盗声望。
8. 南大陆：完成殖民地、多阵营、死亡遗迹。
9. 神弃之地：完成白银城、黑暗生存、神话副本。
10. 异空间：完成灵界、灰雾、梦境、历史迷雾的高阶整合。

这个顺序和你给的地图树保持一致，只是把鲁恩王国内部的廷根、贝克兰德、普利兹港、地下黑市进一步拆为前后衔接的小版本。

## 12. 每个区域的验收标准

每个地图章节完成时，至少满足：

- 有 1 张主地图和 2 到 4 个子场景。
- 有 1 条主线任务链。
- 有 2 到 4 条支线或可选事件。
- 有至少 3 名关键 NPC。
- 有 1 个区域专属系统。
- 有 1 个区域终局副本或高潮事件。
- 有能引导玩家进入下一区域的剧情钩子。

如果某个区域只做了地图但没有系统和剧情闭环，就不算完成 MVP。

本地图 MVP 需要和资料库文档联动：

- 区域剧情、地点和任务记录在本文件。
- 后续建议再单独拆出 `characters.md`、`organizations.md`、`quests.md` 和 `items-data.md`。

## 14. 总结

这一版 MVP 不再把《诡秘之主》压缩成一个廷根 Demo，而是按世界地图顺序制定长期章节化路线。开发上仍然应该从鲁恩王国开始，因为那里承担基础系统和情绪入口；但文档结构已经为后续因蒂斯、弗萨克、费内波特、五海、南大陆、神弃之地和异空间留好了位置。

最关键的开发策略是：每个地图只先做一个最小但完整的章节。地图越往后，能力越高阶，系统越复杂；前面的地图必须稳定支撑调查、魔药、扮演、塔罗会、封印物和身份系统，后面的世界才不会散。
