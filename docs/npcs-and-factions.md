# NPC 与势力系统设计文档

> 文档用途：定义项目中的核心势力、代表 NPC、玩家关系、任务来源、商店内容、敌对关系和声望奖励。  
> 适用范围：第一卷“雾城疑案”、后续城市扩展、任务系统、声望系统、商店系统、阵营关系、NPC 对话和剧情分支。  
> 当前范围：先准备第一阶段可开发势力，不一次性写完整世界所有组织。

## 1. 设计目标

势力不是背景设定摆件，而是任务、商店、敌对、声望、地图权限和剧情分支的来源。

每个势力至少要回答：

1. 它相信什么。
2. 它控制哪里。
3. 谁代表它和玩家说话。
4. 它给玩家什么任务。
5. 它卖什么或提供什么资源。
6. 它和谁敌对。
7. 玩家提高声望后能获得什么。
8. 玩家得罪它后会付出什么代价。

第一卷中，玩家不需要立刻理解所有大势力，只需要先接触：

- 一个官方非凡者组织。
- 一个邪教组织。
- 一个地下交易网络。
- 一个普通人阵营。
- 一个教会势力。
- 一个贵族势力的入口。
- 一个海盗势力的预告。

## 2. 势力字段格式

每个势力统一使用以下字段：

| 字段 | 说明 |
| --- | --- |
| 势力 ID | 用于任务、声望、商店和对话，例如 `faction_lamplighters` |
| 势力名称 | 玩家可见名称 |
| 势力类型 | 官方组织、邪教组织、地下交易会、海盗势力、贵族势力、教会势力、普通人阵营 |
| 理念 | 该势力的行动原则和价值观 |
| 主要据点 | 地图上的势力据点 |
| 代表 NPC | 与玩家互动的关键 NPC |
| 与玩家关系 | 初始态度与可变化方向 |
| 提供任务 | 可发布的主线、支线、日常委托或隐藏任务 |
| 出售物品 | 商店、补给、黑市、封印物申请或特殊交易 |
| 敌对势力 | 冲突对象 |
| 声望奖励 | 声望提升后解锁的权限、道具、任务和剧情 |

建议声望区间：

| 声望等级 | 数值 | 表现 |
| --- | --- | --- |
| 敌对 | -3 到 -1 | 追捕、拒绝交易、派人监视 |
| 中立 | 0 | 可对话，但不给核心资源 |
| 合作 | 1 到 2 | 开放普通任务、基础商店和情报 |
| 信任 | 3 到 4 | 开放高级任务、折扣、禁区通行 |
| 核心盟友 | 5 | 关键剧情支援、稀有资源和结局分支 |

## 4. 势力详表

### 4.1 守灯人组织

势力 ID：`faction_lamplighters`  
势力名称：守灯人组织  
势力类型：官方组织  
定位：官方非凡者组织  
功能：新手引导、案件任务、封印物申请、晋升许可  
对应玩法：调查、值班、收容、晋升许可、官方声望、档案阅读

| 字段 | 内容 |
| --- | --- |
| 理念 | 在普通人看不见的地方维持城市秩序。异常必须被调查、记录、封存，必要时可以牺牲少数人保护多数人。 |
| 主要据点 | 黑荆棘事务所、雾城灯塔档案室、市警局密档室、地下封印间 |
| 代表 NPC | 艾琳娜・格雷、赫伯特・瓦尔、守灯人书记员、封印物管理员 |
| 与玩家关系 | 初始为审查和观察；若玩家坦白线索并配合调查，可转为合作；若隐瞒污染物或私自交易，会被监视 |
| 提供任务 | 神秘死亡案、污染样本回收、失控者处置、夜间值班、封印物护送、晋升前审查 |
| 出售物品 | 镇静药剂、净化火柴、简易净化粉、仪式蜡烛、低阶魔药材料；高声望可申请低危封印物临时使用 |
| 敌对势力 | 灰烛会、失控者群体、非法黑市卖家、污染封印物走私者 |
| 声望奖励 | 档案权限、安全屋、官方补给、晋升许可、封印物申请、队友支援、污染治疗折扣 |

声望节点：

| 声望 | 解锁 |
| --- | --- |
| 1 | 临时调查许可、基础补给 |
| 2 | 安全屋使用权、低阶档案阅读 |
| 3 | 魔药材料配给、晋升审查加成 |
| 4 | 低危封印物临时申请、官方支援任务 |
| 5 | 关键剧情救援、特殊结局支持 |

### 4.2 灰烛会

势力 ID：`faction_gray_candle_cult`  
势力名称：灰烛会  
势力类型：邪教组织  
定位：第一卷主要敌对势力  
功能：案件制造者、污染源、失控者来源、最终仪式敌人  
对应玩法：潜入、追踪、净化、战斗、污染风险、邪教关注度

| 字段 | 内容 |
| --- | --- |
| 理念 | 认为雾、火、死亡和幻觉是通向更高启示的门。普通人的痛苦只是点燃真相的蜡油。 |
| 主要据点 | 雾港码头仓库、废弃钟楼、蓝幕剧院地下室、下城区秘密祈祷点 |
| 代表 NPC | 灰烛会传教士、“白面先生”、被污染的码头工、隐藏资助人 |
| 与玩家关系 | 默认敌对；玩家可假装加入以潜入，但会提高污染和邪教关注度 |
| 提供任务 | 不直接提供正常任务；可通过卧底、诱饵交易和假委托触发邪教任务链 |
| 出售物品 | 污染蜡块、伪造魔药材料、禁忌仪式纸页、邪教护符；多数通过黑市或战利品获得 |
| 敌对势力 | 守灯人组织、圣辉教区、沉钟街居民网络、部分旧钟交易会成员 |
| 声望奖励 | 不建议做正向声望；可做“邪教关注度”，关注度越高，伏击、假线索和污染诱导越多 |

邪教关注度：

| 关注度 | 表现 |
| --- | --- |
| 1 | 玩家被秘密观察，街角出现灰烛符号 |
| 2 | 黑市出现针对玩家的假情报 |
| 3 | 夜间遭遇伏击或污染梦境 |
| 4 | 关键 NPC 被威胁，调查路线受阻 |
| 5 | 最终仪式难度提高，邪教高层提前转移证据 |

### 4.3 旧钟交易会

势力 ID：`faction_old_bell_exchange`  
势力名称：旧钟交易会  
势力类型：地下交易会  
定位：黑市和非官方神秘学交易网络  
功能：材料购买、情报交易、非法仪式、封印物线索、替代路线  
对应玩法：黑市声望、讨价还价、假货鉴定、污染风险、地下路线

| 字段 | 内容 |
| --- | --- |
| 理念 | 没有什么不能交易，只是价格不一定用钱支付。情报、材料、人情和秘密都是货币。 |
| 主要据点 | 老莫里斯书店地下室、旧钟巷、雾港仓库后门、流动拍卖包厢 |
| 代表 NPC | 老莫里斯、走私中间人莲娜、黑市鉴定师、戴面纱的拍卖主持 |
| 与玩家关系 | 初始谨慎中立；玩家守信用会提升地下声望；告密或赖账会被封锁交易 |
| 提供任务 | 寻找稀有材料、鉴定污染物、护送交易、调查假货、潜入拍卖会 |
| 出售物品 | 魔药材料、灵摆、仪式蜡烛、纸人、净化粉、走私武器、低可信情报、污染样本 |
| 敌对势力 | 守灯人组织执法派、灰烛会黑吃黑成员、贵族私兵、竞争黑市 |
| 声望奖励 | 折扣、稀有材料、隐藏拍卖、封印物传闻、地下通行证、假身份文件 |

交易风险：

- 低声望时更容易买到污染材料或假货。
- 与守灯人组织关系过高时，部分黑市商人会拒绝交易。
- 出售封印物线索可能引来邪教或贵族势力抢夺。

### 4.4 黑帆同盟

势力 ID：`faction_black_sail_league`  
势力名称：黑帆同盟  
势力类型：海盗势力  
定位：海上篇和港口走私线的预告势力  
功能：航线情报、悬赏、海上材料、走私货物、下一阶段地图入口  
对应玩法：海盗声望、航线选择、悬赏、海上异常、走私检查

| 字段 | 内容 |
| --- | --- |
| 理念 | 陆地的法律只到码头为止，海上的秩序由炮火、契约和风暴决定。 |
| 主要据点 | 雾港外海、黑帆酒馆、走私船舱、远洋暗礁据点 |
| 代表 NPC | “红帆”伊莱亚斯、独眼航海士、黑帆酒馆老板娘、走私船医 |
| 与玩家关系 | 第一卷只露面或间接接触；可通过旧钟交易会和雾港码头任务建立初始关系 |
| 提供任务 | 走私货物追踪、悬赏目标情报、海上失踪船调查、秘密离城路线 |
| 出售物品 | 海图、走私短刃、火药、海兽材料、风暴护符、船票、禁运药剂 |
| 敌对势力 | 风暴系教会、港口官方组织、贵族商会、竞争海盗团 |
| 声望奖励 | 航线折扣、海上护送、海盗悬赏、特殊船员、岛屿线索 |

第一卷使用方式：

- 不把海盗线完全展开。
- 通过“黑伞人的车票”“雾港码头走私货物”“黑帆酒馆传闻”埋下入口。
- 第二卷或海上章节再扩展为完整阵营。

### 4.5 兰伯特家族

势力 ID：`faction_lambert_family`  
势力名称：兰伯特家族  
势力类型：贵族势力  
定位：上城区政治、舞会、宅邸案件和贵族阴谋入口  
功能：社会身份、贵族委托、禁区通行、资源交换、政治压力  
对应玩法：社交、潜入、礼仪、身份伪装、贵族声望

| 字段 | 内容 |
| --- | --- |
| 理念 | 秩序必须由有资格的人掌握。真相可以被保存，但不一定要被公布。 |
| 主要据点 | 上城区兰伯特宅邸、议会包厢、私人博物馆、慈善晚宴 |
| 代表 NPC | 塞西莉亚・兰伯特夫人、兰伯特家族管家、年轻继承人、私人律师 |
| 与玩家关系 | 初始疏远；玩家需要身份、介绍信或任务契机进入贵族圈 |
| 提供任务 | 宅邸失窃、舞会调查、贵族丑闻遮掩、私人封印物鉴定、政治保护交换 |
| 出售物品 | 不直接售卖普通商品；可提供邀请函、介绍信、稀有拍卖资格、贵族护符 |
| 敌对势力 | 贫民区激进组织、竞争贵族、部分教会审查者、黑市敲诈者 |
| 声望奖励 | 上城区通行、贵族身份背书、拍卖权限、警局压力减免、特殊剧情分支 |

设计注意：

- 贵族势力不应只有“坏人”功能，也可以提供合法保护和高级情报。
- 与贵族合作可能降低普通人阵营好感。
- 过度依赖贵族保护会让官方组织怀疑玩家动机。

### 4.6 圣辉教区

势力 ID：`faction_saint_radiance_parish`  
势力名称：圣辉教区  
势力类型：教会势力  
定位：净化、治疗、慈善、异端审查和污染收容  
功能：理智恢复、污染净化、医疗支线、教会任务、道德压力  
对应玩法：净化、治疗、救援、审查、教会声望

| 字段 | 内容 |
| --- | --- |
| 理念 | 光必须照进雾中，但照亮不等于宽恕。污染应被净化，异端应被审查。 |
| 主要据点 | 圣辉教堂、贫民救济所、地下净化室、教区墓园 |
| 代表 NPC | 玛塔修女、伊诺克神父、教区医师、异端审查助理 |
| 与玩家关系 | 初始友善但谨慎；若玩家污染过高或携带邪教物，会被要求净化或审查 |
| 提供任务 | 净化污染物、救助贫民、调查异端、处理墓园异常、护送圣物 |
| 出售物品 | 圣水、净化粉、镇静药剂、治疗药膏、低阶护符、反污染仪式材料 |
| 敌对势力 | 灰烛会、污染封印物走私者、亵渎墓园者、部分黑市商人 |
| 声望奖励 | 治疗折扣、污染净化、教会护符、墓园通行、救援支援、理智恢复加成 |

与官方组织区别：

- 守灯人组织偏调查、收容和执法。
- 圣辉教区偏净化、治疗和信仰秩序。
- 二者可以合作，也会在“是否公开真相”“是否处决污染者”上产生分歧。

### 4.7 沉钟街居民网络

势力 ID：`faction_sunken_bell_residents`  
势力名称：沉钟街居民网络  
势力类型：普通人阵营  
定位：城市日常、情感支线、谣言、目击者和市民恐慌  
功能：普通人视角、线索来源、道德选择、街区状态  
对应玩法：街区声望、市民恐慌、谣言、保护 NPC、案件证词

| 字段 | 内容 |
| --- | --- |
| 理念 | 活下去，守住家人和街坊，别让大人物的秘密毁掉普通人的生活。 |
| 主要据点 | 沉钟街、红砖酒馆、报童路口、码头工宿舍、公共浴场 |
| 代表 NPC | 罗莎・温特、报童妮可、酒馆老板娘玛莎、码头工家属、巡警赫伯特 |
| 与玩家关系 | 初始陌生；玩家保护居民、说出部分真相、解决案件会提高街区声望 |
| 提供任务 | 失踪人口、目击证词、家庭遗物、街区传闻、普通案件、救援请求 |
| 出售物品 | 食物、普通药品、旧衣物、报纸、低价工具、街区情报 |
| 敌对势力 | 灰烛会、压榨性贵族代理人、暴力黑帮、污染失控者 |
| 声望奖励 | 证词、藏身处、街区捷径、低价补给、群众掩护、结局情感支持 |

市民恐慌联动：

| 恐慌度 | 表现 |
| --- | --- |
| 0-1 | 街区正常，NPC 愿意提供传闻 |
| 2 | 夜间行人减少，物价小幅上涨 |
| 3 | 部分商店提前关门，目击者逃离 |
| 4 | 街区出现封锁，谣言变成恐慌 |
| 5 | 普通任务关闭，邪教或官方强势介入 |

## 6. 势力关系矩阵

| 势力 | 盟友 / 可合作 | 敌对 / 冲突 | 主要矛盾 |
| --- | --- | --- | --- |
| 守灯人组织 | 圣辉教区、市警局、部分普通人 | 灰烛会、污染黑市、失控者 | 维持秩序与保密 |
| 灰烛会 | 被收买的走私者、腐化贵族代理人 | 守灯人组织、圣辉教区、普通人阵营 | 用污染和献祭追求启示 |
| 旧钟交易会 | 黑帆同盟、部分贵族、玩家 | 守灯人执法派、灰烛会黑吃黑成员 | 利益优先，拒绝被任何一方完全控制 |
| 黑帆同盟 | 旧钟交易会、部分走私商 | 港口官方、风暴系教会、贵族商会 | 控制海上货路和禁运品 |
| 兰伯特家族 | 教会高层、警局上层、私人商会 | 竞争贵族、贫民区激进者、敲诈者 | 维护家族权力和名誉 |
| 圣辉教区 | 守灯人组织、普通人阵营 | 灰烛会、污染封印物交易者 | 净化污染和维护信仰秩序 |
| 沉钟街居民网络 | 圣辉教区、可信玩家、少数警员 | 灰烛会、压迫者、失控者 | 普通人生存与真相保护 |

## 7. 声望与任务联动

### 7.1 声望变化来源

| 行为 | 影响 |
| --- | --- |
| 向守灯人组织完整汇报污染物 | 守灯人声望 +1，旧钟交易会声望可能 -1 |
| 私下购买黑市材料 | 旧钟交易会声望 +1，守灯人组织信任可能下降 |
| 救助污染事件中的普通人 | 沉钟街居民网络 +1，圣辉教区 +1 |
| 为贵族隐瞒丑闻 | 兰伯特家族 +1，普通人阵营可能下降 |
| 保留邪教仪式物 | 灰烛会关注度 +1，污染风险上升 |
| 净化灰烛会据点 | 圣辉教区 +1，灰烛会关注度 +1 |
| 放走走私者换取海上情报 | 黑帆同盟 +1，守灯人组织可能下降 |

### 7.2 声望奖励类型

| 奖励类型 | 示例 |
| --- | --- |
| 地图权限 | 上城区邀请函、黑市入口、官方档案室、港口暗道 |
| 商店内容 | 魔药材料、净化道具、封印物线索、走私武器 |
| 任务解锁 | 贵族舞会、黑市拍卖、教会净化、海盗航线 |
| 战斗支援 | 官方队友、教会治疗、普通人藏身处、海盗撤离路线 |
| 剧情分支 | 官方结局、地下结局、普通人保护结局、贵族交易结局 |

### 7.3 第一卷结局倾向

第一卷的势力选择不应做成简单善恶，而应让玩家在“真相公开”“城市稳定”“个人自由”和“保护普通人”之间取舍。

| 结局倾向 | 触发方向 | 结果 |
| --- | --- | --- |
| 官方结案 | 守灯人组织和圣辉教区声望较高，关键证据完整上交 | 案件被压入机密档案，城市表面稳定，玩家获得官方推荐和下一城合法身份 |
| 公开真相 | 沉钟街居民网络声望较高，玩家公开贵族、邪教和污染证据 | 普通人获得真相和支援，但城市恐慌上升，贵族势力和部分官方人员敌视玩家 |
| 黑市交易结案 | 旧钟交易会或黑帆同盟关系较高，玩家用证据换取资源和离城路线 | 玩家获得非法路线、稀有情报和自由行动空间，但官方信任下降 |
| 贵族遮掩 | 兰伯特家族声望较高，玩家帮助压下部分丑闻 | 上城区资源和保护增加，普通人阵营不满，后续可能出现被掩盖的污染后果 |
| 污染失控结案 | 灰烛会关注度过高，关键净化或封印失败 | 最终仪式难度提高，城市部分区域污染加重，下一卷开局带有后遗症 |

### 7.4 声望反馈原则

声望最好在两幕内给玩家可见反馈，而不是只在菜单中改变数字。

| 玩家行为 | 可见反馈 |
| --- | --- |
| 帮助沉钟街居民 | 报童主动报信，酒馆老板提供藏身处，街头 NPC 改口称呼玩家 |
| 获得守灯人信任 | 警员放行，档案室开放，任务结算时提供净化补给 |
| 得罪旧钟交易会 | 黑市涨价，假货概率上升，线人拒绝露面 |
| 提升圣辉教区声望 | 治疗费用下降，污染净化成功率提高，教堂安全屋开放 |
| 激怒灰烛会 | 夜间伏击增加，梦境污染出现，关键 NPC 被威胁 |
| 获得贵族支持 | 上城区通行更容易，但贫民区 NPC 对玩家态度变冷 |

## 9. MVP 实现范围

第一版建议先实现：

- 守灯人组织声望。
- 旧钟交易会声望。
- 灰烛会关注度。
- 沉钟街居民网络声望。
- 圣辉教区净化服务。
- 兰伯特家族只作为后续入口，不做完整贵族政治。
- 黑帆同盟只作为海上篇预告，不做完整海盗声望。

暂时不做：

- 全国家级势力战争。
- 所有教会完整派系树。
- 所有贵族家族关系网。
- 海上所有海盗团。
- 可加入邪教的完整路线。
- 动态经济和大规模阵营战争。

## 10. 总结

NPC 与势力系统应服务于玩家的选择，而不只是世界观列表。

第一卷中，玩家在这些势力之间移动：

```text
普通人案件
  ↓
守灯人组织审查
  ↓
旧钟交易会和黑市情报
  ↓
灰烛会污染事件
  ↓
圣辉教区净化与救助
  ↓
贵族和海盗线索露出
  ↓
根据玩家选择结算声望、关系和下一卷入口
```

后续章节只需要沿用本格式，逐步追加新城市、新教会、新海盗团、新贵族家族和新地下组织。
