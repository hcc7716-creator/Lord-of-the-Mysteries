## 一句话核心

《诡秘之主》是一部克苏鲁式神秘学、蒸汽工业、教会政治与超凡成长结合的长篇网络小说：现代人周明瑞在一次转运仪式后醒来，成为鲁恩王国廷根市的克莱恩·莫雷蒂，并在追查自身穿越真相的过程中逐步卷入诸神、旧日、源质与末日危机。

## 主线梳理

### 开局：穿越、廷根与值夜者

周明瑞在地球上进行转运相关仪式后，醒来成为克莱恩·莫雷蒂。原身死于离奇枪击，房间里留下与神秘学相关的线索。克莱恩为了弄清自身处境，一边维持普通人的生活，一边接触到鲁恩王国官方超凡组织“值夜者”。

他最早进入“占卜家”途径，学习魔药、仪式、封印物、失控风险和“扮演法”。廷根篇的表层是刑侦与神秘案件，深层是笔记、邪神、幕后操纵与值夜者小队的牺牲。邓恩·史密斯等人的死亡成为克莱恩后续复仇和成长的情感核心。

### 灰雾与塔罗会

克莱恩发现自己能进入灰雾之上的神秘空间，并借此伪装成高位存在“愚者”。他召集奥黛丽、阿尔杰等人成立塔罗会。塔罗会初期是信息交换和资源交易平台，随后逐步成为跨国家、跨组织、跨途径的隐秘势力。

塔罗会在结构上很适合游戏化：它既是任务发布中心，也是情报网络、角色培养系统、阵营关系系统和主线推进装置。

### 贝克兰德：身份伪装与大都市阴谋

克莱恩离开廷根后进入鲁恩首都贝克兰德，以夏洛克·莫里亚蒂等身份活动。他追踪因斯·赞格威尔，接触军情九处、教会、贵族、地下超凡圈和各种秘密组织。贝克兰德篇突出雾霾、贫富分化、贵族政治、邪教献祭和工业时代都市恐怖。

这一阶段克莱恩掌握更多非凡战斗与调查手段，并意识到官方教会、王室、邪神教团、秘密组织之间存在错综复杂的利益冲突。

### 海上与冒险：格尔曼·斯帕罗

克莱恩以疯狂冒险家“格尔曼·斯帕罗”的身份活跃于海上。他在五海、殖民地、拜亚姆、海盗势力和反抗组织之间活动，接触罗塞尔遗留信息、死神相关线索、灵界旅行和更高层级的封印物。

这一阶段的叙事重心从城市调查转向冒险、赏金、海盗、秘宝和多地探索。格尔曼身份强化了主角“多马甲”的玩法潜力：同一个玩家角色可以在不同身份下拥有不同声望、可进入区域和阵营关系。

### 历史真相：罗塞尔、古神与旧日

随着序列提升，克莱恩逐渐知道世界并不是传统意义上的异世界。罗塞尔大帝留下的日记、远古遗迹、白银城与神弃之地线索共同指向一个更大的真相：当下世界与地球、前文明、源质、旧日支配者和末日危机紧密相连。

小说后半段把“个人求生”扩展为“文明存续”：序列、神位、唯一性、源质、锚、信仰和旧日污染共同构成高位超凡体系。角色越接近神明，越需要处理人性、锚定和疯狂侵蚀。

### 高序列与结局：愚者、阿蒙与沉眠

克莱恩最终走向“愚者”相关的高位竞争，与阿蒙、亚当、查拉图等强者周旋。他通过身份、仪式、历史投影、欺诈与同伴支援完成升格，成为序列 0“愚者”。

结局的核心不是简单胜利，而是主角为了抵抗体内更古老意志的复苏而进入沉眠。塔罗会和“愚者”教会延续其影响，世界仍面对外神和末日压力，为后续故事留下空间。

## 世界观设定

### 时代与风格

- 表层时代近似 19 世纪维多利亚工业社会：蒸汽机、枪械、报纸、侦探、工厂、雾霾、贫民区、贵族沙龙。
- 深层世界是神秘学社会：教会、封印物、秘密组织、仪式魔法、灵界、梦境、邪神污染与不可直视知识。
- 风格基调兼具调查悬疑、都市恐怖、冒险奇谭、神话史诗和末日危机。

### 地理与重要区域

- 鲁恩王国：主角前中期主要活动国家，首都贝克兰德是政治、金融、教会与地下超凡圈中心。
- 廷根市：开局城市，值夜者小队所在地，情感与复仇线起点。
- 贝克兰德：大都市篇核心舞台，适合做成多城区 2D 箱庭地图。
- 五海与罗思德群岛：海盗、殖民、冒险和海上超凡者舞台。
- 拜亚姆：海上篇重要据点，殖民统治与本地反抗冲突明显。
- 神弃之地：被隔绝、缺乏阳光、怪物横行的末世区域，白银城长期在此求生。
- 白银城：神弃之地人类幸存聚落，承担“追寻光明”的群像线。
- 灰雾之上 / 源堡：克莱恩的核心金手指空间，也是塔罗会召开地点。
- 灵界、星界、梦境、历史迷雾：高阶超凡活动的重要空间。

### 超凡体系

- 非凡者通过服用魔药进入“序列”体系，从序列 9 逐步晋升到序列 0。
- 每条途径有固定晋升路线，魔药配方、主材料、辅助材料、晋升仪式和扮演要求共同限制成长。
- “扮演法”要求非凡者以符合魔药名称的方式行动，消化魔药，降低失控风险。
- 失控是核心代价：力量、知识、污染、情绪和外神影响都可能让人变成怪物或丧失自我。
- 非凡特性守恒与聚合让死亡、晋升、封印物和高阶竞争具有资源循环逻辑。
- 高序列涉及“唯一性”“锚”“信仰”“神国”“源质”和“旧日”层级。

## 主要角色

### 克莱恩·莫雷蒂 / 周明瑞

- 原身份：现代地球人周明瑞。
- 穿越身份：鲁恩王国廷根市大学毕业生克莱恩·莫雷蒂。
- 主要马甲：愚者、夏洛克·莫里亚蒂、格尔曼·斯帕罗、道恩·唐泰斯、梅林·赫尔墨斯等。
- 成长路径：从低序列“占卜家”逐步晋升到序列 0“愚者”。
- 核心动机：活下去、回家、守护家人与同伴、为廷根牺牲者复仇、阻止末日危机。
- 人物弧线：普通人求生者 -> 官方小队成员 -> 隐秘组织首领 -> 多身份操盘手 -> 高位神明 -> 为保留自我而沉眠。

### 塔罗会成员

- 愚者：克莱恩伪装的高位存在，塔罗会核心。
- 正义：奥黛丽·霍尔，贵族少女，观众途径，负责心理、贵族圈与人性视角。
- 倒吊人：阿尔杰·威尔逊，风暴教会成员，老练、现实，提供海上与教会情报。
- 太阳：戴里克·伯格，白银城少年，连接神弃之地与主世界。
- 魔术师：佛尔思·沃尔，作家，学徒/门相关途径，连接亚伯拉罕家族和贝克兰德日常线。
- 月亮：埃姆林·怀特，血族，药剂与血族社会线。
- 隐者：嘉德丽雅，海上强者，连接罗塞尔、贝尔纳黛和神秘知识线。
- 星星：伦纳德·米切尔，克莱恩旧友，值夜者线延续。
- 审判：休·迪尔查，军情与执法相关线。
- 世界：克莱恩操控的另一个席位，主要用于让塔罗会成员与“格尔曼”发生交易和行动联系。

### 关键盟友与中立者

- 邓恩·史密斯：廷根值夜者队长，早期情感核心。
- 伦纳德·米切尔：廷根值夜者成员，后期成为塔罗会“星星”。
- 老尼尔：早期神秘学启蒙角色，体现失控代价。
- 阿兹克·艾格斯：死神相关高位存在转世，主角重要引路人。
- 莎伦与马里奇：节制派相关盟友，承担地下行动线。
- 蕾妮特·缇尼科尔：灵界信使，高阶盟友。
- 威尔·昂赛汀：命运相关高位存在，提供关键帮助。
- 贝尔纳黛·古斯塔夫：罗塞尔之女，连接罗塞尔遗产和海上高阶势力。
- 罗塞尔·古斯塔夫：前穿越者、皇帝、日记作者，其遗产贯穿世界真相线。

### 主要对手

- 因斯·赞格威尔：廷根悲剧重要幕后者，主角复仇线核心敌人。
- 阿蒙：错误途径高位存在，寄生、欺诈和规则漏洞能力极强，是终局最重要对手之一。
- 亚当：观众/空想家相关高位存在，擅长布局和叙事操控。
- 查拉图：愚者相关高位竞争者。
- 乔治三世：鲁恩王室成神计划核心人物。
- 真造物主与极光会：邪神信仰与污染相关主线敌对势力。
- 玫瑰学派、魔女教派、秘修会等：在不同阶段提供案件、组织冲突和高阶威胁。
- 外神及旧日力量：最终层级的末日威胁，不是传统单一反派，而是宇宙尺度压力。

## 组织与阵营

### 官方与教会

- 黑夜教会 / 值夜者：早期主角所属官方超凡组织，负责处理夜晚、梦境、灵体、封印物等案件。
- 风暴教会 / 代罚者：海洋与风暴相关官方组织，阿尔杰所属体系。
- 蒸汽与机械教会 / 机械之心：偏技术、机械、炼金与封印物研究。
- 永恒烈阳教会、知识与智慧教会、大地母神教会、战神教会：共同构成正神教会格局。

### 隐秘组织

- 塔罗会：克莱恩借灰雾建立的信息与行动网络。
- 极光会：真造物主信徒组织，常与疯狂、献祭、污染相关。
- 心理炼金会：观众途径相关组织，涉及心理与社会操控。
- 密修会：愚者相关途径的重要组织，查拉图相关。
- 玫瑰学派：被缚者/放纵派与节制派冲突相关。
- 魔女教派：魔女途径，灾祸、阴谋与政治渗透。
- 亚伯拉罕家族：门途径家族，受诅咒与门先生影响。
- 摩斯苦修会、生命学派、血族等：补充不同途径、种族和知识体系。

## 可转译成 2D 游戏的设计点

### 最适合借鉴的玩法骨架

- 调查循环：接案 -> 收集线索 -> 占卜/询问/潜入 -> 识别超凡来源 -> 净化或封印。
- 序列成长：低阶职业树 -> 魔药材料 -> 晋升仪式 -> 新能力 -> 更高风险。
- 多身份系统：玩家可切换公开身份、地下身份、海上身份、组织领袖身份，影响地图权限与 NPC 反应。
- 理智/污染系统：越使用高危能力越强，但越接近失控。
- 隐秘集会：类似塔罗会的周期性会议，可做成资源交易、队友派遣、情报整合和主线推进。
- 阵营声望：教会、王室、地下组织、海盗、殖民地、反抗势力之间互相制衡。
- 封印物装备：强力道具附带明确副作用，形成高风险构筑。
- 城市箱庭：雾都大都市、多城区、日夜循环、报纸系统、案件链和随机神秘事件。

### 应原创替换的高风险元素

- 不直接使用“愚者、塔罗会、序列、非凡者、灰雾、源堡、阿蒙、克莱恩”等强识别名词。
- 不照搬 22 条途径与序列名称，可以保留“阶梯式超凡职业 + 扮演/代价”的抽象机制。
- 不照搬廷根灾难、贝克兰德大雾霾、阿蒙决战等标志性剧情。
- 不使用塔罗牌席位作为组织结构的唯一核心，可改成钟表、剧院、档案馆、炼金行会、秘密电台等原创隐喻。
- 不照搬正神教会和外神格局，可重做宗教、科技、梦境、星象或民俗体系。

### 可以安全学习的抽象原则

- “力量必须付代价”：每种能力都带来心理、社会或身体风险。
- “信息也是资源”：调查、占卜、报纸、密信、仪式记录都能成为游戏货币。
- “身份是玩法”：玩家的面具越多，可进入的社会层级越多，但暴露风险越高。
- “组织驱动剧情”：比单线任务更适合长线推进，玩家可逐步经营自己的隐秘网络。
- “神秘不可完全解释”：保留未知感比把设定全做成数值表更有味道。

## 后续项目建议

1. 先确定游戏核心循环：城市调查、动作冒险、卡牌构筑、RPG 成长或叙事解谜。
2. 设计一套原创超凡体系：保留“阶梯成长 + 风险代价”，但更换所有名称、材料、仪式和终局神话。
3. 先做一个可玩切片：一座城市、一个案件、三条能力路线、两个阵营、一个失控结局。
4. 把本档案转成两份新文档：`original-world-bible.md` 和 `game-design-document.md`，避免开发过程中继续依赖原作专有设定。
