# 任务与剧情系统设计文档：第一卷

> 文档用途：定义第一卷主线任务、任务字段格式、剧情分支、消化度与污染联动。  
> 适用范围：区域开放世界剧情 RPG 的主线任务、支线任务、案件系统、NPC 对话、地图解锁、技能教学和魔药晋升流程。  
> 当前范围：只写第一卷“雾城疑案”。后续城市、海上区域、灵界和神弃之地暂不展开，等开发需要再追加。

## 1. 写作原则

本任务文档不追求一开始写完整所有剧情，而是先把第一卷做成可以开发、可以拆任务、可以接系统的剧情骨架。

第一卷需要完成三件事：

1. 让玩家理解这个世界不是普通蒸汽都市，而是有非凡、污染、仪式和失控存在。
2. 让玩家完成从普通人到序列 9 占卜家的第一次晋升。
3. 让玩家在结尾发现雾城只是更大阴谋的入口，从而自然前往下一座城市。

剧情推进要遵循“案件驱动”：

- 玩家先遇到异常事件。
- 通过调查获得线索。
- 用占卜、灵视、仪式验证线索。
- 接触官方非凡组织和地下神秘学网络。
- 面对邪教、失控者和污染源。
- 通过魔药晋升获得新能力。
- 发现更高层隐秘，但不在第一卷完全解释。

## 2. 任务文档格式

每个任务至少包含以下字段：

| 字段 | 说明 |
| --- | --- |
| 任务 ID | 用于数据表和任务系统，例如 `V01-MQ-01` |
| 任务名称 | 玩家任务日志中显示的名称 |
| 触发条件 | 玩家何时接到任务 |
| 任务目标 | 可以被系统追踪的目标 |
| 涉及 NPC | 关键对话、敌人、导师或委托人 |
| 涉及地图 | 任务发生的区域或副本 |
| 奖励 | 经验、金钱、材料、技能、情报、消化度 |
| 失败条件 | 任务失败、进入分支或增加代价的条件 |
| 剧情分支 | 玩家选择导致的结果差异 |
| 是否影响消化度 | 是否增加、减少或解锁消化度 |
| 是否增加污染 | 是否可能增加污染值 |

建议 ID 命名：

| 类型 | 示例 | 说明 |
| --- | --- | --- |
| 主线任务 | `V01-MQ-01` | 第一卷主线任务 |
| 支线任务 | `V01-SQ-01` | 第一卷支线任务 |
| 教学任务 | `V01-TU-01` | 第一卷系统教学任务 |
| 案件节点 | `V01-CA-01` | 案件调查节点 |
| 晋升任务 | `V01-AD-01` | 魔药调配和晋升任务 |

## 3. 第一卷概述：雾城疑案

第一卷发生在一座被灰雾、蒸汽管线、教堂钟声和煤烟覆盖的港口都市。玩家醒来后发现自己卷入一起神秘死亡事件：死者身上没有明显伤口，房间却布满奇怪符号、焦黑纸片和被打碎的镜子。

第一卷主线流程：

1. 玩家醒来，发现自己卷入神秘死亡事件。
2. 接触官方非凡组织。
3. 学习占卜、灵视、仪式。
4. 调查邪教与失控者。
5. 获得占卜家魔药。
6. 完成第一次晋升。
7. 发现更高层隐秘，进入下一座城市。

第一卷主题：

| 主题 | 说明 |
| --- | --- |
| 怀疑现实 | 玩家不知道自己为什么在案发现场醒来 |
| 神秘学入门 | 通过占卜、灵视、仪式认识世界规则 |
| 官方与地下 | 玩家同时接触官方组织和黑市情报 |
| 失控代价 | 邪教徒和失控者展示非凡力量的风险 |
| 第一次晋升 | 以占卜家魔药作为成长闭环 |
| 更大阴谋 | 第一卷结尾只揭示冰山一角 |

### 3.1 第一卷核心剧情变量

第一卷不需要一开始做复杂声望系统，但建议先记录几个隐藏或半公开变量，让任务分支能影响后续关卡、NPC 对话和结尾评价。

| 变量 | 建议字段 | 区间 | 作用 |
| --- | --- | --- | --- |
| 官方信任度 | `official_trust` | 0-5 | 影响官方非凡组织是否提供安全屋、材料、档案权限和支援 |
| 地下声望 | `underground_reputation` | 0-5 | 影响黑市、书店、走私者和地下线人的交易价格与情报质量 |
| 邪教关注度 | `cult_attention` | 0-5 | 影响伏击、假线索、污染诱导事件和最终仪式难度 |
| 市民恐慌度 | `citizen_panic` | 0-5 | 影响街区封锁、普通 NPC 存活、物价和警力分布 |
| 罗莎关系 | `rosa_relationship` | 0-5 | 影响死者家庭线索、情感支线和卷尾告别 |
| 案件真相完整度 | `case_truth_progress` | 0-100 | 影响玩家能否在卷尾看懂黑伞人与灰烛会背后的更高层隐秘 |

MVP 中这些变量可以先不显示具体数值，只在任务结算中给出文字反馈，例如“官方开始信任你”“灰烛会似乎注意到了你的调查”“街区居民正在恐慌”。等任务系统稳定后，再决定是否做成可见声望条。

## 6. 第一卷主线任务

### V01-MQ-01

任务 ID：`V01-MQ-01`  
任务名称：雾中醒来  
触发条件：新游戏开始，玩家在案发公寓醒来。  
任务目标：

- 检查房间内的尸体。
- 找到自己的随身物品。
- 阅读破损的案件纸条。
- 避开或应对第一批警员。
- 离开案发公寓。

涉及 NPC：

- 赫伯特・瓦尔
- 罗莎・温特
- 两名普通警员

涉及地图：

- 案发公寓
- 雾城下城区

奖励：

- 解锁案件日志。
- 获得物品：破损怀表、沾灰纸条。
- 解锁基础交互：调查、拾取、案件记录。

失败条件：

- 玩家在教学战斗或追逐中生命值归零。
- 玩家销毁关键线索，任务不会失败，但进入“证据不足”分支。

剧情分支：

- 若玩家主动等待警员进入，可通过对话进入警局审问线。
- 若玩家逃离现场，会被官方组织后续追踪。
- 若玩家隐藏尸体痕迹，罗莎后续信任度下降。

是否影响消化度：否。玩家尚未服食魔药。  
是否增加污染：可能。若玩家触摸污染符号，污染值 +5。

### V01-MQ-02

任务 ID：`V01-MQ-02`  
任务名称：死者留下的线索  
触发条件：完成 `V01-MQ-01`，并获得案件日志。  
任务目标：

- 前往市警局或下城区收集死者身份信息。
- 询问罗莎・温特。
- 调查案发前一晚的目击者。
- 找到“灰烛会”符号的第一次记录。

涉及 NPC：

- 赫伯特・瓦尔
- 罗莎・温特
- 下城区报童
- 酒馆老板

涉及地图：

- 市警局
- 下城区酒馆
- 案发公寓周边街巷

奖励：

- 获得线索：灰烛符号、午夜钟声、失踪账本。
- 解锁地图：老莫里斯书店。
- 少量金钱或调查补贴。

失败条件：

- 关键目击者在支线时间内离开城市。
- 玩家威胁罗莎，导致她暂时拒绝提供家庭线索。

剧情分支：

- 走官方路线：通过警局档案获得死者工作记录。
- 走地下路线：通过酒馆和黑市传闻获得灰烛会传言。
- 两条路线都能推进主线，但 NPC 信任度不同。

是否影响消化度：否。  
是否增加污染：一般不增加；若玩家强行阅读污染账本，污染值 +5。

### V01-MQ-03

任务 ID：`V01-MQ-03`  
任务名称：黑荆棘事务所  
触发条件：玩家发现灰烛会符号，或在调查中被异常影子追踪。  
任务目标：

- 与艾琳娜・格雷见面。
- 接受官方非凡组织的基础审查。
- 了解非凡者、魔药、失控和污染的基本概念。
- 选择是否临时协助官方调查。

涉及 NPC：

- 艾琳娜・格雷
- 官方组织书记员
- 赫伯特・瓦尔

涉及地图：

- 黑荆棘事务所
- 市警局

奖励：

- 解锁安全屋。
- 解锁理智值、污染值、灵性值 UI。
- 获得临时调查许可。
- 获得基础道具：净化火柴、空白纸张。

失败条件：

- 玩家在审查中隐瞒所有关键线索，会进入“被监视协助”分支。
- 玩家攻击官方成员，主线转入高压追捕状态，不建议 MVP 开放。

剧情分支：

- 坦白案发现场经历：艾琳娜信任度 +1。
- 隐瞒怀表和纸条：后续官方会发现，信任度 -1。
- 质疑官方组织：解锁更多世界观解释对话。

是否影响消化度：否。  
是否增加污染：否。

任务名称：占卜、灵视与仪式  
触发条件：完成 `V01-MQ-03`，并接受临时协助。  
任务目标：

- 在黑荆棘事务所完成灵视教学。
- 使用灵摆占卜判断案件物品关联。
- 学习基础净化仪式。
- 使用仪式检查破损怀表。

涉及 NPC：

- 艾琳娜・格雷
- 老莫里斯
- 官方组织书记员

涉及地图：

- 黑荆棘事务所仪式室
- 老莫里斯书店

奖励：

- 解锁技能：灵视。
- 解锁技能：灵摆占卜。
- 解锁基础仪式交互。
- 获得材料：银粉、蜡烛、低阶草药。

失败条件：

- 教学任务不直接失败。
- 若玩家连续错误操作仪式，会增加污染并触发艾琳娜警告。

剧情分支：

- 严格按步骤完成：获得“谨慎的学徒”评价。
- 擅自改动仪式材料：获得额外异常线索，但污染值 +10。
- 向老莫里斯购买黑市材料：提前解锁地下路线。

是否影响消化度：否。玩家还未晋升，但会解锁后续消化度系统说明。  
是否增加污染：可能。错误仪式污染值 +10。

- 雾港码头

任务目标：

- 在雾港码头追踪失控者。
- 使用灵视确认污染源。
- 在战斗中学习逃生、净化或控制思路。
- 决定击杀、封印或救援失控者。

涉及 NPC：

- 失控的码头工
- 艾琳娜・格雷
- 赫伯特・瓦尔
- 码头工家属

涉及地图：

- 雾港码头
- 废弃仓库

奖励：

- 解锁战斗教学：灵视识别弱点。
- 获得非凡材料线索。
- 官方组织信任度变化。
- 若成功封印，获得额外净化材料。

是否影响消化度：否。  
是否增加污染：可能。靠近污染核心或灵性透支时污染值 +5 到 +15。

是否增加污染：可能。黑市污染材料或非凡特性替代会增加污染值。

### V01-AD-01

任务 ID：`V01-AD-01`  
任务名称：第一次晋升  
触发条件：完成 `V01-MQ-06`，并准备好占卜家魔药材料。  
任务目标：

- 在安全屋调配占卜家魔药。
- 检查理智值、污染值、灵性值和锚点值。
- 服食魔药。
- 承受晋升幻觉。
- 完成第一次占卜家的自我确认。

涉及 NPC：

- 艾琳娜・格雷
- 老莫里斯
- 黑伞人

涉及地图：

- 黑荆棘事务所安全屋
- 晋升幻觉空间

奖励：

- 晋升为序列 9：占卜家。
- 解锁消化度系统。
- 强化技能：灵摆占卜、灵视。
- 获得第一条扮演法提示。

失败条件：

- 污染值过高时强行晋升，进入污染晋升分支。
- 调配材料严重错误，触发晋升失败或短暂失控。
- 理智值过低，晋升幻觉难度提高。

剧情分支：

- 安全晋升：官方信任度 +1，污染值不上升。
- 冒险晋升：提前获得异常线索，但污染值上升。
- 被黑伞人注视：解锁隐藏记忆片段，神秘知识承受度受到考验。

## 9. 消化度与污染联动

第一卷中，消化度只在玩家成为占卜家后正式启用。此前任务可以记录“潜在扮演行为”，但不直接显示消化度。

| 行为 | 消化度影响 | 污染影响 |
| --- | --- | --- |
| 用占卜辅助调查，而不是替代调查 | 占卜家消化度 +5 到 +10 | 无 |
| 正确解读象征线索 | 占卜家消化度 +5 | 无 |
| 错误占卜后仍固执相信 | 不增加或减少消化收益 | 污染值 +5 |
| 占卜高位存在或黑伞人本体 | 可能获得隐藏线索 | 污染值 +10 到 +20 |
| 完成案件复盘 | 占卜家消化度 +10 | 理智值小幅恢复 |
| 使用污染仪式物推进主线 | 无或少量增加 | 污染值持续上升 |

## 12. 总结

第一卷“雾城疑案”的目标不是把整个世界讲完，而是让玩家完成第一次完整的诡秘体验：

```text
醒来卷入案件
  ↓
调查神秘死亡
  ↓
接触官方非凡组织
  ↓
学习占卜、灵视、仪式
  ↓
追查邪教与失控者
  ↓
获得占卜家魔药
  ↓
完成第一次晋升
  ↓
以新的身份重新理解案件
  ↓
发现更高层隐秘并离开雾城
```

后续开发只需要沿用本任务格式，逐卷追加新的主线、支线、地图和 NPC，不需要一次性写完所有剧情。
