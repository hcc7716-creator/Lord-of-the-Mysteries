## 1. 使用说明

本档案分为三类资料：

- 魔药配方：记录序列名、主材料、辅助材料、晋升仪式。
- 非凡特性：记录世界观规则、析出/聚合/保存方式、已知外观样例。
- 封印物品：记录封印物等级、能力、负面效果和游戏化备注。

整理原则：

- 全途径配方索引来自公开维基模块 `Module:Sequence/standard` 的结构化资料。
- 序列中文名直接记录；材料名暂保留英文源名，后续可以逐条翻译成本项目统一中文。
- 原资料未列出主材料、辅助材料或仪式的条目，标注为“待补充/未公开”。
- 同序列非凡特性通常可作为核心材料替代来源，但使用特性晋升会带来更高精神烙印和失控风险。

## 2. 魔药体系基础

魔药是非凡者晋升的主要方式。一个标准魔药配方通常包含：

- 主材料：承载对应序列力量的核心材料，很多时候可以用同序列的非凡特性替代。
- 辅助材料：稳定状态、缓冲精神冲击、提高成功率，多数低序列辅助材料本身不一定有强灵性。
- 晋升仪式：从序列 5 开始通常需要仪式。仪式本质上用于平衡污染、稳定人格、建立锚点或满足序列象征。
- 扮演法：服食魔药后通过符合序列名称和象征的行为消化魔药，降低失控风险。

游戏化建议：

- 主材料可以作为任务核心奖励。
- 辅助材料可以由黑市、值夜者仓库、采集、委托任务获得。
- 非凡特性可以作为更危险但更直接的替代材料。
- 晋升仪式应设计成剧情任务，而不是单纯点击升级。

## 5. 非凡特性规则

### 5.1 基本定义

非凡特性是非凡能力的核心来源，源头与最初造物主相关。非凡者、非凡生物、封印物、神奇物品都可能容纳非凡特性。非凡者死亡后，体内非凡特性会析出，并通常带有原拥有者的精神烙印。

### 5.2 关键规律

| 规律 | 说明 | 游戏化表达 |
| --- | --- | --- |
| 不灭定律 | 非凡特性不会真正消失，非凡者死亡后会析出。 | 击败非凡敌人后掉落“特性/材料”，但带污染。 |
| 守恒定律 | 同一途径与序列的高层资源总量有限。 | 高序列晋升需要争夺唯一资源。 |
| 聚合定律 | 同途径、相邻途径或兼容途径之间会互相吸引，常表现为命运层面的巧合。 | 玩家持有特性越多，越容易遭遇相关事件或敌人。 |
| 精神烙印 | 死者残留意志会影响后来服食者。 | 使用特性晋升会增加失控风险，需扮演法/锚点压制。 |
| 高序列神性 | 序列 4 起包含更强神性与疯狂风险。 | 半神以上晋升必须做仪式与锚点系统。 |
| 特性活化 | 部分特性可拥有“活着”的性质，甚至形成有意识物品。 | 高级封印物可主动逃离、诱导、攻击或谈判。 |

### 5.3 保存与处理

- 生非凡特性比已经调配好的魔药更容易长期保存。
- 若非凡特性缺少正确封印，长时间接触环境或物品，可能与其融合，形成神奇物品或封印物。
- 特性存放环境需要定期更换；安全做法是 24 小时内更换一次，正确封印后可保存数月到数年。
- 已调配魔药不仅人类能吸收，非生命物品也可能“喝掉”魔药并变成封印物。

## 6. 封印物分级规则

封印物是拥有强大效果和明显负面代价的超凡物品，通常由非凡特性与物体融合、非凡者失控死亡后遗留，或工匠途径高阶非凡者制作而成。

| 等级 | 危险度 | 大致对应力量 | 使用限制 |
| --- | --- | --- | --- |
| 0 级 | 极度危险、最高机密 | 天使层级，约序列 1-2 | 不得询问、传播、描述或窥探；通常只封存在最高级地点。 |
| 1 级 | 高度危险 | 圣者层级，约序列 3-4 | 只能有限使用，需要主教、执事等高权限。 |
| 2 级 | 危险 | 中序列，约序列 5-7 | 可谨慎使用；一般需要队长/主教级权限。 |
| 3 级 | 相当危险 | 低序列，约序列 8-9 | 正式成员在多人行动中可申请使用。 |
| 未知级 | 未正式编号或资料不全 | 不定 | 按实际风险处理。 |

## 10. 详细设定版扩写

> 本节从索引版资料继续扩写为游戏设定版。当前先完成第一条途径：愚者/占卜家途径。后续途径确认格式后再继续追加。

### 10.1 愚者/占卜家途径 `Fool Pathway`

#### 10.1.1 途径定位

愚者途径是本项目最优先实装的主角途径，也是最适合承担新手教学、调查玩法、身份伪装和后期高阶奇迹玩法的途径。它的低序列从“占卜、预感、观察命运”开始，中序列逐渐转向“表演、替身、变脸、秘偶操控”，高序列则进入“历史、奇迹、命运愚弄、概念操控”的层级。

游戏中这条途径的核心体验不是正面输出，而是“提前知道、假装知道、制造误导、用替身和历史逃过必死局”。玩家使用愚者途径时，应该经常面临一个选择：是用占卜获得更安全的信息，还是冒着污染和暴露风险强行窥探更高层级的秘密。

适配地图：

- 鲁恩王国：廷根市、贝克兰德、地下黑市。用于占卜、值夜者调查、侦探委托、塔罗会和身份伪装。
- 五海区域：格尔曼·斯帕罗身份、秘偶和神奇物品玩法逐渐成熟。
- 神弃之地：通过塔罗会和灰雾间接影响白银城。
- 异空间：灰雾之上、历史迷雾是该途径的后期核心地图。

适合玩家类型：

- 喜欢调查、推理、伏笔、逃脱、隐秘操控的玩家。
- 不适合一开始就想正面砍杀的玩家。
- 后期强度很高，但前期需要谨慎、规划和信息管理。

#### 10.1.2 总体系统设计

愚者途径在游戏里可以拆成五条能力线：

| 能力线 | 代表序列 | 游戏作用 |
| --- | --- | --- |
| 占卜与预感 | 占卜家、小丑 | 获取线索、预警危险、降低任务失败率。 |
| 表演与误导 | 小丑、魔术师、无面人 | 战斗闪避、替身逃脱、身份伪装、对话欺骗。 |
| 替身与秘偶 | 魔术师、秘偶大师、诡法师 | 规避死亡、远程行动、控制敌人、队伍外置化。 |
| 历史与奇迹 | 古代学者、奇迹师 | 召唤历史投影、完成愿望、扭转关键事件。 |
| 命运愚弄 | 诡秘侍者、愚者 | 影响时间、历史和命运，进入终局规则层。 |

基础资源：

- 灵性：使用占卜、灵视、仪式和部分技能时消耗。
- 消化度：通过扮演法提升，决定晋升安全性。
- 失控风险：接触高位知识、频繁占卜、使用特性晋升、灰雾外窥探高位存在时上升。
- 神秘度：塔罗会和伪装高位存在时使用，代表“愚者”形象是否稳定。
- 历史锚点：后期进入历史迷雾、召唤投影和完成奇迹时使用。

#### 10.1.3 序列 9：占卜家 / Seer

基础说明：

- 游戏阶段：鲁恩王国 - 廷根市开局。
- 角色定位：调查辅助、新手非凡能力入口。
- 核心主题：尊重命运，但不能迷信命运。
- 主要使用者：克莱恩·莫雷蒂。

魔药配方：

- 主材料：拉瓦章鱼血液 10 ml；星水晶 50 g。
- 辅助材料：纯水 100 ml；夜香草液 13 滴；金薄荷叶 7 片；毒堇汁 3 滴；龙血草粉末 9 g。
- 材料来源设计：拉瓦章鱼血液和星水晶可由黑荆棘安保公司配给；辅助材料可由老尼尔教学任务、药剂店或值夜者仓库获得。

非凡特性：

- 外观：蓝色血液、虚幻气泡、柔软结晶、星空般光点。
- 游戏解释：这是低序列“命运感知”与“灵界联系”的结晶化表现。
- 获取方式：击败或收容占卜家途径低序列失控者后析出；也可作为魔药主材料替代。
- 风险：直接使用特性晋升会携带前任精神烙印，使占卜结果出现干扰词或幻听。

可用能力：

- 灵摆占卜：判断是/否、方向、物品联系和危险程度。
- 梦境占卜：进入短梦境片段，获得象征性线索。
- 纸笔占卜：在案件笔记中生成关键词提示。
- 灵视：观察气场、灵体痕迹、污染残留。
- 仪式入门：学会基础祈祷、净化、献祭和仪式布置。

游戏任务设计：

- 新手任务“成为占卜家”：在老尼尔指导下收集材料、调配魔药、服食并第一次开启灵视。
- 教学任务“占卜不是万能”：玩家对安提哥努斯笔记相关线索占卜时得到模糊或危险结果，系统提示高位事物会干扰占卜。
- 日常任务“占卜俱乐部”：为普通人占卜，提升魔药消化度。

失控表现：

- 占卜结果反复出现无意义符号。
- 灵视关闭后仍能看见残影。
- 对命运解释过度自信，触发错误推理选项。
- 高污染时会在案件面板出现“你已经知道答案了”之类的诱导文本。

#### 10.1.4 序列 8：小丑 / Clown

基础说明：

- 游戏阶段：廷根后段或贝克兰德前置。
- 角色定位：生存、闪避、表演、情绪压制。
- 核心主题：看见命运的残酷，却仍以笑容掩饰痛苦。

魔药配方：

- 主材料：成年霍纳奇斯灰山羊独角结晶；完整人脸玫瑰。
- 辅助材料：纯水 80 ml；曼陀罗/毒苹果汁液 5 滴；黑边太阳花粉末 7 g；金斗篷草粉末 10 g；毒堇汁 3 滴。
- 材料来源设计：主材料可作为廷根篇终局前后的高价值奖励；霍纳奇斯相关材料应与安提哥努斯家族笔记线绑定。

非凡特性：

- 外观：带蓝意的拇指大小球形血团，可违反常理地悬浮。
- 游戏解释：小丑特性表现为“身体控制、危险预感、情绪面具”的混合。
- 风险：特性残留会诱导角色以玩笑或表演掩盖真实情绪，严重时会出现情感割裂。

可用能力：

- 危险预感：敌人攻击前出现短暂提示。
- 身体协调：提高闪避、攀爬、平衡和近战回避。
- 表情控制：在审讯、伪装、塔罗会中降低暴露风险。
- 滑稽闪避：战斗中通过夸张动作规避致命打击。

游戏任务设计：

- “笑容练习”：在占卜俱乐部、街头表演或危险对话中扮演小丑。
- “廷根的笑”：终局后让玩家理解小丑卷名，把情绪压制与剧情悲剧绑定。

失控表现：

- 角色无法表现真实悲伤。
- 对话选项出现不合时宜的玩笑。
- 战斗中自动做出夸张闪避，但消耗更多灵性。
- 高风险时 UI 短暂出现小丑面具裂纹。

#### 10.1.5 序列 7：魔术师 / Magician

基础说明：

- 游戏阶段：贝克兰德篇开始成型。
- 角色定位：替身、位移、战斗技巧、逃生。
- 核心主题：魔术的重点不是超自然，而是误导观众。

魔药配方：

- 主材料：雾树人真实根茎；黑纹黑豹全部脊髓液。
- 辅助材料：纯水 60 ml；雾树人汁液 30 ml；水滴宝石粉末 3 g；幻草精油 4 滴。
- 材料来源设计：可由贝克兰德地下交易会、冒险委托或塔罗会交易获得。

非凡特性：

- 外观：灰白色拳头大小物体，沟壑状、柔韧，像取出的脑组织，触碰后变成黏稠液体。
- 游戏解释：特性强调“认知误导”和“替身技巧”，外观接近脑组织可暗示思维和欺骗。

可用能力：

- 纸人替身：抵消一次致命伤害或转移诅咒。
- 火焰跳跃：在火源之间短距离移动。
- 空气弹：低成本远程攻击。
- 魔术误导：在潜入和对话中制造短暂注意力偏移。
- 快速脱身：被包围时消耗纸人和灵性撤离。

游戏任务设计：

- “侦探的逃生课”：玩家被地下组织追击，必须用纸人、火源和误导逃离。
- “舞台与观众”：通过一次公开表演或伪装行动提升消化度。

失控表现：

- 替身与本体记忆短暂混淆。
- 火焰跳跃落点偏移。
- 纸人替身自动触发，浪费关键道具。
- 玩家越依赖误导，越容易在重要 NPC 面前失去可信度。

#### 10.1.6 序列 6：无面人 / Faceless

基础说明：

- 游戏阶段：贝克兰德中段。
- 角色定位：身份伪装、潜入、社会层级切换。
- 核心主题：当你能成为任何人时，要记得自己是谁。

魔药配方：

- 主材料：千面狩猎者变异垂体；人皮阴影特性。
- 辅助材料：千面狩猎者血液 80 ml；黑曼陀罗汁液 5 滴；龙牙草粉末 10 g；深海娜迦头发 3 根。
- 材料来源设计：千面狩猎者适合做成贝克兰德地下狩猎任务；深海娜迦头发可由五海交易提前引入。

非凡特性：

- 外观：半透明暗绿色胶状物，内部不时浮现不同面孔。
- 游戏解释：特性本身就是“身份流动”的具象化。

可用能力：

- 外貌重塑：改变面容、体型、声音和基础身份。
- 身份档案：记录已经扮演过的人，方便快速切换。
- 伪装通行：进入贵族沙龙、军情机关、地下聚会等限制区域。
- 情绪模仿：复制目标的语气和习惯，降低被识破概率。

游戏任务设计：

- “夏洛克身份”：正式建立侦探身份。
- “潜入贵族沙龙”：玩家必须选择合适身份、服装和谈吐。
- “我是谁”：长期使用伪装后触发自我认同检查。

失控表现：

- 镜子中出现不同脸。
- NPC 偶尔叫错玩家当前身份。
- 案件笔记中的主角名字被划掉重写。
- 高污染时玩家可能短暂忘记原本目标。

#### 10.1.7 序列 5：秘偶大师 / Marionettist

基础说明：

- 游戏阶段：贝克兰德后段到五海前置。
- 角色定位：控制、远程行动、危险侦察。
- 核心主题：控制别人越多，自己越不像人。

魔药配方：

- 主材料：古代怨灵粉尘；六翼石像鬼核心结晶。
- 辅助材料：苏尼亚岛黄金泉水 80 ml；德拉戈树皮 10 g；古代怨灵残余灵性；六翼石像鬼眼睛 1 对。
- 晋升仪式：在美人鱼歌声中服食魔药。
- 材料来源设计：美人鱼歌声可以作为五海区域前置大任务；古代怨灵和石像鬼适合放在陵墓、遗迹或封印地。

非凡特性：

- 外观：黑色无瞳眼睛。
- 游戏解释：眼睛象征“看见灵体之线”和“支配目标”。

可用能力：

- 灵体之线感知：看见目标与灵体层面的连接。
- 秘偶化进程：持续控制目标，使其逐步变成秘偶。
- 秘偶侦察：操控秘偶进入危险区域。
- 远程施法节点：通过秘偶释放部分能力。

游戏任务设计：

- “美人鱼之歌”：玩家必须在海上找到安全听见歌声的方式。
- “第一具秘偶”：引导玩家理解控制进程、道德风险和战斗用途。

失控表现：

- 玩家开始用“工具”而非“人”理解 NPC。
- 秘偶在非指令状态下模仿玩家动作。
- 灵体之线画面过载，导致战斗界面混乱。

#### 10.1.8 序列 4：诡法师 / Bizarro Sorcerer

基础说明：

- 游戏阶段：五海后段或高阶都市副本。
- 角色定位：半神层级、秘偶军团、舞台式战斗。
- 核心主题：人生、战斗和晋升都变成一场精心设计的演出。

魔药配方：

- 主材料：诡法邪怪主眼；灵界掠夺者真实魂体。
- 辅助材料：诡法邪怪血液 200 ml；灵界掠夺者粉尘 30 g；红发桦树皮 10 g；黄金葡萄藤 1 段；指甲大小自制橡胶面具。
- 晋升仪式：依靠自身力量和策略，在众多观众前组织盛大表演，杀死半神层次非凡生物，并在表演结束时服食魔药。

非凡特性：

- 外观：幽蓝单眼；或掌心大小人形物，内部有透明液体、气泡、黑光和蛆虫意象。
- 游戏解释：单眼承接秘偶大师的“看见和控制”，人形物承接“表演舞台上的假人”。

可用能力：

- 秘偶网络：同时操控多个秘偶。
- 位置替换：与秘偶交换位置以规避致命攻击。
- 舞台布置：提前在区域内安置秘偶、符号和逃生点。
- 半神压制：对低序列敌人形成强控制。

游戏任务设计：

- “盛大表演”：区域终局副本，玩家要先安排观众、舞台、诱饵、秘偶，再击杀半神敌人完成晋升。

失控表现：

- 玩家把整个城市视作舞台。
- 秘偶网络反向传来陌生人格。
- 战斗中出现“观众不够”“表演尚未完成”等强迫性提示。

#### 10.1.9 序列 3：古代学者 / Scholar of Yore

基础说明：

- 游戏阶段：神弃之地、历史迷雾、后期主线。
- 角色定位：历史投影、古代知识、跨时代召唤。
- 核心主题：不属于当前时代的人，才能真正触碰历史。

魔药配方：

- 主材料：福根之犬眼睛 1 对；雾之魔狼异变心脏 1 颗。
- 辅助材料：福根之犬血液 100 ml；雾之魔狼白霜结晶 30 g；大量真实古代历史记录。
- 晋升仪式：与现实隔绝至少 300 年，成为“不属于当前时代的历史”后服食魔药。

非凡特性：

- 外观：暗红燃烧眼球；由白雾丝线组成的奇异心脏。
- 游戏解释：眼睛代表回望历史，白雾心脏代表历史迷雾中的生命残响。

可用能力：

- 历史迷雾进入：进入特殊异空间地图。
- 历史投影召唤：召唤曾拥有过或了解过的人物/物品投影。
- 古代记录解析：读取真实历史，绕过伪造文献。
- 时间错位抗性：对部分命运、历史干涉有抵抗。

游戏任务设计：

- “成为历史”：不能真的让玩家等待 300 年，可通过神弃之地、灰雾、时间封印或特殊副本完成等价隔绝。
- “真实历史记录”：玩家需要收集古代文献、白银城记录和罗塞尔资料。

失控表现：

- 玩家无法分清当前时间和历史片段。
- 召唤出的投影出现错误版本。
- 地图上出现已经毁灭或尚未出现的建筑残影。

#### 10.1.10 序列 2：奇迹师 / Miracle Invoker

基础说明：

- 游戏阶段：高阶主线后段。
- 角色定位：愿望、奇迹、事件逆转。
- 核心主题：奇迹并非凭空而来，它需要积累、回应和代价。

魔药配方：

- 主材料：黑暗魔狼心脏 1 颗。
- 辅助材料：黑暗魔狼血液 300 ml；时之虫 1 条；星之虫 1 条。
- 晋升仪式：将一段被遗弃的历史带回当前时代。

非凡特性：

- 外观：类似透明水晶心脏，不时冒出含有幻象的气泡。
- 游戏解释：每个气泡都可表现为一个愿望、一个可能性或一段被遗忘的历史。

可用能力：

- 愿望积累：帮助 NPC 完成愿望，积累奇迹权重。
- 奇迹触发：在关键场景实现恢复、逃生、召回、修复等高阶效果。
- 历史回归：把失落地点、人物线索或物品短暂带回现实。

游戏任务设计：

- “被遗弃的历史”：将神弃之地、白银城或某个被封印区域重新带回当前世界。
- “愿望清单”：长期支线系统，玩家完成越多愿望，奇迹成功率越高。

失控表现：

- 愿望被扭曲实现。
- NPC 的小愿望挤占主线奇迹资源。
- 玩家开始相信自己能随意改写一切。

#### 10.1.11 序列 1：诡秘侍者 / Attendant of Mysteries

基础说明：

- 游戏阶段：终局前置。
- 角色定位：命运剧场、秘偶城镇、灵界区域创造。
- 核心主题：所有人都像秘偶一样生活时，谁还拥有真实命运。

魔药配方：

- 主材料：原资料未公开，可用诡秘侍者非凡特性作为核心要求。
- 辅助材料：九种灵界特产。
- 晋升仪式：建立一座只由秘偶组成的城镇，为每个秘偶设计命运轨迹，使其互动形成足够真实的生活图景，并在灵界产生对应区域。

非凡特性：

- 外观：半透明黑色天鹅绒幕布，疯狂吸收周围光线，逐渐透明虚幻。
- 游戏解释：幕布象征舞台、遮蔽和命运背后的操盘。

可用能力：

- 秘偶城镇：建立可进入的灵界区域。
- 命运轨迹设计：为秘偶设定行动逻辑，形成自动化情报和防御网络。
- 大范围隐秘：遮蔽区域、事件或身份。
- 高阶替身：让“死亡”“失踪”“被看见”出现可操控版本。

游戏任务设计：

- “秘偶之城”：终局前大型经营/仪式任务，玩家需要收集秘偶、设计身份、维持城镇真实感。
- “谁在生活”：检查每个秘偶命运是否自洽，失败会导致灵界区域崩塌。

失控表现：

- 玩家开始把真实 NPC 也纳入命运轨迹设计。
- 秘偶城镇中出现未被设计的居民。
- 灵界区域反向影响现实城市。

#### 10.1.12 序列 0：愚者 / Fool

基础说明：

- 游戏阶段：最终章。
- 角色定位：神位、命运愚弄、历史与时间层规则操控。
- 核心主题：愚弄时间、历史或命运，同时不能被更古老的意志吞没。

魔药/晋升核心：

- 主材料：愚者唯一性；三份诡秘侍者非凡特性。
- 辅助要求：至少掌控四分之一历史迷雾。
- 晋升仪式：愚弄时间、历史或命运一次。

非凡特性：

- 普通意义上的外观资料不适合低阶道具化。
- 游戏中应表现为“唯一性 + 高序列特性 + 历史迷雾控制权”的组合，而不是普通掉落物。

可用能力：

- 愚弄历史：让某段历史的结果在有限范围内被重新解释。
- 愚弄时间：制造时间错位、延迟结果或让事件看似已经发生。
- 愚弄命运：让既定结局偏转到另一条可能性。
- 高阶灰雾权柄：强化塔罗会、历史投影、秘偶、奇迹等系统。

游戏任务设计：

- “愚弄命运”：终局仪式战，不应做成普通 boss 战。玩家需要同时管理历史迷雾、塔罗会支援、锚点、人性和旧日意志侵蚀。
- “沉眠结局”：晋升不是纯胜利，结局应保留克莱恩为对抗更古老意志而沉眠的代价。

失控表现：

- 玩家 UI、存档、任务日志被历史改写。
- 已完成任务出现另一种版本。
- NPC 对玩家的称呼在“克莱恩”“愚者”“陌生人”之间变化。
- 若人性锚点不足，结局进入旧日意志复苏方向。

| 物品 | 关联方式 | 游戏用途 |
| --- | --- | --- |
| 安提哥努斯家族笔记 | 愚者途径高危知识与开局谜团 | 廷根篇主线核心，推动克莱恩接触值夜者。 |

| 源堡/灰雾之上 | 愚者途径最高层级空间和权柄核心 | 塔罗会、灰雾占卜、复活伏笔、终局高阶系统。 |

#### 10.1.14 本途径优先开发顺序

第一阶段只做序列 9 到序列 8：

- 占卜家：灵视、灵摆占卜、梦境占卜、案件提示、占卜俱乐部扮演。
- 小丑：危险预感、闪避、表情控制、廷根终局情绪表现。

第二阶段扩展序列 7 到序列 6：

- 魔术师：纸人替身、火焰跳跃、空气弹、逃生玩法。
- 无面人：身份系统、潜入、侦探身份、贵族沙龙伪装。

第三阶段扩展序列 5 到序列 4：

- 秘偶大师：灵体之线、秘偶控制、远程侦察。
- 诡法师：秘偶网络、舞台战斗、半神晋升任务。

第四阶段扩展序列 3 到序列 0：

- 古代学者：历史迷雾、历史投影。
- 奇迹师：愿望系统、奇迹资源。
- 诡秘侍者：秘偶城镇、灵界区域。
- 愚者：愚弄时间、历史、命运的终局仪式。

#### 10.1.15 本途径设计注意事项

- 不要把占卜做成万能搜索按钮。占卜应该给方向、象征、风险提示，而不是直接给答案。
- 小丑的核心不是搞笑，而是用笑容隐藏痛苦。
- 魔术师的核心不是法师炮台，而是误导、替身和逃生。
- 无面人一定要绑定身份风险，否则会变成普通换装系统。
- 秘偶大师以后，玩家会变强，但道德压力也要随之上升。
- 高序列能力必须与历史、命运、锚点、人性绑定，不能只做成伤害倍率。
- 愚者途径最重要的游戏感是“我早已准备好了”，而不是“我数值最高”。

### 10.2 错误/偷盗者途径 `Error Pathway`

#### 10.2.1 途径定位

错误途径是“偷窃、欺骗、漏洞、寄生、身份替换、时间混乱”的途径。它和愚者途径同属诡秘相关高危体系，适合做成游戏里的反派路线、潜入路线、黑市路线和高阶敌人机制。低序列偏盗窃、诈骗、解密；中序列开始偷取能力、梦境和身体控制权；高序列则进入寄生、规则漏洞、身份替换、命运木马和时间错误。

如果愚者途径的体验是“我提前准备好了”，错误途径的体验就是“规则本身有漏洞”。玩家或敌人使用这条途径时，不应该只是数值变强，而应该让任务、身份、时间、道具、对话选项甚至 UI 规则都出现被偷走或被替换的感觉。

适配地图：

- 鲁恩王国：地下黑市、贝克兰德诈骗案、偷盗者低序列敌人、帕列斯与伦纳德伏笔。
- 因蒂斯共和国：罗塞尔遗物盗窃、博物馆潜入、政治骗局。
- 五海区域：海盗诈骗、身份替换、神奇物品盗取。
- 神弃之地：阿蒙分身、寄生和高阶欺诈威胁。
- 异空间：时间混乱、命运河流、错误权柄和高阶终局机制。

适合游戏定位：

- 作为敌人：非常适合制造“玩家以为安全但实际被动手脚”的任务。
- 作为可选路线：适合潜入、偷窃、诈骗、破解、身份伪装玩法。
- 作为高阶 Boss：适合阿蒙式敌人，通过分身、寄生、单片眼镜、时间错误和命运替换制造压迫。

#### 10.2.2 总体系统设计

错误途径可以拆成五条能力线：

| 能力线 | 代表序列 | 游戏作用 |
| --- | --- | --- |
| 偷窃与转移 | 偷盗者、盗火人 | 偷取物品、状态、能力、行动机会。 |
| 欺骗与社交漏洞 | 诈骗师、欺瞒导师 | 伪造身份、诱导 NPC、绕过规则。 |
| 解密与破译 | 解密学者 | 破解密码、仪式、机关、封印规则。 |
| 梦境与寄生 | 窃梦家、寄生者 | 入侵梦境、偷取念头、附身、控制目标。 |
| 时间与命运错误 | 命运木马、时之虫、错误 | 替换身份、制造时间错位、利用现实规则漏洞。 |

基础资源：

- 盗窃点：用于偷取物品、能力、状态或行动机会。
- 暴露度：偷窃、诈骗、寄生被察觉时上升，达到阈值触发追捕或反制。
- 漏洞标记：解密或观察后在目标身上生成，可用于更高级偷取。
- 寄生槽：中高序列后用于记录已寄生目标、分身或临时容器。
- 时间误差：高序列资源，用于回避攻击、延迟结果、制造行动错位。

#### 10.2.3 序列 9：偷盗者 / Marauder

基础说明：

- 游戏阶段：鲁恩王国、地下黑市、新手敌人或可选潜入路线。
- 角色定位：偷窃、潜行、物品转移。
- 核心主题：偷走看得见的东西，也偷走别人对“安全”的信任。

魔药配方：

- 主材料：血斑黑蚊 1 只；食烛者核心 1 个。
- 辅助材料：他人血液 100 ml；不同人的指甲碎片 9 份；蓝宝石 1 枚；马鞭草粉末 10 g。
- 材料来源设计：血斑黑蚊可放在贫民区、下水道或黑市虫商；食烛者核心可来自夜间异常事件；他人血液和指甲碎片应带来道德风险或黑市交易风险。

非凡特性：

- 外观：半透明黑色断指。
- 游戏解释：断指象征“偷窃之手”和身体的一部分被非凡能力异化。
- 获取方式：击败低序列偷盗者、收容偷盗者失控者或从黑市购买。
- 风险：携带者更容易产生顺手牵羊冲动，背包中可能出现来历不明的小物件，也可能丢失自己的东西。

可用能力：

- 顺手牵羊：从 NPC 或场景中偷取小物品。
- 开锁直觉：提高开锁、解除简单机关的成功率。
- 偷取注意力：短时间降低目标警觉。
- 逃跑步伐：偷窃失败后获得短距离逃离加成。

游戏任务设计：

- “黑市小偷”：玩家追踪一个偷走魔药材料的低序列偷盗者。
- “断指特性”：首次见到非凡特性析出，让玩家理解“死亡后掉落能力核心”的规则。

失控表现：

- 自动偷走无价值物品，增加暴露度。
- 玩家背包中出现陌生物，触发巡警盘问。
- 对话中出现“拿走它”的诱导选项。
- 高污染时，玩家可能短暂失去某个道具，过一段时间才发现被自己藏起来。

#### 10.2.4 序列 8：诈骗师 / Swindler

基础说明：

- 游戏阶段：鲁恩黑市、贝克兰德地下聚会、因蒂斯政治沙龙。
- 角色定位：欺骗、话术、伪造交易、社交渗透。
- 核心主题：偷盗者偷物品，诈骗师偷判断力。

魔药配方：

- 主材料：人脸猪笼草 1 株；摄魂虫群幼虫 1 份。
- 辅助材料：纯水 100 ml；他人眼泪 20 ml；青金石 1 枚；白栗香膏 10 g。
- 材料来源设计：人脸猪笼草适合放在古老温室、地下植物店或费内波特森林交易；他人眼泪可以通过剧情委托获得，避免纯刷材料。

非凡特性：

- 外观：红色透明唇形结晶。
- 游戏解释：唇形体现话术、诱导、谎言和承诺。
- 风险：使用者会越来越依赖欺骗，真实承诺的可信度下降。

可用能力：

- 虚假承诺：让 NPC 暂时相信一个合理谎言。
- 交易误导：降低购买价格或提高出售价格，但增加黑市记仇风险。
- 伪造动机：在审问中给出可信但不完整的解释。
- 情绪钩子：诱导目标说出更多信息。

游戏任务设计：

- “假配方交易”：玩家需要识破诈骗师出售的伪造魔药配方。
- “谁骗了谁”：可选任务允许玩家反向诈骗黑市中介，获得材料但提高地下组织警惕。

失控表现：

- 玩家说真话时也被 NPC 怀疑。
- 交易界面出现虚假优惠。
- 部分对话选项会自动夸大承诺，后续形成债务任务。

#### 10.2.5 序列 7：解密学者 / Cryptologist

基础说明：

- 游戏阶段：贝克兰德、因蒂斯博物馆、地下陵墓、封印物机关。
- 角色定位：密码、语言、仪式结构、机关规则破解。
- 核心主题：所有秩序都可以被拆成符号，所有符号都可能藏着漏洞。

魔药配方：

- 主材料：斯芬克斯脑 1 个；成年摄魂虫群 1 份。
- 辅助材料：斯芬克斯血液 80 ml；摄魂虫群黏液 10 滴；月长石 1 枚；野玫瑰 1 朵；自设计密码 1 套。
- 材料来源设计：自设计密码可以做成玩家任务：需要在案件笔记中设计一套可被 NPC 使用的暗号系统。

非凡特性：

- 外观：散发雾气的无色晶体。
- 游戏解释：无色代表信息未被固定，雾气代表隐藏意义。

可用能力：

- 密码破译：破解日记、机关、地下组织暗语。
- 仪式拆解：看出仪式缺口、材料替代和危险节点。
- 符号联想：在案件面板中自动连接隐藏线索。
- 反伪造：识别假文件、假签名、假封印编号。

游戏任务设计：

- “罗塞尔展品暗码”：因蒂斯博物馆章节中破解展品编号。
- “封印规则漏洞”：在危险封印物场景中找到不违反规则的通行方式。

失控表现：

- 玩家把普通文字也看成密码。
- 案件面板出现过多无意义连线。
- 破解成功后留下“被破解者也看见了你”的反追踪风险。

#### 10.2.6 序列 6：盗火人 / Prometheus

基础说明：

- 游戏阶段：贝克兰德中后段、五海冒险、神奇物品争夺。
- 角色定位：偷取能力、偷取灵性、偷取机会。
- 核心主题：从神那里偷火，也从敌人那里偷走最关键的一瞬间。

魔药配方：

- 主材料：水晶线虫 1 条；披风幽影附着物 1 份。
- 辅助材料：新偷来的酒 100 ml；披风幽影残余粉末 10 g；黄水晶 1 枚；落叶松精油 10 滴；火种 1 份。
- 材料来源设计：“新偷来的酒”应该要求玩家在不被发现的情况下完成偷窃；火种可来自仪式、太阳途径物品或风暴后残火。

非凡特性：

- 外观：微型未点燃水晶火炬。
- 游戏解释：火炬代表被偷来的启示、能力和文明之火。

可用能力：

- 偷取能力：短时间偷取目标一个低中阶能力或技能效果。
- 偷取灵性：削弱敌人技能释放，补充自身资源。
- 偷取攻击机会：让敌人的下一次行动延迟或失败。
- 火种标记：在目标身上留下漏洞，便于后续偷取。

游戏任务设计：

- “偷火”：从一个强敌或封印物守卫身上偷取关键能力，用它打开无法通过的区域。
- “能力归还”：偷取能力后若不及时处理，会引发原主追踪或能力反噬。

失控表现：

- 偷来的能力不稳定，可能释放错误版本。
- 自身技能栏短暂混入不属于自己的技能。
- 过度偷取会让敌人和玩家的状态互相污染。

#### 10.2.7 序列 5：窃梦家 / Dream Stealer

基础说明：

- 游戏阶段：贝克兰德心理线、梦境世界、塔罗会支线。
- 角色定位：梦境入侵、情报偷取、潜意识操控。
- 核心主题：梦境是人最没有防备的房间。

魔药配方：

- 主材料：食梦鼠心脏 1 颗；堕落气息之灵 1 份。
- 辅助材料：食梦鼠血液 30 ml；去灵性化的堕落气息 30 ml；天青石 1 枚；薰衣草纯露 10 滴；因理想破灭而流下的他人眼泪 50 ml。
- 晋升仪式：成为至少 30 个人潜意识梦境中的关键反派或配角；服食魔药时这些梦境必须正在发生。
- 材料来源设计：他人“理想破灭之泪”应绑定剧情人物，不能当普通材料刷。

非凡特性：

- 外观：婴儿手掌，五根细长手指会随环境改变颜色；也可表现为苍白带阴影的小婴儿手。
- 游戏解释：婴儿手象征潜意识、梦境中的抓取和未成形人格。

可用能力：

- 偷取梦境片段：获得 NPC 深层秘密。
- 梦境潜入：进入目标梦境副本。
- 梦中反派：在梦中塑造恐惧形象，改变目标态度。
- 偷取念头：短暂获取目标下一步行动。

游戏任务设计：

- “三十个梦”：大型晋升任务，玩家需要让 30 个 NPC 在梦里记住同一个形象。
- “破碎理想”：收集关键 NPC 的梦境眼泪，推动人物支线。

失控表现：

- 玩家入睡后进入陌生人的梦。
- 梦中获得的信息真假混杂。
- 目标 NPC 白天会认出玩家在梦中的形象。

#### 10.2.8 序列 4：寄生者 / Parasite

基础说明：

- 游戏阶段：高阶贝克兰德、神弃之地、阿蒙相关章节。
- 角色定位：寄生、附身、分身、隐秘控制。
- 核心主题：你以为你是自己，其实你的念头里住着别人。

魔药配方：

- 主材料：夺魂者死后结晶 1 份；傀儡邪虫 1 只。
- 辅助材料：夺魂者血液 100 ml；傀儡邪虫分泌物 9 滴；紫水晶 1 枚；樱桃李汁 10 滴；被囚禁灵魂 1 个。
- 晋升仪式：说服九个目标自愿献出其拥有的一切来维持你。

非凡特性：

- 外观：内部有半透明蠕虫蠕动的钻石。
- 游戏解释：钻石是外表的纯净与坚固，蠕虫是内部寄生和替换。

可用能力：

- 寄生标记：悄悄附着在目标身上，读取或干扰行动。
- 临时附身：短时间控制低抵抗目标。
- 分身视角：通过寄生目标获得远程视野。
- 寄生逃生：本体受致命威胁时转移到预留寄生体。

游戏任务设计：

- “谁是阿蒙”：在一个小镇或队伍中找出被寄生者。
- “自愿献出”：晋升任务应强调心理操控和信任破坏，而不是暴力压制。

失控表现：

- 玩家行动前出现另一个意识的建议。
- 队伍成员对话被寄生意识打断。
- 存档名或角色名短暂变成陌生名字。

#### 10.2.9 序列 3：欺瞒导师 / Mentor of Deceit

基础说明：

- 游戏阶段：高阶区域、城市级规则任务。
- 角色定位：规则漏洞、秩序崩塌、非暴力破局。
- 核心主题：不需要打破规则，只要让规则自己互相矛盾。

魔药配方：

- 主材料：沼泽巨人双掌；亵渎祭司核心。
- 辅助材料：沼泽巨人血液 80 ml；亵渎祭司真菌帷幕 20 g；黄金 30 g；水堇纯露 10 滴；因晋升者欺骗而死的灵界怨魂 9 个。
- 晋升仪式：不使用暴力，寻找并利用规则漏洞，让一个不小于村庄的区域秩序暂时崩塌。

非凡特性：

- 外观：无色真菌帷幕，像一层窗帘。
- 游戏解释：帷幕遮住规则的真实结构，真菌暗示规则从内部被腐蚀。

可用能力：

- 规则漏洞：识别区域制度、封印、契约中的冲突点。
- 欺瞒导师：让 NPC 按照错误理解执行规则。
- 非暴力崩塌：通过文件、命令、谣言、程序漏洞制造区域混乱。
- 责任转移：让敌人承担自己规则造成的反噬。

游戏任务设计：

- “让城市自己开门”：不战斗，通过法律、身份、印章和调令让封锁区域开放。
- “村庄秩序崩塌”：晋升任务，玩家必须制造系统性漏洞，但控制伤亡。

失控表现：

- 玩家无法相信任何规则文本。
- 已签订的契约出现隐藏条款。
- 城市 NPC 行为逻辑短暂混乱。

#### 10.2.10 序列 2：命运木马 / Trojan Horse of Destiny

基础说明：

- 游戏阶段：终局前高阶身份战。
- 角色定位：身份替换、命运侵入、长期伪装。
- 核心主题：木马不是强行攻破城墙，而是让城门主动打开。

魔药配方：

- 主材料：巨人王庭祭司死后结晶，或夺魂者之王竖眼。
- 辅助材料：古老河流水 100 ml；灵之虫 1 条；星之虫 1 条；大量被相信为真的虚假历史记录。
- 晋升仪式：不依赖非凡能力，通过伪装、欺骗等方式取代他人身份，并获得其社交圈信任直到该身份自然死亡；高位目标可缩短仪式时间。

非凡特性：

- 外观：拥有多个镜面切面的晶体，映照不同身份。
- 游戏解释：每个镜面都是一种被社会承认的身份，晶体本体则是隐藏的入侵者。

可用能力：

- 命运植入：让一个身份被世界短暂承认为真实。
- 社交圈继承：继承目标部分关系网和权限。
- 伪史写入：让虚假经历在有限范围内变得可信。
- 身份死亡：主动结束一个身份，转移因果追踪。

游戏任务设计：

- “活成另一个人”：长期潜入任务，玩家要在不使用非凡能力的情况下维持身份。
- “虚假历史档案”：收集并传播伪史，使其成为晋升材料。

失控表现：

- 玩家真实身份的记忆被替换。
- 队友对玩家的记忆出现多个版本。
- 地图权限错乱，某些门承认假身份却拒绝真身份。

#### 10.2.11 序列 1：时之虫 / Worm of Time

基础说明：

- 游戏阶段：终局高阶敌人、阿蒙相关核心。
- 角色定位：时间错位、分身、冻结、延迟、回溯感。
- 核心主题：当时间也能被偷走，死亡和胜利都不再可靠。

魔药配方：

- 主材料：奥尔米尔右眼。
- 辅助材料：古老时钟 1 个；使用过的历书 1 本；隐秘命运片段 1 段。
- 晋升仪式：让一座城市陷入至少七天的时间混乱。

非凡特性：

- 外观：光与影组成的水晶柱，内部有快速游动的十二环节时之虫，附近一切仿佛冻结。
- 游戏解释：十二环节可用于表现时间分段、分身和循环。

可用能力：

- 时间偷取：偷走目标一回合、一段冷却或一次行动结果。
- 时间冻结：小范围冻结场景或敌人行动。
- 时间分身：制造多个不完全同步的自身。
- 延迟死亡：把伤害结果延后结算。

游戏任务设计：

- “七日错城”：城市中每天循环错位，NPC 记忆和任务状态不一致。
- “单片眼镜”：每次出现都让玩家怀疑当前时间线是否真实。

失控表现：

- 任务日志日期混乱。
- 同一 NPC 在不同地点同时出现。
- 玩家刚获得的物品显示为“早已失去”。
- 战斗回合顺序被打乱。

#### 10.2.12 序列 0：错误 / Error

基础说明：

- 游戏阶段：终局神位竞争。
- 角色定位：漏洞权柄、神位偷取、替换仪式。
- 核心主题：成神不是自己完成仪式，而是在别人的仪式中成为那个错误。

魔药/晋升核心：

- 主材料：错误唯一性；三份时之虫非凡特性。
- 辅助要求：从命运河流中提取的三座信标。
- 晋升仪式：在他人的序列 0 成神仪式中取代对方，不一定是对方喝下魔药的一刻，而是在整个仪式过程中完成替换。

非凡特性：

- 外观：曾表现为水晶单片眼镜，后可表现为黑色手套。
- 游戏解释：单片眼镜用于高压恐惧符号；黑色手套可代表偷窃、替换和概念化错误。

可用能力：

- 概念偷取：偷走身份、距离、时间、能力、结果等抽象概念。
- 仪式替换：在敌人完成关键仪式时插入自身结果。
- 漏洞权柄：让规则中的例外变成主规则。
- 错误降临：使局部世界进入不合逻辑但自洽的状态。

游戏任务设计：

- “成为错误”：终局仪式战，玩家不是正面打败目标，而是发现仪式中唯一可替换的点。
- “信标收集”：从命运河流中提取三座信标，每座信标对应一段身份、时间或因果。

失控表现：

- 游戏规则被反转，例如治疗造成伤害、门通向错误地点、任务完成条件被偷走。
- 存档、地图、角色名出现错误版本。
- 玩家越了解错误，错误越了解玩家。

#### 10.2.13 途径专属封印物与神奇物品关联

| 物品 | 关联方式 | 游戏用途 |
| --- | --- | --- |
| 单片眼镜 | 阿蒙与错误途径最强视觉符号 | 高阶敌人出现前的恐惧提示，看到单片眼镜就代表可能被寄生或替换。 |
| 时之虫 | 序列 1 特性与阿蒙分身基础 | 高阶材料、时间混乱副本、错误途径晋升核心。 |
| 断指 / Broken Finger | 低阶偷盗者特性物品化 | 提升偷窃能力，同时诱发偷窃癖。 |
| 万能钥匙 / Master Key | 门途径物品，但与漏洞和迷路风险可联动 | 作为错误/门相邻体系的探索道具，满月风险可触发隐藏任务。 |
| 0-08 | 不是错误途径物品，但与“巧合、漏洞、叙事操控”有玩法互文 | 可用于设计“玩家以为是错误途径，实际是 0-08 操控”的误导。 |

#### 10.2.14 本途径优先开发顺序

第一阶段做低阶敌人与黑市玩法：

- 偷盗者：偷物品、开锁、偷注意力。
- 诈骗师：对话欺骗、假交易、黑市骗局。
- 解密学者：密码、机关、封印规则破解。

第二阶段做贝克兰德/因蒂斯潜入玩法：

- 盗火人：偷能力、偷行动机会。
- 窃梦家：梦境潜入、偷念头。

第三阶段做高阶反派玩法：

- 寄生者：附身、寄生、找出被替换的人。
- 欺瞒导师：规则漏洞、城市秩序崩塌。

第四阶段做终局/阿蒙相关玩法：

- 命运木马：长期身份替换。
- 时之虫：时间混乱、分身、延迟结算。
- 错误：神位替换、概念偷取、规则漏洞终局。

#### 10.2.15 本途径设计注意事项

- 错误途径不应该只是“盗贼职业”。它的核心是偷走规则、身份、时间和结果。
- 低序列可以偏潜行和社交，中高序列必须逐渐变得诡异。
- 寄生机制要有强烈不安全感：玩家不能轻易确认谁还是本人。
- 时间混乱不能过度影响可玩性，必须用清晰 UI 标记“当前时间层”。
- 阿蒙相关内容要谨慎使用，不要过早让玩家正面击败高阶阿蒙，否则会削弱压迫感。
- 偷能力玩法要限制时长、目标等级和副作用，否则会破坏所有职业平衡。
- 错误途径最适合做“任务规则被改写”的惊吓点，但要保证玩家事后能理解原因。

### 10.3 门/学徒途径 `Door Pathway`

#### 10.3.1 途径定位

门途径是“空间、旅行、记录、传送、放逐、星空”的途径。它非常适合承担地图解锁、跨区域旅行、隐藏房间、空间谜题和后期星界/灵界穿梭系统。低序列偏戏法、门锁、短距离移动和能力记录；中序列开始具备旅行能力；高序列进入星空、坐标、位面和门先生相关的危险。

游戏中这条途径的核心体验是“去不该去的地方，看见不该看的东西”。它既能让玩家快速穿梭，也必须带来迷路、满月低语、星空污染和坐标错误等风险。

#### 10.3.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 学徒、戏法大师、占星人 | 开锁、戏法、短距离移动、星象提示。 |
| 中序列 | 记录官、旅行家 | 记录能力、跨地图旅行、传送门。 |
| 半神 | 秘法师、漫游者 | 空间封锁、放逐、异空间副本。 |
| 天使以上 | 旅法师、星之匙、门 | 星界穿梭、位面坐标、门先生污染。 |

#### 10.3.3 材料与非凡特性设计

门途径材料应尽量分布在“空间异常地点”：废弃门廊、地下陵墓、灵界裂隙、满月仪式场、星象观测台。材料获取任务不要只是采集，最好设计成“找到正确入口”“记住坐标”“在错误房间里返回现实”等空间谜题。

非凡特性表现建议：

- 低序列特性可表现为钥匙、门框碎片、星光晶体、透明符号。
- 中序列特性可表现为空间折叠晶体、记录书页、漂浮坐标点。
- 高序列特性应与星空、门、坐标、无尽走廊和不可名状低语绑定。

#### 10.3.4 地图与任务适配

- 鲁恩王国：万能钥匙、非法潜入、地下黑市隐藏入口。
- 因蒂斯共和国：博物馆密室、机械机关门、罗塞尔隐藏展厅。
- 五海区域：远距离海岛传送、秘密岛屿入口。
- 神弃之地：无法正常抵达的遗迹、巨人王庭空间封锁。
- 异空间：灵界、星界、门先生污染、坐标迷失。

代表任务：

- “打不开的门”：玩家发现一扇只有满月时才存在的门。
- “记录一次能力”：记录官阶段要求玩家观察并成功记录一个敌方能力。
- “旅行家的坐标”：玩家必须建立安全坐标，否则传送到危险区域。
- “门后的低语”：高序列任务中，满月时会听见门先生的呼喊。

#### 10.3.5 失控与副作用

- 玩家更容易迷路，地图小地图出现错误房间。
- 传送门偶尔通向相邻但不正确的地点。
- 满月期间听见呼喊，灵性和理智持续下降。
- 记录能力失败时会记录到“错误版本”，释放后产生反噬。
- 高序列污染下，玩家会在门、窗、镜子中看见星空。

#### 10.3.6 开发优先级

第一阶段只做学徒到记录官：

- 开锁、短距离位移、戏法干扰、记录一次低序列能力。

第二阶段做旅行家：

- 跨区域旅行、传送坐标、逃离危险地图。

第三阶段做高阶：

- 空间封锁、星界门、门先生污染、星之匙坐标系统。

### 10.4 空想家/观众途径 `Visionary Pathway`

#### 10.4.1 途径定位

空想家途径是“观察、心理、梦境、暗示、集体潜意识、叙事操控”的途径。它适合做成社交、心理战、梦境副本、NPC 情绪管理和高阶剧情干涉系统。低序列偏观察和读心，中序列偏催眠和梦境行走，高序列则进入操控集体潜意识、编写命运故事和让想象成为现实。

这条途径的核心体验是“看懂人心，然后改变人心”。它不适合做普通输出职业，真正强处在于提前看穿谎言、引导 NPC、改变群体情绪、制造梦境和推动剧情走向。

#### 10.4.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 观众、读心者、心理医生 | 看情绪、读表层想法、治疗心理创伤。 |
| 中序列 | 催眠师、梦境行者 | 暗示、催眠、梦境潜入。 |
| 半神 | 操纵师、织梦人 | 操控人群、编织大型梦境。 |
| 天使以上 | 洞察者、作家、空想家 | 洞察本质、书写故事、让想象成真。 |

#### 10.4.3 材料与非凡特性设计

观众途径材料应更多来自“精神、眼睛、梦境、心理创伤”相关任务，而不是普通怪物掉落。比如曼哈尔鱼眼球、羊角黑鱼血液等低序列材料可以通过贵族渠道、心理炼金会交易或神秘学沙龙获得；高序列材料则应与梦境世界、集体潜意识和空想家遗迹绑定。

非凡特性表现建议：

- 低序列特性偏眼睛、透明晶体、情绪色彩。
- 中序列特性偏梦境碎片、催眠钟摆、精神结晶。
- 高序列特性应表现为书页、羽毛笔、故事线、虚幻国度。

#### 10.4.4 地图与任务适配

- 鲁恩王国：奥黛丽早期成长、贵族沙龙、心理暗示。
- 贝克兰德：心理炼金会、观众途径支线、贵族阴谋。
- 因蒂斯共和国：政治演说、群体心理和革命叙事。
- 梦境世界：梦境行者和织梦人核心地图。
- 异空间：集体潜意识海、空想家高阶任务。

代表任务：

- “贵族沙龙观察课”：玩家通过观察表情和语气判断谁在说谎。
- “读心边界”：读心者能听见表层想法，但会被训练过的人误导。
- “梦境治疗”：心理医生进入 NPC 创伤梦境，修复关键记忆。
- “故事开始了”：作家阶段让一个区域的人相信同一套叙事。

#### 10.4.5 失控与副作用

- 过度观察会失去共情，把人当成行为样本。
- 读心会听见过多杂音，造成 UI 噪声。
- 催眠失败会让目标产生心理抗性或反向暗示。
- 梦境行走过深时，玩家可能把梦境情绪带回现实。
- 高序列失控时，玩家写下的推测可能影响现实剧情，造成不可控后果。

#### 10.4.6 开发优先级

第一阶段做观众到心理医生：

- 情绪观察、读心提示、心理治疗支线。

第二阶段做催眠师和梦境行者：

- 暗示对话、梦境副本、心理防御。

第三阶段做高阶：

- 群体潜意识、叙事改写、空想成真终局机制。

### 10.5 太阳途径 `Sun Pathway`

#### 10.5.1 途径定位

太阳途径是“光明、净化、歌颂、公证、驱邪、神圣裁决”的途径。它适合承担反亡灵、反污染、驱散黑暗、治疗精神污染和白银城生存玩法。低序列偏赞歌和净化，中序列偏公证、神圣火焰和群体增益，高序列则进入逐光者、纯白天使和太阳权柄。

这条途径的核心体验是“用光照亮黑暗，但光也会灼伤”。它对死灵、邪灵、堕落污染、黑暗怪物非常有效，但也可能带来狂热、过度净化和对异端的强迫性审判。

#### 10.5.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 歌颂者、祈光人、太阳神官 | 增益、照明、净化、驱邪。 |
| 中序列 | 公证人、光之祭司 | 契约公证、神圣攻击、群体恢复。 |
| 半神 | 无暗者、正义导师 | 领域净化、规则裁决、反污染屏障。 |
| 天使以上 | 逐光者、纯白天使、太阳 | 大范围光明、神圣权柄、黑暗终结。 |

#### 10.5.3 材料与非凡特性设计

太阳途径材料应集中在白银城、神弃之地、永恒烈阳教会、古代太阳遗迹和净化仪式中。低序列材料可以通过教会、黑市或神弃之地狩猎获得；高序列材料必须绑定重大净化副本或神话级任务。

非凡特性表现建议：

- 低序列特性偏金色、白色、歌声、羽毛、火焰。
- 中序列特性偏徽章、圣杯、光辉结晶。
- 高序列特性偏太阳碎片、纯白羽翼、光之王冠。

#### 10.5.4 地图与任务适配

- 鲁恩王国：低阶净化封印物、邪灵案件。
- 神弃之地：白银城核心生存体系。
- 黑暗荒原：光源管理、怪物驱散。
- 巨人王庭：太阳、古神和神话真相。
- 南大陆：亡灵和死神污染净化任务。

代表任务：

- “第一束光”：在神弃之地维持光源，保护探索队。
- “公证契约”：公证人阶段让契约具备非凡约束力。
- “无暗领域”：清理一个被黑暗污染的区域。
- “逐光者”：带领白银城寻找真实阳光。

#### 10.5.5 失控与副作用

- 角色会变得过度热情、过度赞美太阳。
- 净化技能可能伤害被污染但仍可救的人。
- 黑暗环境中灵性消耗加倍，产生强烈焦虑。
- 高污染时 UI 出现过曝白光，难以分辨细节。
- 高序列狂热会把不同立场 NPC 直接判定为异端。

#### 10.5.6 开发优先级

第一阶段做歌颂者到太阳神官：

- 光源、净化、赞歌增益、亡灵感知。

第二阶段做公证人和光之祭司：

- 契约、群体净化、神圣攻击。

第三阶段做高阶：

- 无暗领域、逐光者路线、白银城迁徙终局。

### 10.6 暴君/水手途径 `Tyrant Pathway`

#### 10.6.1 途径定位

暴君途径是“海洋、风暴、雷霆、愤怒、航海、灾难”的途径。它适合支撑五海区域、海盗战、风暴教会、船只系统和大型自然灾害。低序列偏水性、体魄和愤怒爆发，中序列偏航海、风、雷和海洋法术，高序列则进入海王、天灾、雷神和暴君权柄。

这条途径的核心体验是“驾驭风暴，也被风暴驱使”。它战斗表现强烈、直接、粗暴，但副作用是情绪失控、冲动、破坏性强，容易把潜入任务变成公开冲突。

#### 10.6.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 水手、暴怒之民、航海家 | 水下行动、近战、海图、航海直觉。 |
| 中序列 | 风眷者、海洋歌者 | 风、雷、水法术，海战支援。 |
| 半神 | 灾难主祭、海王 | 大范围天气、海啸、风暴领域。 |
| 天使以上 | 天灾、雷神、暴君 | 天灾级破坏、雷霆权柄。 |

#### 10.6.3 材料与非凡特性设计

暴君途径材料应主要来自海洋生物、风暴遗迹、海盗战利品和风暴教会仓库。低序列材料可通过普利兹港、罗思德群岛和拜亚姆获得；中高序列材料应绑定危险海域、幽灵船、海怪和风暴仪式。

非凡特性表现建议：

- 低序列特性偏鱼鳞、气囊、海螺、蓝色结晶。
- 中序列特性偏风暴核心、雷霆纹路、海浪状晶体。
- 高序列特性偏王冠、三叉戟、雷云、海啸意象。

#### 10.6.4 地图与任务适配

- 鲁恩王国：普利兹港、风暴教会检查、出海前置。
- 五海区域：罗思德群岛、拜亚姆、幽灵船海域、秘密岛屿。
- 南大陆：殖民航线和海上冲突。
- 异空间：风暴领域、海神信仰、海洋权柄。

代表任务：

- “第一次出海”：玩家学习船只移动、天气、海图。
- “风暴教会审查”：处理与阿尔杰相关的教会身份风险。
- “海洋歌者”：用歌声影响海浪、船员士气和海怪。
- “天灾降临”：高阶任务中选择引导或阻止一场灾难。

#### 10.6.5 失控与副作用

- 情绪容易暴怒，对话选项出现挑衅和威胁。
- 水手在陆地干燥环境中状态下降。
- 风暴技能容易误伤队友和场景物件。
- 高阶天气能力可能改变整张地图状态。
- 长期使用会让 NPC 评价玩家“像风暴一样不可靠”。

#### 10.6.6 开发优先级

第一阶段做水手到航海家：

- 水下行动、海图、船只移动、近战爆发。

第二阶段做风眷者和海洋歌者：

- 风、雷、水法术，海战辅助，船员增益。

第三阶段做高阶：

- 海王领域、天灾天气、风暴权柄和大型海战。

### 10.7 白塔/阅读者途径 `White Tower Pathway`

#### 10.7.1 途径定位

白塔途径是“阅读、解析、推理、学习、复制、知识权柄”的途径。它非常适合做成游戏里的情报分析、弱点识别、法术复制、图书馆研究和百科式成长系统。低序列偏阅读和推理，中序列开始解析非凡能力，高序列进入知识导师、预言、全知和白塔权柄。

这条途径的核心体验是“知道即力量”。玩家并不一定正面最强，但只要收集足够信息，就能看穿敌人弱点、破解仪式、复制能力、找到最省成本的解法。

#### 10.7.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 阅读者、推理学员、侦探 | 快速阅读、线索推理、弱点识别。 |
| 中序列 | 博学者、秘术导师 | 学习法术、解析能力、指导队友。 |
| 半神 | 预言家、贤者 | 预言、知识领域、仪式破解。 |
| 天使以上 | 智天使、全知之眼、白塔 | 全知视野、知识权柄、规则解析。 |

#### 10.7.3 材料与非凡特性设计

白塔途径材料应与图书馆、大学、知识教会、古代文献、观察器官和智慧生物相关。低序列材料可以来自鲁恩大学、贝克兰德图书馆、知识与智慧教会；高序列材料应来自神话档案、隐秘知识库和星界记录。

非凡特性表现建议：

- 低序列特性偏书页、眼睛、透明晶体、墨水。
- 中序列特性偏书本、镜片、符号矩阵。
- 高序列特性偏知识之眼、白塔模型、全息书库。

#### 10.7.4 地图与任务适配

- 鲁恩王国：侦探推理、大学档案、值夜者报告。
- 因蒂斯共和国：罗塞尔博物馆、机械图纸、隐秘展厅。
- 费内波特王国：古老森林知识、生命学派资料。
- 神弃之地：白银城古代记录、巨人王庭真相。
- 异空间：知识投影、预言副本。

代表任务：

- “读完一屋档案”：限时阅读大量资料，找出唯一矛盾。
- “能力解析”：观察敌人三次后获得弱点提示。
- “复制一次法术”：秘术导师阶段临时学习队友或敌人的能力。
- “不完整预言”：预言家给出多个可能结局，玩家选择验证路径。

#### 10.7.5 失控与副作用

- 信息过载，案件面板出现过多推论。
- 玩家开始相信知识能解释一切，降低对神秘污染的警觉。
- 复制能力可能复制到副作用。
- 高序列失控时，角色会把人当成信息样本或变量。

#### 10.7.6 开发优先级

第一阶段做阅读者到侦探：

- 快速阅读、线索联想、弱点提示。

第二阶段做博学者和秘术导师：

- 能力解析、临时复制、队友教学。

第三阶段做高阶：

- 预言、多结局模拟、知识领域、白塔权柄。

### 10.8 倒吊人/秘祈人途径 `Hanged Man Pathway`

#### 10.8.1 途径定位

倒吊人途径是“秘祈、倾听、血肉、堕落、牧羊、暗影、真实造物主污染”的途径。它适合做邪教敌人、极光会任务、污染副本、血肉异变、能力吞噬和高风险力量路线。低序列是听取低语和祈祷，中序列开始血肉与堕落能力，高序列进入黑骑士、暗天使和倒吊人权柄。

这条途径的核心体验是“向高处祈求，得到的未必是救赎”。它力量强、代价重，非常适合做高风险高收益路线。

#### 10.8.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 秘祈人、倾听者、暗影苦修士 | 低语、暗影、邪神知识、污染侦测。 |
| 中序列 | 蔷薇主教、牧羊人 | 血肉魔法、吞噬灵魂、使用他人能力。 |
| 半神 | 黑骑士、三首圣堂 | 堕落领域、血肉军团、高阶污染。 |
| 天使以上 | 秽语长老、暗天使、倒吊人 | 亵渎语言、神性堕落、牺牲权柄。 |

#### 10.8.3 材料与非凡特性设计

倒吊人途径材料应与邪教仪式、血肉、暗影、亵渎教堂、真实造物主信仰和极光会据点绑定。材料获取必须有精神污染风险，不能像普通素材采集。

非凡特性表现建议：

- 低序列特性偏黑色肉块、耳朵、低语结晶。
- 中序列特性偏血肉花、羊皮、暗影核心。
- 高序列特性偏倒吊人影像、亵渎文字、血肉圣徽。

#### 10.8.4 地图与任务适配

- 鲁恩王国：真实造物主子嗣、极光会、梅高欧丝事件。
- 贝克兰德：邪教献祭、贫民区污染。
- 五海区域：海盗邪教、蠕动的饥饿。
- 神弃之地：堕落神庙、真实造物主线索。
- 异空间：污染低语和献祭仪式。

代表任务：

- “第一次听见”：玩家接触秘祈人后，低语进入 UI。
- “牧羊人战斗”：敌人使用多个被吞噬灵魂的能力。
- “血肉教堂”：地图本身会生长、封门、吞噬尸体。
- “牺牲还是救赎”：玩家必须决定是否使用污染力量救人。

#### 10.8.5 失控与副作用

- 听见真实造物主或高位存在低语。
- 技能越强，血肉异变越明显。
- 使用吞噬来的能力会继承目标精神残留。
- 祈祷可能招来回应，也可能招来注视。
- 高序列失控时，角色会把牺牲视为理所当然。

#### 10.8.6 开发优先级

第一阶段做秘祈人到暗影苦修士：

- 低语、暗影、污染感知、邪教敌人。

第二阶段做蔷薇主教和牧羊人：

- 血肉变形、能力吞噬、灵魂栏位。

第三阶段做高阶：

- 堕落领域、暗天使、真实造物主污染终局。

### 10.9 黑夜/不眠者途径 `Darkness Pathway`

#### 10.9.1 途径定位

黑夜途径是“黑暗、安眠、梦境、隐秘、恐惧、厄运、宁静”的途径。它是廷根值夜者体系的核心之一，适合做官方调查、夜间任务、梦境压制、隐秘封锁和黑夜教会高阶支援。低序列偏不眠和夜间行动，中序列进入梦魇、安魂和恐惧，高序列掌握隐秘、厄运和黑暗权柄。

这条途径的核心体验是“在黑暗里守夜”。它不是邪恶黑暗，而是用黑夜包容危险、遮蔽秘密、安抚亡者和保护普通人。

#### 10.9.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 不眠者、午夜诗人、梦魇 | 夜间行动、精神抗性、梦境控制。 |
| 中序列 | 安魂师、灵巫 | 安抚亡灵、梦境战斗、灵体沟通。 |
| 半神 | 厄运法师、隐秘之仆 | 厄运、隐秘、黑暗领域。 |
| 天使以上 | 恐惧主教、黑暗天使、黑夜 | 大范围隐秘、恐惧权柄、夜之国度。 |

#### 10.9.3 材料与非凡特性设计

黑夜途径材料应主要来自午夜花、梦境生物、墓地、黑夜教会仓库和夜间异常事件。低序列材料适合由值夜者系统配给；高序列材料则应绑定黑夜教会高层、隐秘天使、梦境与星界。

非凡特性表现建议：

- 低序列特性偏黑色羽毛、月光结晶、暗色眼睛。
- 中序列特性偏梦境碎片、安魂铃、灵体纱布。
- 高序列特性偏夜幕、隐秘帷幕、恐惧王冠。

#### 10.9.4 地图与任务适配

- 鲁恩王国：廷根值夜者、黑荆棘安保公司、查尼斯门。
- 贝克兰德：圣赛缪尔教堂、夜间封锁、梦境审查。
- 南大陆：墓地、亡灵、安魂任务。
- 异空间：梦境世界、黑夜隐秘领域。

代表任务：

- “守夜任务”：夜间巡逻，处理普通人看不见的灵异事件。
- “队长的梦”：梦魇阶段进入目标梦境审讯。
- “安魂仪式”：让亡者平静，避免转化为怨灵。
- “隐秘封锁”：将一次高危事件从公众认知中抹除。

#### 10.9.5 失控与副作用

- 长期不眠导致现实和梦境界限模糊。
- 玩家在白天状态下降，夜晚感知增强。
- 梦魇能力可能困住自己。
- 隐秘使用过度会让 NPC 忘记玩家的帮助。
- 高序列失控时，角色可能把所有真相都埋进黑夜。

#### 10.9.6 开发优先级

第一阶段做不眠者到梦魇：

- 夜间行动、梦境审讯、精神抗性。

第二阶段做安魂师和灵巫：

- 亡灵安抚、灵体沟通、梦境战斗。

第三阶段做高阶：

- 隐秘领域、厄运、黑夜教会神降相关内容。

### 10.10 死神/收尸人途径 `Death Pathway`

#### 10.10.1 途径定位

死神途径是“尸体、亡灵、灵魂、死亡、冥界、陵寝”的途径。它适合南大陆、拜朗遗迹、死神陵寝、亡灵对话和死亡污染玩法。低序列偏收尸、掘墓、通灵，中序列开始看守冥界之门、摆渡亡者，高序列进入死亡执政官、苍白皇帝和死神权柄。

这条途径的核心体验是“死亡不是终点，而是一套仍在运行的秩序”。它既可以帮助调查尸体和亡灵，也会让玩家越来越接近死者世界。

#### 10.10.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 收尸人、掘墓人、通灵者 | 尸检、亡灵沟通、墓地探索。 |
| 中序列 | 通灵师、看门人 | 控灵、冥界门、死亡仪式。 |
| 半神 | 不死者、摆渡人 | 死亡抗性、亡灵军队、灵魂摆渡。 |
| 天使以上 | 死亡执政官、苍白皇帝、死神 | 死亡领域、亡者国度、死神权柄。 |

#### 10.10.3 材料与非凡特性设计

死神途径材料应来自墓园、战场、南大陆遗迹、拜朗陵寝和亡灵生物。低序列材料可以通过值夜者、墓园支线、戴莉教学获得；高序列材料必须与拜朗帝国遗产、死神陵寝和古代亡灵军团绑定。

非凡特性表现建议：

- 低序列特性偏黑百合、尸布、乌鸦眼球、冷色晶体。
- 中序列特性偏门、钥匙、灵魂结晶、墓碑碎片。
- 高序列特性偏苍白王冠、冥河、死亡权杖。

#### 10.10.4 地图与任务适配

- 鲁恩王国：戴莉、通灵审查、尸检案件。
- 南大陆：拜朗遗迹、殖民城镇、死神陵寝。
- 五海区域：幽灵船海域、亡灵船员。
- 神弃之地：古代死亡仪式和神话残响。

代表任务：

- “尸体会说话”：通过尸检和通灵还原案件。
- “墓地巡夜”：处理新生怨灵。
- “看门人”：阻止冥界缝隙扩大。
- “摆渡亡者”：护送一批灵魂穿过危险区域。

#### 10.10.5 失控与副作用

- 体温降低，普通 NPC 感到不适。
- 亡灵会主动靠近玩家。
- 通灵过度会被亡者记忆污染。
- 看门人阶段可能在现实看见冥界门缝。
- 高序列失控时，角色会把活人也视为“未来的死者”。

#### 10.10.6 开发优先级

第一阶段做收尸人到通灵者：

- 尸检、亡灵沟通、墓地调查。

第二阶段做通灵师和看门人：

- 控灵、死亡仪式、冥界裂隙。

第三阶段做高阶：

- 拜朗陵寝、亡灵军团、死亡领域。

### 10.11 黄昏巨人/战士途径 `Twilight Giant Pathway`

#### 10.11.1 途径定位

黄昏巨人途径是“战士、巨力、防御、守护、黄昏、衰败”的途径。它适合做近战坦克、战场前排、巨人遗迹、弗萨克战争和神弃之地巨人王庭内容。低序列偏体魄和武器，中序列偏守护和黎明骑士，高序列进入黄昏、巨人和衰败权柄。

这条途径的核心体验是“站在队伍最前面，承受无法回避的攻击”。它不是灵巧路线，而是重甲、盾牌、巨剑、冲锋、守护和以身体对抗灾难。

#### 10.11.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 战士、格斗家、武器大师 | 近战、招架、武器专精。 |
| 中序列 | 黎明骑士、守护者 | 重甲、防御光辉、保护队友。 |
| 半神 | 猎魔者、银骑士 | 反怪物、军团冲锋、银白领域。 |
| 天使以上 | 荣耀者、神之手、黄昏巨人 | 黄昏衰败、巨人权柄、神话战场。 |

#### 10.11.3 材料与非凡特性设计

黄昏巨人途径材料应来自巨人遗迹、战场、弗萨克北境、神弃之地和巨人王庭。低序列材料可通过军营、佣兵、战士公会或战场敌人获得；高序列材料必须绑定巨人王庭、黄昏领域和古神遗产。

非凡特性表现建议：

- 低序列特性偏骨骼、肌肉、金属结晶。
- 中序列特性偏盔甲碎片、盾牌纹路、黎明光辉。
- 高序列特性偏巨人心脏、黄昏太阳、衰败王座。

#### 10.11.4 地图与任务适配

- 弗萨克帝国：北境冰原、战神要塞、冰封遗迹。
- 神弃之地：白银城战士、巨人王庭。
- 鲁恩王国：战士类敌人、官方行动前排。
- 异空间：黄昏战场、古神记忆。

代表任务：

- “守住门口”：玩家必须承受多轮攻击保护平民撤离。
- “武器大师试炼”：使用不同武器击败敌人，解锁武器动作。
- “巨人王庭回声”：在巨人遗迹中面对古代黄昏残影。
- “荣耀与衰败”：高阶任务让玩家理解黄昏不是光明，而是万物走向终末。

#### 10.11.5 失控与副作用

- 身体逐渐巨大化或笨重化。
- 对正面战斗产生执念，不愿后退。
- 高防御带来感知迟钝，容易被精神或命运类能力针对。
- 黄昏污染会让周围物品加速老化、锈蚀。

#### 10.11.6 开发优先级

第一阶段做战士到武器大师：

- 近战、格挡、武器熟练度。

第二阶段做黎明骑士和守护者：

- 护盾、保护队友、重甲玩法。

第三阶段做高阶：

- 巨人王庭、黄昏领域、神话战场。

### 10.12 魔女/刺客途径 `Demoness Pathway`

#### 10.12.1 途径定位

魔女途径是“刺杀、教唆、疾病、魅惑、镜面、灾祸、灾难传播”的途径。它适合做暗杀敌人、灾祸事件、魔女教派、镜中空间和政治阴谋。低序列偏刺客和教唆，中序列进入女巫、欢愉、痛苦和疾病，高序列则是绝望、灾难、不老和魔女权柄。

这条途径的核心体验是“美丽、欲望和灾难纠缠在一起”。在游戏里要避免把它简化成魅惑职业，它更重要的是灾祸扩散、社会操控和让普通案件突然恶化。

#### 10.12.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 刺客、教唆者、女巫 | 暗杀、挑拨、诅咒和初级法术。 |
| 中序列 | 欢愉、痛苦 | 魅惑、痛苦控制、疾病传播。 |
| 半神 | 绝望、灾难 | 区域灾祸、瘟疫、镜面逃脱。 |
| 天使以上 | 不老、灾祸、魔女 | 大灾难、命运灾祸、魔女权柄。 |

#### 10.12.3 材料与非凡特性设计

魔女途径材料应来自毒花、蜘蛛、疾病源、镜面仪式、魔女教派据点和灾祸发生地。材料获取任务应强调危险社交和隐藏身份，而不是公开战斗。

非凡特性表现建议：

- 低序列特性偏黑羽、毒花、匕首状晶体。
- 中序列特性偏镜片、红色液体、痛苦面具。
- 高序列特性偏灾祸王冠、衰败花朵、无龄面容。

#### 10.12.4 地图与任务适配

- 鲁恩王国：特莉丝线、魔女教派支线。
- 贝克兰德：政治渗透、贵族阴谋、灾祸传播。
- 因蒂斯共和国：社交沙龙和政治刺杀。
- 五海区域：海盗港口中的魔女情报网。
- 异空间：镜面世界、灾祸仪式。

代表任务：

- “被教唆的人”：普通 NPC 突然做出极端行为，玩家追查教唆者。
- “镜中逃脱”：魔女敌人通过镜面移动，玩家需要打破镜面网络。
- “疾病晚宴”：贵族沙龙中爆发隐秘疾病和情绪操控。
- “灾难前夜”：玩家必须判断灾祸会在哪里降临。

#### 10.12.5 失控与副作用

- 情绪、欲望和痛苦感知增强。
- 周围人更容易被挑拨，关系值波动加大。
- 使用疾病类能力会留下长期地图污染。
- 镜中可能出现另一个自己。
- 高序列失控时，角色会把灾难视为艺术或仪式。

#### 10.12.6 开发优先级

第一阶段做刺客和教唆者：

- 暗杀、潜行、挑拨 NPC。

第二阶段做女巫到痛苦：

- 诅咒、镜面、疾病、情绪控制。

第三阶段做高阶：

- 灾难事件、魔女教派主线、区域级灾祸。

### 10.13 红祭司/猎人途径 `Red Priest Pathway`

#### 10.13.1 途径定位

红祭司途径是“猎人、挑衅、阴谋、火焰、战争、军团指挥”的途径。它适合做战术战斗、战场指挥、挑衅敌人、战争副本和弗萨克/因蒂斯/鲁恩冲突。低序列偏追踪和挑衅，中序列偏阴谋和纵火，高序列进入铁血骑士、战争主教、征服者和红祭司权柄。

这条途径的核心体验是“把冲突引向自己设计好的战场”。它不是单纯战士路线，而是猎人式布置、战争指挥和用挑衅让敌人犯错。

| 天使以上 | 天气术士、征服者、红祭司 | 战争灾难、征服权柄、火焰军团。 |

#### 10.13.3 材料与非凡特性设计

红祭司途径材料应来自猎物、战场、火焰生物、军队仓库和战争遗迹。低序列材料可通过狩猎、追踪任务获得；高序列材料应绑定战役、城市攻防和军团仪式。

#### 10.13.4 地图与任务适配

- 弗萨克帝国：战神要塞、战争前线。

- 五海区域：海盗战争、舰队冲突。

代表任务：

- “挑衅者训练”：用语言或行动让敌人放弃防守。
- “纵火路线”：玩家布置火点，制造撤离或混乱。
- “阴谋家的棋盘”：通过几个小事件引导敌对组织互相冲突。
- “战争主教”：指挥多支小队完成战场目标。

#### 10.13.5 失控与副作用

- 角色容易挑衅过度，引来不必要战斗。
- 火焰技能可能造成城市声望下降。
- 阴谋家阶段会让玩家越来越倾向牺牲小人物达成大目标。
- 高序列战争污染会让和平区域也出现冲突事件。

#### 10.13.6 开发优先级

第一阶段做猎人到纵火家：

- 追踪、挑衅、火焰陷阱。

第二阶段做阴谋家和收割者：

- 战术布置、连锁击杀、敌方士气。

第三阶段做高阶：

- 军团指挥、战争领域、征服仪式。

### 10.14 隐者/窥秘人途径 `Hermit Pathway`

#### 10.14.1 途径定位

隐者途径是“神秘知识、窥探、秘术、卷轴、星象、童话与神秘再现”的途径。它适合做法术收集、神秘学研究、卷轴制作、隐秘组织和嘉德丽雅/贝尔纳黛相关内容。低序列偏窥秘和法术，中序列偏卷轴、占星和秘术，高序列进入神秘学家、知识皇帝和隐者权柄。

这条途径的核心体验是“知识有力量，也有毒”。玩家学习越多秘术，就越能应对复杂局面，但也越容易看见不该看的存在。

#### 10.14.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 窥秘人、格斗学者、巫师 | 神秘感知、基础法术、近战补强。 |
| 中序列 | 卷轴教授、星象师 | 制作卷轴、星象预测、法术储备。 |
| 半神 | 神秘学家、预言大师 | 高阶秘术、预言、隐秘仪式。 |
| 天使以上 | 知识导师、知识皇帝、隐者 | 知识权柄、秘术体系、隐秘国度。 |

#### 10.14.3 材料与非凡特性设计

隐者途径材料应来自神秘学书籍、星象台、秘术组织、海上强者和罗塞尔遗产。低序列材料可以在黑市和神秘学圈获得；高序列材料应与贝尔纳黛、罗塞尔、隐秘贤者污染和知识教会冲突绑定。

非凡特性表现建议：

- 低序列特性偏眼睛、书页、符号晶体。
- 中序列特性偏卷轴、星盘、墨水。
- 高序列特性偏知识王冠、秘术书库、隐秘星空。

#### 10.14.4 地图与任务适配

- 鲁恩王国：神秘学入门、黑市书籍。
- 因蒂斯共和国：罗塞尔博物馆、隐秘学派。
- 五海区域：嘉德丽雅、贝尔纳黛、海上知识交易。
- 异空间：星象、隐秘贤者低语、知识污染。

代表任务：

- “不可读之书”：玩家必须判断一本书能否安全阅读。
- “卷轴教授”：制作一次可储存法术的卷轴。
- “星象之夜”：通过星象预测海上风暴或敌人行动。
- “隐秘贤者低语”：知识带来能力，也带来污染追踪。

#### 10.14.5 失控与副作用

- 阅读禁忌知识后出现幻听。
- 法术卷轴可能释放错误法术。
- 窥秘失败会被高位存在反向看见。
- 玩家越了解隐秘，越容易触发知识污染。
- 高序列失控会把世界看成符号和秘术结构。

#### 10.14.6 开发优先级

第一阶段做窥秘人到巫师：

- 神秘感知、基础法术、危险阅读。

第二阶段做卷轴教授和星象师：

- 卷轴制作、星象预测、法术库。

第三阶段做高阶：

- 神秘学家、知识皇帝、隐秘贤者污染。

### 10.15 完美者/通识者途径 `Paragon Pathway`

#### 10.15.1 途径定位

完美者途径是“通识、鉴定、工匠、机械、炼金、科技、完美造物”的途径。它适合做装备制造、封印物改造、机械机关、蒸汽与机械教会、罗塞尔遗产和因蒂斯科技线。低序列偏鉴定和通识，中序列进入工匠和机械专家，高序列则能制造强大神奇物品、炼金生命和完美造物。

这条途径的核心体验是“把神秘变成工程”。它可以让玩家把材料、非凡特性和机械结构转化为可控装备，但风险是造物失控、物品获得意识、技术崇拜和人性工具化。

#### 10.15.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 通识者、考古学家、鉴定师 | 鉴定、拆解、识别材料和古物。 |
| 中序列 | 工匠、天文学家 | 制造神奇物品、机械机关、观测星象。 |
| 半神 | 炼金术士、古代炼金师 | 炼金生命、封印物改造、大型装置。 |
| 天使以上 | 知识皇帝、文明启蒙者、完美者 | 文明技术、完美造物、机械权柄。 |

#### 10.15.3 材料与非凡特性设计

完美者途径材料应来自机械教会、因蒂斯工厂、罗塞尔博物馆、地下实验室和古代遗迹。非凡特性不应只作为晋升材料，也应成为装备制造核心。

非凡特性表现建议：

- 低序列特性偏齿轮、镜片、金属结晶。
- 中序列特性偏机械心脏、炼金核心、星象仪零件。
- 高序列特性偏文明火种、完美机械胚胎、知识装置。

#### 10.15.4 地图与任务适配

- 因蒂斯共和国：特里尔、罗塞尔博物馆、地下陵墓。
- 鲁恩王国：机械之心、封印物鉴定。
- 五海区域：船只改造、海上装备制造。
- 异空间：机械与星象装置连接灵界坐标。

代表任务：

- “鉴定未知封印物”：判断物品能力和副作用。
- “第一件神奇物品”：用低序列特性制造可用装备。
- “罗塞尔机关”：破解前穿越者留下的机械谜题。
- “炼金生命失控”：玩家制造或处理一个产生意识的造物。

#### 10.15.5 失控与副作用

- 角色把人和非凡特性都当成零件。
- 制造物品可能拥有自己的意志。
- 鉴定失败会误判封印物副作用。
- 过度依赖装备会降低自身非凡能力成长。

#### 10.15.6 开发优先级

第一阶段做通识者到鉴定师：

- 物品鉴定、材料识别、古物解析。

第二阶段做工匠和天文学家：

- 装备制造、机械机关、星象观测。

第三阶段做高阶：

- 炼金生命、文明装置、完美造物。

### 10.16 命运之轮/怪物途径 `Wheel of Fortune Pathway`

#### 10.16.1 途径定位

命运之轮途径是“怪物、幸运、厄运、概率、预感、命运摆动”的途径。它适合做随机事件、概率检定、幸运爆发、厄运转移和命运支线。低序列偏异常直觉和危险预感，中序列开始操控幸运与厄运，高序列进入命运循环、概率支配和命运之轮权柄。

这条途径的核心体验是“你不知道下一次骰子会落在哪里”。它不稳定，但能在绝境中制造奇迹，也可能把小风险放大成灾难。

#### 10.16.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 怪物、机器、幸运者 | 危险预感、概率提示、幸运触发。 |
| 中序列 | 灾祸教士、赢家 | 厄运转移、幸运积累、概率操控。 |
| 半神 | 混乱行者、先知 | 命运分支、随机事件重排。 |
| 天使以上 | 混沌猎手、命运天使、命运之轮 | 命运循环、概率支配、唯一性骰子。 |

#### 10.16.3 材料与非凡特性设计

命运之轮途径材料应来自异常幸运/厄运事件、概率异常地点、命运相关生物和威尔·昂赛汀支线。材料获取要带随机性，但不能让玩家纯看脸；最好通过多条路径提高成功概率。

非凡特性表现建议：

- 低序列特性偏骰子、硬币、怪异眼睛。
- 中序列特性偏厄运黑云、幸运光点、概率结晶。
- 高序列特性偏命运轮盘、红点骰子、循环时钟。

#### 10.16.4 地图与任务适配

- 鲁恩王国：异常好运/厄运支线。
- 贝克兰德：威尔相关任务、命运干涉。
- 五海区域：海上随机灾难和幸运航线。
- 异空间：命运河流、概率之骰。

代表任务：

- “幸运的一天”：玩家连续遇到好运，但每次好运背后都转移了别人的厄运。
- “厄运追踪”：找到厄运源头，把它引向可承受区域。
- “概率之骰”：高阶封印物任务，骰子结果改变地图事件。
- “命运分叉”：玩家看到多个可能结局，但只能提高其中一个概率。

#### 10.16.5 失控与副作用

- 随机事件频率上升。
- 好运后必有代价，可能由队友承担。
- 厄运转移会影响 NPC 关系。
- 高序列失控时，任务目标、奖励和失败条件都可能随机化。

#### 10.16.6 开发优先级

第一阶段做怪物到幸运者：

- 预感、幸运触发、小概率事件。

第二阶段做灾祸教士和赢家：

- 厄运转移、幸运资源、概率调节。

第三阶段做高阶：

- 命运分支、概率之骰、命运之轮终局。

### 10.17 母亲/耕种者途径 `Mother Pathway`

#### 10.17.1 途径定位

母亲途径是“生命、丰饶、种植、治疗、繁殖、自然生长”的途径。它适合费内波特王国、丰饶平原、古老森林、大地母神教会和生命污染玩法。低序列偏种植和治疗，中序列进入生物培育和自然仪式，高序列涉及生命创造、丰饶过剩和母亲权柄。

这条途径的核心体验是“生命本身也会过量”。它可以治疗、恢复、种植和生产资源，但失控时会带来过度繁殖、肉体畸变和生命力泛滥。

#### 10.17.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 耕种者、医师、丰收祭司 | 种植、治疗、资源生产。 |
| 中序列 | 生物学家、德鲁伊 | 生物培育、植物控制、自然变形。 |
| 半神 | 古代炼金师、荒芜主母 | 生命创造、生态改造、丰饶污染。 |
| 天使以上 | 生命赐予者、荒芜母巢、母亲 | 生命权柄、繁殖领域、母神神性。 |

#### 10.17.3 材料与非凡特性设计

母亲途径材料应来自费内波特农田、古老森林、神庙、稀有植物和生命学派。材料获取可以和种植系统结合：玩家不是只采集，还要培育、浇灌、等待或净化。

非凡特性表现建议：

- 低序列特性偏种子、绿色晶体、药草根须。
- 中序列特性偏树心、胎盘状结晶、活化藤蔓。
- 高序列特性偏生命胚胎、母巢核心、丰饶王冠。

#### 10.17.4 地图与任务适配

- 费内波特王国：丰饶平原、古老森林、大地神庙。
- 鲁恩王国：治疗支线、药草种植。
- 南大陆：疫病与生命畸变。
- 异空间：生命母巢、自然灵性副本。

代表任务：

- “异常丰收”：农作物过度生长，村庄被生命力吞没。
- “医师的边界”：治疗一个本该死亡的人，引发生命债务。
- “森林会移动”：德鲁伊阶段与活森林沟通。
- “母巢污染”：高阶区域中生命不再区分人、动物和植物。

#### 10.17.5 失控与副作用

- 伤口愈合过快，形成异常组织。
- 周围植物疯长，堵塞道路。
- 治疗可能附带繁殖、寄生或畸变风险。
- 高序列失控时，角色会把个体生命视为生态材料。

#### 10.17.6 开发优先级

第一阶段做耕种者到丰收祭司：

- 种植、治疗、食物和材料生产。

第二阶段做生物学家和德鲁伊：

- 生物改造、植物控制、森林地图互动。

第三阶段做高阶：

- 生命创造、丰饶污染、母巢副本。

### 10.18 月亮/药师途径 `Moon Pathway`

#### 10.18.1 途径定位

月亮途径是“药剂、血族、月相、恢复、毒性、变异”的途径。它适合药剂制作、血族社会、满月诅咒、治疗与毒药双重玩法。低序列偏药师和驯兽，中序列进入吸血鬼、男爵、子爵等血族层级，高序列则与月亮、血液、再生和母神污染相关。

这条途径的核心体验是“药可以救人，也可以改变人”。它比母亲途径更偏药剂和血液，既能治疗，也能制造毒药、强化身体和处理满月影响。

#### 10.18.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 药师、驯兽师、吸血鬼 | 药剂、毒药、血液恢复、驯兽。 |
| 中序列 | 男爵、深红学者 | 血族能力、月相法术、药剂强化。 |
| 半神 | 巫王、美神 | 高阶药剂、魅惑、血液领域。 |
| 天使以上 | 荒芜主母、自然行者、月亮 | 月亮权柄、血族高阶、生命月相。 |

#### 10.18.3 材料与非凡特性设计

月亮途径材料应来自药店、血族据点、满月仪式、稀有动物、月光草药和地下诊所。材料获取适合做成采药、炼药和血液交易系统。

非凡特性表现建议：

- 低序列特性偏药瓶、红色晶体、兽牙。
- 中序列特性偏血色月牙、血族心脏、深红结晶。
- 高序列特性偏月冕、血池、自然月相核心。

#### 10.18.4 地图与任务适配

- 鲁恩王国：埃姆林、血族、地下药剂交易。
- 费内波特王国：大地母神与血族关系。
- 南大陆：疫病、毒药、血液仪式。
- 异空间：满月低语、月亮污染。

代表任务：

- “调配第一瓶药剂”：药师教学，治疗或毒杀由玩家选择。
- “满月病人”：满月期间 NPC 状态恶化，玩家需要压制。
- “血族债务”：接受血族帮助后必须偿还血液或服务。
- “深红月冕”：神奇物品任务，抵抗满月但带来吸血冲动。

#### 10.18.5 失控与副作用

- 对血液产生渴望。
- 药剂效果过强，带来成瘾或变异。
- 满月期间能力增强但理智下降。
- 治疗和毒药界限模糊，NPC 会质疑玩家伦理。

#### 10.18.6 开发优先级

第一阶段做药师到吸血鬼：

- 药剂、毒药、血液恢复、满月反应。

第二阶段做男爵和深红学者：

- 血族社交、月相法术、药剂强化。

第三阶段做高阶：

- 月亮权柄、血族政治、满月污染终局。

### 10.19 深渊/罪犯途径 `Abyss Pathway`

#### 10.19.1 途径定位

深渊途径是“罪恶、欲望、恶魔、杀戮、诅咒、深渊污染”的途径。它适合做恶魔敌人、连环杀人案、欲望诱导、犯罪调查和南大陆/地下黑市高危支线。低序列偏犯罪、冷血和感知恶意，中序列逐渐恶魔化，高序列进入欲望使徒、恶魔君王和深渊权柄。

这条途径的核心体验是“罪恶会回应罪恶”。它非常适合做恐怖案件和高压追踪，但不适合给玩家无代价使用；一旦可用，必须伴随强烈道德和失控风险。

#### 10.19.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 罪犯、冷血者、连环杀手 | 恶意感知、暗杀、犯罪痕迹。 |
| 中序列 | 恶魔、欲望使徒 | 恶魔化、欲望诱导、诅咒攻击。 |
| 半神 | 恶魔君王、污秽君王 | 深渊领域、腐化、恐惧传播。 |
| 天使以上 | 暗侧、深渊、未知高阶 | 深渊权柄、罪恶国度。 |

#### 10.19.3 材料与非凡特性设计

深渊途径材料应来自犯罪现场、恶魔遗留、血腥祭祀、地下邪教和深渊污染区域。材料获取不应鼓励玩家无脑犯罪，可以通过调查、收容、净化或黑市交易获得，但每一步都应有风险。

非凡特性表现建议：

- 低序列特性偏黑红结晶、利爪、带血器官。
- 中序列特性偏恶魔角、欲望火焰、诅咒心脏。
- 高序列特性偏深渊裂隙、污秽王冠、黑色太阳。

#### 10.19.4 地图与任务适配

- 鲁恩王国：连环杀人案、恶魔支线。
- 贝克兰德：欲望使徒和地下犯罪网络。
- 南大陆：深渊污染、邪教祭坛。
- 异空间：深渊裂隙和罪恶梦境。

代表任务：

- “杀人案现场”：玩家通过尸体、情绪残留和恶意痕迹追踪罪犯。
- “恶魔的交易”：敌人提出短期有利但长期堕落的选择。
- “欲望诱导”：NPC 被放大欲望后做出失控行为。
- “深渊裂口”：高阶区域不断生成罪恶幻象和恶魔敌人。

#### 10.19.5 失控与副作用

- 玩家更容易感知并放大他人恶意。
- 对话中出现残酷、诱导、杀戮选项。
- 恶魔化会提高战斗力但降低 NPC 信任。
- 高序列失控时，角色可能主动制造罪恶以获得力量。

#### 10.19.6 开发优先级

第一阶段做罪犯到连环杀手：

- 犯罪调查、恶意感知、暗杀敌人。

第二阶段做恶魔和欲望使徒：

- 恶魔化、欲望诱导、诅咒。

第三阶段做高阶：

- 深渊领域、恶魔君王、罪恶国度。

### 10.20 被缚者/囚犯途径 `Chained Pathway`

#### 10.20.1 途径定位

被缚者途径是“囚禁、节制、怨灵、狼人、活尸、束缚与释放”的途径。它适合做玫瑰学派、节制派与放纵派冲突、怨灵同伴、地下监牢和身体异变玩法。低序列偏囚犯和疯狂压抑，中序列进入狼人、活尸、怨魂，高序列则掌握束缚、诅咒和异变权柄。

这条途径的核心体验是“越能约束自己，越能使用怪物般的力量”。它和深渊途径相邻但气质不同：深渊是放纵罪恶，被缚者是把怪物关在身体里。

#### 10.20.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 囚犯、疯子、狼人 | 自我束缚、狂化、近战异变。 |
| 中序列 | 活尸、怨魂 | 高生存、灵体化、诅咒。 |
| 半神 | 木偶、沉默门徒 | 束缚敌人、沉默能力、诡异控制。 |
| 天使以上 | 古代邪物、神孽、被缚者 | 极端异变、束缚权柄、玫瑰学派核心。 |

#### 10.20.3 材料与非凡特性设计

被缚者途径材料应来自监牢、怨灵场所、玫瑰学派据点、狼人和活尸敌人。材料获取可以做成“收容而非杀死”任务，强调节制派的风格。

非凡特性表现建议：

- 低序列特性偏锁链、爪子、狂化血液。
- 中序列特性偏腐败心脏、怨灵结晶、尸布。
- 高序列特性偏封印锁、沉默门、神孽血肉。

#### 10.20.4 地图与任务适配

- 鲁恩王国：莎伦、马里奇、地下行动。
- 南大陆：玫瑰学派放纵派据点。
- 贝克兰德：节制派支线、怨灵战斗。
- 异空间：束缚领域、沉默门。

代表任务：

- “节制与放纵”：玩家选择帮助节制派或处理放纵派怪物。
- “怨魂委托”：以灵体状态穿过墙壁完成潜入。
- “锁住自己”：晋升任务要求玩家在狂化状态下不伤害无辜者。
- “沉默门徒”：区域内所有主动能力被压制，只能靠道具和队友配合。

#### 10.20.5 失控与副作用

- 狂化冲动和自我束缚值对抗。
- 玩家在低血量时可能自动变异。
- 怨魂状态会被净化类能力误伤。
- 过度节制会降低情绪表达，影响 NPC 关系。

#### 10.20.6 开发优先级

第一阶段做囚犯到狼人：

- 自我束缚、狂化、近战异变。

第二阶段做活尸和怨魂：

- 高生存、灵体穿行、诅咒。

第三阶段做高阶：

- 束缚领域、沉默、玫瑰学派终局。

### 10.21 黑皇帝/律师途径 `Black Emperor Pathway`

#### 10.21.1 途径定位

黑皇帝途径是“律师、贿赂、腐化、规则扭曲、秩序漏洞、王权”的途径。它适合做政治、法律、贵族、腐败、王室成神计划和帝国秩序玩法。低序列偏法律和话术，中序列偏贿赂与腐化，高序列可以扭曲规则、建立秩序、复活和走向黑皇帝权柄。

这条途径的核心体验是“规则不是束缚，而是可以被解释、利用和扭曲的武器”。它适合贝克兰德政治篇、因蒂斯政治线和王室阴谋。

#### 10.21.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 律师、野蛮人、贿赂者 | 法律话术、威慑、交易腐化。 |
| 中序列 | 腐化男爵、混乱导师 | 扭曲秩序、制造混乱、腐化组织。 |
| 半神 | 混乱猎手、熵之公爵 | 规则反制、秩序崩坏、权力领域。 |
| 天使以上 | 弑序亲王、黑皇帝 | 王权、复活秩序、帝国规则。 |

#### 10.21.3 材料与非凡特性设计

黑皇帝途径材料应来自法院、贵族、王室、贿赂交易、法律文本和政治事件。高序列材料必须绑定国家秩序、陵墓、王权仪式和乔治三世成神计划。

非凡特性表现建议：

- 低序列特性偏法典、金币、黑色印章。
- 中序列特性偏腐化冠冕、扭曲契约、混乱权杖。
- 高序列特性偏黑皇帝王座、帝国印玺、秩序陵寝。

#### 10.21.4 地图与任务适配

- 贝克兰德：贵族、王室、军情九处、乔治三世线。
- 因蒂斯共和国：政治演说和权力交易。
- 南大陆：殖民地法律和压迫秩序。
- 异空间：规则扭曲副本。

代表任务：

- “用法律打开门”：不用潜入，用法律文件取得进入权。
- “贿赂链条”：追查一条从小官员到贵族的腐败网络。
- “秩序扭曲”：敌人把玩家的合法行动判定为违法。
- “黑皇帝陵寝”：高阶任务绑定复活、陵墓和王权仪式。

#### 10.21.5 失控与副作用

- 玩家越来越倾向用利益解释一切。
- 贿赂成功后会形成长期债务。
- 扭曲规则可能反噬自身队友。
- 高序列失控时，角色会认为自己就是法律和秩序。

#### 10.21.6 开发优先级

第一阶段做律师到贿赂者：

- 法律对话、威慑、交易。

第二阶段做腐化男爵和混乱导师：

- 腐化组织、规则冲突、混乱状态。

第三阶段做高阶：

- 王权领域、复活秩序、黑皇帝仪式。

### 10.22 审判者/仲裁人途径 `Justiciar Pathway`

#### 10.22.1 途径定位

审判者途径是“仲裁、治安、审讯、法官、惩戒、秩序、裁决”的途径。它和黑皇帝途径相邻，但方向相反：黑皇帝扭曲规则，审判者建立规则并执行惩罚。它适合官方执法、军情九处、封锁区、审判仪式和区域禁令玩法。

这条途径的核心体验是“制定规则，然后让所有人付出违反规则的代价”。它适合做控制、禁魔、审讯、法律领域和城市秩序系统。

#### 10.22.2 序列玩法推进

| 阶段 | 序列 | 游戏定位 |
| --- | --- | --- |
| 低序列 | 仲裁人、治安官、审讯者 | 执法、审讯、命令、限制行动。 |
| 中序列 | 法官、惩戒骑士 | 禁令、惩戒、战斗裁决。 |
| 半神 | 混乱猎手、秩序导师 | 区域规则、秩序压制。 |
| 天使以上 | 秩序之手、审判者 | 世界级规则、审判权柄。 |

#### 10.22.3 材料与非凡特性设计

审判者途径材料应来自警署、法院、军方、仲裁仪式、古老法典和秩序神庙。辅助材料中的法律文本、法槌、长期不变法典等，都很适合做任务道具，而不是普通材料。

非凡特性表现建议：

- 低序列特性偏石板、法槌、徽章、闪电晶体。
- 中序列特性偏审判剑、锁链、秩序树枝。
- 高序列特性偏黄铜法典、秩序之手、审判天平。

#### 10.22.4 地图与任务适配

- 鲁恩王国：警署、军情九处、贝克兰德执法任务。
- 因蒂斯共和国：政治审判和秩序冲突。
- 弗萨克帝国：军事法庭和要塞禁令。
- 异空间：规则审判副本。

代表任务：

- “审讯室”：玩家通过证据、威慑和禁令让嫌疑人开口。
- “封锁街区”：制定临时规则，限制异常扩散。
- “惩戒骑士”：违反规则的敌人会受到自动惩罚。
- “公正审判”：高阶仪式要求在严重事件中完成足够公正的审判。

#### 10.22.5 失控与副作用

- 玩家会变得刻板、缺乏弹性。
- 禁令过强可能伤害无辜者。
- 审判失败会反噬自身权威。
- 高序列失控时，角色可能把复杂人性简化成有罪/无罪。

#### 10.22.6 开发优先级

第一阶段做仲裁人到审讯者：

- 执法、命令、审讯、证据压制。

第二阶段做法官和惩戒骑士：

- 区域禁令、惩罚、战斗裁决。

第三阶段做高阶：

- 秩序领域、公正审判、世界规则。

## 12. 途径数据结构文档

> 第 8 章“数据化字段建议”是早期的简略字段清单，便于快速记录魔药配方、非凡特性和封印物。  
> 本章是正式开发数据结构规范，用于后续拆分为 JSON、数据库表、TypeScript 类型、编辑器表单和 Codex 自动化开发输入。

### 12.0 设计原则

22 条途径和 220 个序列如果只用自然语言记录，很难直接服务游戏开发。统一数据结构的目标是把“原著资料”“Wiki 整理”“官方补充”和“游戏化改编”分开标注，让同一份资料既能保留考据来源，也能被游戏系统、UI、背包、魔药调配、晋升、任务和 Codex 后续开发稳定读取。

使用规则：

- 中文字段用于玩家显示，例如 `pathway_name_cn`、`sequence_name_cn`、`ability_name_cn`。
- 英文字段用于代码、数据库、文件名和跨语言引用，例如 `pathway_id`、`sequence_id`、`ability_id`。
- 原著资料、Wiki 资料、官方补充资料和游戏化改编内容必须用 `canon_status` 与 `source_note` 分开标注。
- 游戏化字段可以服务玩法，但不能和考据字段混在一起；涉及改编时应写入 `game_adaptation_note` 或对应设计备注。
- 同一实体只保留一个稳定 `id`，显示名、描述、来源和数值可后续迭代。

#### 12.0.1 命名规范

所有 `id` 使用小写英文 + 下划线，不使用空格、中文、驼峰或特殊符号。

| 类型 | 命名规则 | 示例 |
| --- | --- | --- |
| 途径 | `{pathway}` | `fool`, `error`, `door` |
| 序列 | `{pathway}_{sequence_number}_{sequence_en}` | `fool_09_seer`, `fool_08_clown` |
| 材料 | `mat_{material_en}` | `mat_lavos_squid_blood` |
| 技能 | `skill_{sequence_or_pathway}_{ability_en}` | `skill_seer_spiritual_vision` |
| 仪式 | `ritual_{pathway}_{sequence_number}_{short_name}` | `ritual_fool_04_bizarro_sorcerer` |
| 非凡特性 | `char_{pathway}_{sequence_number}_{sequence_en}` | `char_fool_09_seer` |
| 封印物 | `artifact_{code_or_level}_{artifact_en}` | `artifact_0_08_quill_of_alzuhod` |
| 任务 | `quest_{region}_{short_name}` | `quest_tingen_become_seer` |

#### 12.0.2 `canon_status` 枚举

`canon_status` 用于区分资料可靠性与改编性质。任何用于玩法、UI 或数据库的记录都应尽量填写该字段。

| 值 | 含义 |
| --- | --- |
| `novel_confirmed` | 第一部小说正文明确出现 |
| `book2_confirmed` | 《宿命之环》中出现 |
| `official_supplement` | 作者访谈、官方百科、官方补充设定 |
| `wiki_only` | 来自 Wiki 整理，尚未二次核对 |
| `game_adapted` | 为了游戏玩法进行的改编 |
| `original_placeholder` | 项目原创占位设定 |

#### 12.0.3 `source_note` 来源说明

`source_note` 用于记录更具体的资料来源，可写成字符串或字符串数组。推荐值包括：

- `第一部小说正文`
- `《宿命之环》`
- `Lord of the Mysteries Wiki`
- `官方百科`
- `作者补充`
- `游戏化改编`

#### 12.0.4 通用 TypeScript 类型

```ts
type CanonStatus =
  | "novel_confirmed"
  | "book2_confirmed"
  | "official_supplement"
  | "wiki_only"
  | "game_adapted"
  | "original_placeholder";

type SourceNote =
  | "第一部小说正文"
  | "《宿命之环》"
  | "Lord of the Mysteries Wiki"
  | "官方百科"
  | "作者补充"
  | "游戏化改编"
  | string;

type Id = string;
```

### 12.1 Pathway 途径表

用于记录一整条途径，例如愚者途径、错误途径、门途径。途径表是所有序列、材料、能力、仪式和非凡特性的上级索引。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `pathway_id` | string | 是 | 稳定代码 ID，例如 `fool`。 |
| `pathway_name_cn` | string | 是 | 中文显示名，例如 `愚者途径`。 |
| `pathway_name_en` | string | 是 | 英文显示名，例如 `Fool Pathway`。 |
| `group` | string | 是 | 途径分组，可按相邻途径、源质、阵营或开发批次分类。 |
| `adjacent_pathways` | string[] | 是 | 相邻途径 ID 列表，例如 `["error", "door"]`。 |
| `theme_keywords` | string[] | 是 | 主题关键词，用于 UI 标签和策划检索。 |
| `gameplay_role` | string | 是 | 在游戏中的总体定位。 |
| `main_mechanics` | string[] | 是 | 该途径主要玩法机制。 |
| `risk_theme` | string | 是 | 失控、污染或副作用主题。 |
| `available_sequences` | number[] | 是 | 当前资料或版本可用序列号。 |
| `unlock_region` | string | 是 | 首次解锁区域。 |
| `source_note` | SourceNote[] | 是 | 资料来源。 |
| `canon_status` | CanonStatus | 是 | 资料状态。 |

JSON 示例：

```json

  "pathway_id": "fool",
  "pathway_name_cn": "愚者途径",
  "pathway_name_en": "Fool Pathway",
  "group": "cowardly_trio",
  "adjacent_pathways": ["error", "door"],
  "theme_keywords": ["占卜", "命运", "表演", "秘偶", "历史", "奇迹"],
  "gameplay_role": "调查辅助、身份伪装、替身逃脱和后期高阶规则操控",
  "main_mechanics": ["divination", "disguise", "marionette_control", "historical_projection"],
  "risk_theme": "过度窥探命运和高位知识导致精神污染、误判和失控",
  "available_sequences": [9, 8, 7, 6, 5, 4, 3, 2, 1, 0],
  "unlock_region": "鲁恩王国-廷根市",
  "source_note": ["第一部小说正文", "Lord of the Mysteries Wiki", "游戏化改编"],
  "canon_status": "wiki_only"
}
```

TypeScript interface 示例：

```ts
interface Pathway {
  pathway_id: Id;
  pathway_name_cn: string;
  pathway_name_en: string;
  group: string;
  adjacent_pathways: Id[];
  theme_keywords: string[];
  gameplay_role: string;
  main_mechanics: string[];
  risk_theme: string;
  available_sequences: number[];
  unlock_region: string;
  source_note: SourceNote[];
  canon_status: CanonStatus;
}
```

### 12.2 Sequence 序列表

用于记录每个序列，例如序列 9 占卜家、序列 8 小丑。序列表是魔药、能力、晋升、任务和 UI 展示的核心数据表。

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `sequence_id` | string | 是 | 稳定序列 ID，例如 `fool_09_seer`。 |
| `pathway_id` | string | 是 | 所属途径 ID。 |
| `sequence_number` | number | 是 | 序列号，0 到 9。 |
| `sequence_name_cn` | string | 是 | 中文序列名。 |

### 12.9 后续拆分建议

正式进入开发时，建议将本章结构拆成独立数据文件：

```text
data/
├── pathways.json
├── sequences.json
├── materials.json
├── abilities.json
├── rituals.json
├── characteristics.json
└── sealed_artifacts.json
```

拆分后应保证：

- 所有关联只引用稳定 `id`，不直接引用显示名。
- UI 读取中文字段，代码逻辑读取 `id`、标签和枚举。
- 任务系统通过 `related_quests`、`unlock_conditions`、`unlock_region` 建立依赖。
- 背包系统读取 `Material`、`BeyonderCharacteristic` 和 `SealedArtifact`。
- 魔药调配系统读取 `Sequence.main_materials`、`Sequence.auxiliary_materials` 与 `Material.can_be_replaced_by_characteristic`。
- 晋升系统读取 `Sequence.advancement_ritual`、`Ritual` 和 `loss_control_risk`。
- Codex 后续开发可用这些结构生成校验脚本、编辑器表单、任务模板和 UI 原型。
